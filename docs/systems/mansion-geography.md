# 大宅地理規範（Mansion Geography）

> 阿圖斯大宅 + 訓練場是**同一座建築複合**（不是兩個獨立世界）。
> 這份文件定義地理結構，作為所有訓練所家庭故事的背景。
> 設計日期：2026-04-19

---

## 核心哲學

**貴族宅邸 + 奴隸工場 = 同一棟建築的兩側**

這符合羅馬 / 中世紀晚期富戶實際結構：
- 正門給客人（貴族用）
- 側門給奴隸 / 工人
- 兩邊永不相見，但共用中間基礎設施（廚房、醫療、倉庫）
- 家族成員**跨兩邊通行**（這是他們的家 + 他們的生意）

## 兩側的通行規則

### 貴賓走正門
- 從不看見訓練場
- 看到的訓練場 = 陽台俯瞰（像看動物園）
- 跟奴隸零接觸

### 奴隸走側門
- 從不進正門
- 偶爾被派到共用中段做事（倉庫、花園清理等）
- 看不到主人家族生活

### 家族成員跨兩邊
- **阿圖斯**：極罕見下來，多從陽台看
- **Livia**：透過管家 / 梅拉傳話，實體極少到側門
- **Marcus**：經常下來（喜歡打沙包）
- **未來女兒**：偶爾潛下來（勾搭戰士線）

### 僕人流動
- **梅拉**（廚房長）：主要在共用廚房，兩批出餐
- **老默**：醫療房位於共用中段，兩邊病人都看
- **管家 / 侍女 / 侍從**（未登場）：主要家側，偶爾傳話給奴隸側
- **獄卒 / 訓練員**：主要奴隸側

## 這個設計解釋了什麼

### 梅拉的存在感
- 梅拉是**廚房長**（不是唯一廚娘）— 她下面還有副手
- 她**同時認識**家側 + 訓練場側的人
- Livia 透過她暗助奴隸 = 合理通道

### Marcus 可以下來打沙包
- 正門 → 家族餐廳 → 私人練習場 → 側門 → 訓練場
- 他當自己家花園走

### 奧蘭的身分轉換
- **當傭人時**：在家側工作（整理房間、遞東西）
- **偷 Marcus 錢包**：在家側被抓
- **降級後**：被移到奴隸側
- **兩次生活的斷裂**：同一棟建築，兩個階級

### 家書能送達
- 磨坊 → 鄉下驛站 → Livia 娘家管道 → 阿圖斯大宅管家 → 梅拉廚房 → 奧蘭
- 主人批准（「這算小恩，留他一口氣」）
- Livia 親自過目（暗中加一句關心）

### 老默能同時照顧奴隸 + 主人家孩子
- 醫療房是共用設施
- 他看奧蘭的傷、也看 Marcus 的小病

## 敘事呈現原則

### 玩家視角
- 玩家**99% 在奴隸側**
- 陽台上的人 = 神秘、遙遠
- 家側的存在 = **只透過敘述、傳話、道具**得知
- 極罕見時刻才能窺視家側（某些結局）

### 敘述技巧
- 「訓練場**後面**傳來笑聲」 — 暗示家族生活
- 「侍從從**那邊**過來」 — 不明說哪邊
- 「樓上窗戶亮著燈」 — 讓玩家想像
- **從不直接寫家側**（保持神秘）

### 劇情高潮時才破牆
- Day 90+ 高 fame：阿圖斯在陽台舉杯
- 奧蘭死後：Livia 可能從陽台俯身看（一次性）
- 結局：某些結局會看到家側（失敗 / 逃脫時破門闖入）

## 其他訓練所的地理變體（未來）

這個「大宅 + 訓練場」是阿圖斯家的特色。其他訓練所可能結構不同：

| 訓練所 | 地理結構 | 特色 |
|---|---|---|
| 阿圖斯家 | 大宅 + 訓練場**同座** | 家族距離近，戲劇張力大 |
| （未來）軍方訓練所 | 軍營 + 訓練場 | 沒有家族，全員軍人 |
| （未來）地下拳場 | 酒館 + 地下室 | 黑市味，沒貴族 |
| （未來）貴族學院 | 學院 + 練武場 | 年輕貴族 + 角鬥士陪練 |

**通用模板**：訓練所 + 上層生活區 + 中段服務區。
