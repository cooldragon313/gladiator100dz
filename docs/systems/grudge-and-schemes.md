## § 0 設計哲學

### 0.1 核心
> 玩家**贏的方式**比贏本身更重要。每場戰鬥的選擇都有明確、可預期的回應。
> 仁慈不是道德口號——是**戰術選項**（少敵人）。
> 殘暴不是個性——是**麻煩的累積**（更多敵人來找你）。

### 0.2 不做隱形機率、做明確後果
- 玩家**戰鬥前知道**「這場是不是公開、誰在看」
- 玩家**選之前知道**「採臉會招仇、放過會減仇」
- **後果可預期、玩家自食其果**

### 0.3 三層敵意來源
1. **派遣對手**（外部、最常見）
2. **對手訓練所**（外部、faction 累積）
3. **訓練所內部 NPC**（既有事件機制、不走本系統 grudge 計分）

## § 1 戰後四選一（取代默默結束）

> ⚠️ **改動**：原 arena-system.md § 3.5「只有大勝才有選擇」→ **每場勝利都強制四選一**。
> 🆕 **2026-05-10**：新增「斷手腳」選項、跟 [義肢系統](blackmarket.md#-23-商品清單) 連動。

### 1.1 戰後 ChoiceModal

```
你贏了。
他倒在地上、還在動。

[ChoiceModal · 戰後四選一 · forced]
  ⚔️ 「斬首。」              → 對手死亡
  🦴 「斷他手腳。」           → 對手生還、永久殘缺（觸發義肢線、玩家可選）
  👁 「讓他記住今天。」        → 對手生還、被羞辱（採臉）
  🕊 「放他一命。」            → 對手生還、感激
```

### 1.2 四選一各自效果

| 選項 | 對手命運 | 個人 grudge | faction grudge | mercy 軸 | fame |
|---|---|---|---|---|---|
| ⚔️ **斬首** | 死、pool 標 `dead:true` | 歸零（人沒了） | **+2**（同袍恨） | `negative weight:1` | **+3** |
| 🦴 **斷手腳** 🆕 | 生還、永久殘缺（隨機 70% 手 / 30% 腿）| **+8**（比採臉狠、比殺輕） | **+5** | **`negative weight:2`** | **+4**（觀眾既驚又興奮）|
| 👁 **讓他記住** | 生還、帶羞辱（採臉） | 基礎 **+5**（場合倍率見 § 3） | 基礎 **+3** | 無 | **+5** |
| 🕊 **放他一命** | 生還、感激 | **-3**（甚至感激） | **-2** | `positive weight:1` | -2（觀眾覺得弱） |

### 1.3 「斷手腳」對對手的後續效果

- 對手永久帶 `severed:<部位>` → 下次再見 attribute -5、戰力降一階
- 對手 personality = `cruel`/`prideful` → 升級為**致命陰招目標**（不擇手段報復）
- 對手 personality = `patient` → 等待時機更久、但下次必用最狠陰招
- 對手 personality = `coward`/`humble` → 退出競技場（不再出現對手池）

### 1.4 為何「斬首」也會有 faction grudge

- 你殺了他、他同訓練所的兄弟會記得
- 但個人 grudge 歸零（沒人記仇）—— 殺了 = 永絕後患（個人）+ 引來組織（faction）
- 設計目的：殺人不是免費的、有戰術代價

### 1.5 中性勝利不存在

不要做「不選的默認」。每場勝利強制玩家選一個——**這是設計核心**。
（戰敗則沒得選、回到傷痕系統 + § 9B 玩家被斷手腳機制）

## § 2 仇恨度計分（個人 + 訓練所）

### 2.1 個人 grudge（per-opponent）

對手池每筆資料新增欄位：
```js
{
  id: 'morras_veteran_1',
  name: '老莫斯',
  // ... 既有欄位
  grudge: 0,              // 累積仇恨
  metPlayer: false,
  playerSparedMe: false,
  playerHumiliatedMe: false,  // 被採臉過、影響再見對白
}
```

**累積方式**（基礎值，未套 § 3 場合倍率 / § 4 特性倍率）：

| 戰鬥結局 | 個人 grudge |
|---|---|
| 普通勝（如戰後彈窗只接「讓他活著」這個變體）| **+1** |
| 採臉「讓他記住」 | **+5** |
| 斬首 | 歸零、人沒了 |
| 放過「放他一命」 | **-3** |

**觸發**：個人 grudge **≥ 5** → 對手親自發動陰招（強度高、單次劇烈）。

### 2.2 訓練所 faction grudge（per-school）

每個對手有 `school` 欄位（例：`morras` / `venus` / `independent`）。
玩家對該 school 的 faction grudge 跨場累積：

| 戰鬥結局 | faction grudge |
|---|---|
| 普通勝 | **+1** |
| 採臉 | **+3** |
| 放過 | **-2** |
| 斬首 | **+2**（同袍恨） |

**觸發**：faction grudge **≥ 8** → 該訓練所**任何成員**都可能對你陰招（中強度、廣泛）。

## § 3 場合倍率 + 立即報復 flag

### 3.1 場合倍率（採臉用）

| 場合 | 採臉個人 grudge | faction grudge | 設立即報復 flag |
|---|---|---|---|
| 🚪 **私下打鬥**（沒人看 / 訓練場切磋）| 基礎 **+5** | +3 | 否 |
| 👁 **侍從派遣**（基本派遣、主人不一定在）| **+7** | +4 | 否 |
| 🎩 **主人觀戰**（特殊事件、主人在台上）| **+12** | +5 | **是** |
| 🏛 **公開賽事**（Day 49 血戰宴會 / Day 75 公開宴會 / Day 100 萬骸祭 / 領主夜宴）| **+20** | +8 | **是** |

### 3.2 立即報復 flag 機制

```
公開場合採臉 + 對手生還 → Flags.set('revenge_target_<oppId>')

下次再遇到該對手：
  → 跳過機率擲、直接觸發陰招（仍要過 § 6 防禦判定）
  → 陰招強度升一級（普通陰招 → 致命陰招）
  → flag 消除（一次性、復仇完成）

如果對手在公開場合直接被殺（斬首）：
  → 個人 grudge 歸零（人沒了）
  → 但 faction grudge +8（同袍恨）
```

### 3.3 戰術張力範例

| 場景 | 玩家選擇 | 後果 |
|---|---|---|
| Day 49 血戰宴會、勝某莫拉斯戰士 | 採臉 | 對手 grudge +20 + 立即報復 flag、faction +8 |
| 同上 | 斬首 | 個人 grudge 歸零、faction +8、fame +3 |
| 同上 | 放過 | 個人 grudge -3、faction -2、fame -2 |

→ **公開場合 = 一次定生死、私下 = 慢慢累**

## § 4 對手特性修正

> 你說的對——**有衝動 / 魯莽 / 殘暴的人本來就更記仇**。

### 4.1 grudge 累積倍率

對手身上若帶以下特性，grudge 累積 **×1.5**（個人 + faction 同步）：

| 特性 | 為何更記仇 |
|---|---|
| `cruel` 殘忍 | 把每一刀砍到底的人不會忘 |
| `impulsive` 衝動 | 看到就想動手、聽到就想反擊 |
| `reckless` 急躁 | 衝動行事、不問緣由報復 |
| `prideful` 驕傲 | 被羞辱 = 名譽傷、必須挽回 |

### 4.2 grudge 累積倍率（反向）

對手帶以下特性、grudge 累積 **×0.7**（修養好、放得下）：

| 特性 | 為何放得下 |
|---|---|
| `patient` 耐心 | 知道有些仗不值得打 |
| `merciful` 仁慈 | 「我也曾被別人放過」 |
| `humble` 謙卑 | 沒那麼在意面子 |
| `loyal` 忠誠 | 把怨氣放在訓練所、不放個人 |

### 4.3 算式

```
final_grudge = base_grudge × 場合倍率 × 對手特性倍率
```

**範例**：
- Day 49 血戰宴會、採臉**殘忍對手**：5 × 4（公開）× 1.5（殘忍）= **+30** + 立即報復 flag
- 私下對練、採臉**仁慈對手**：5 × 1 × 0.7 = **+3.5** ≈ +4
- 同上但對手是**衝動 + 驕傲**：5 × 1 × 1.5 = +7.5（特性疊加只取最高、不疊乘）

## § 5 觸發機率（敵方是否發動陰招）

> 個人 grudge ≥ 5 或 faction grudge ≥ 8 才開始計算。

### 5.1 觸發率公式

```
基礎觸發率 = 0%

+ 個人 grudge ≥ 5  → +20%
+ 個人 grudge ≥ 8  → 額外 +15%（總 35%）
+ 個人 grudge ≥ 12 → 額外 +20%（總 55%）+ 解鎖致命陰招
+ faction grudge ≥ 8  → +10%
+ faction grudge ≥ 15 → 額外 +10%（總 20%）
+ fame ≥ 30 → +5%
+ fame ≥ 60 → +10%（總 +15%）
+ fame ≥ 90 → +15%（總 +30%）
- merciful 特性 → -10%
- mercy 軸正向 ≥ +5 → -5%

封頂 80%（總有條漏網之魚）
```

### 5.2 立即報復 flag 例外

如果該對手有 `revenge_target_<oppId>` flag → **直接 100% 觸發**、不擲。

### 5.3 復仇對手回歸機制（不靠運氣）

`revenge_target_<oppId>` flag 設下後、不該等對手池隨機抽到。
**強制下次派遣 80% 機率 = 該對手**。

> 「他在這場下了重注。**他買通了排程**讓你們再見。」

派遣事件揭露時、侍從會多一句鋪陳：
```
侍從：今日競技有一場。對手——[名字]。
侍從：（他停了一下。）
侍從：⋯⋯就是上次那個。
（你的胃緊了一下。）
```

## § 5.5 重逢戰前對白（revenge_target 出場）

> 對手登場時、若有 `revenge_target` flag → 換成「重逢戰前對白」（取代普通派遣對白）。
> 對白依**對手 personality** 變體。

### 5.5.1 開場對白變體

| Personality | 對白 |
|---|---|
| **cruel / impulsive** | 「終於等到你了。」<br>「上次的羞辱——我每天晚上閉眼都看到。」<br>「今天我要給你好看。」 |
| **prideful** | 「你以為那場結束了？」<br>「我這條命、剩下的就是為了今天。」 |
| **opportunist** | （冷笑）「我在這場上下了重注。」<br>「賭你會死。」 |
| **patient** | 「我等了很久。」<br>「這次不一樣。」（最可怕的一種） |

### 5.5.2 戰鬥開始 buff（一場限定）

- 對手 WIL **+5** / SPD **+3** / 憤怒槽全滿
- 視覺：對手 slot 紅光 + 「🔥 復仇」icon

### 5.5.3 跟「曾被斷手腳」對手重逢

如果對手身上有 `severed:<部位>` 標記（被你斷過手腳的）：
- 開場對白追加：「這隻【手 / 腿】——是你的。今天我要拿你的還回來。」
- buff 額外加成：對手 ATK +10%（仇恨爆發）
- 提高陰招觸發機率 +20%

## § 6 玩家防禦（AGI + 戒心重 + 神經質）

### 6.1 防禦率公式

```
基礎防禦 = AGI × 1.2%（AGI 50 → 60%）

+ wary 戒心重     → +25%
+ neurotic 神經質 → +15%
+ unbreakable 不屈 → +5%
+ fortunate 福星  → +10%（運氣使然）

封頂 85%（永遠 15% 機率被偷襲、保留戲劇）
```

### 6.2 實際被陰招機率 = Trigger × (1 − Defense)

| 玩家狀態 | Trigger | Defense | 實際被陰招 |
|---|---|---|---|
| Day 30、fame 25、AGI 20、無 trait、無大仇 | 0% | 24% | **0%** |
| Day 60、fame 60、AGI 30、3 個對手 grudge=5、無 trait | 30% | 36% | **19%** |
| Day 80、fame 90、AGI 35、cruel、5 對手記仇=8 | 75% | 42% | **44%** ⚠️ |
| Day 80、fame 90、乞丐（戒心重 + 神經質）、AGI 50、merciful、放過 5 人 | 0% | 85% | **0%** ✓ |

→ 殘暴強者**真的會被搞**、仁慈乞丐 = 全免疫。

### 6.3 立即報復 flag 不過 trigger、但仍過 defense

```
revenge_target flag = trigger 100% 強制觸發
→ 仍走 defense 判定（戒心重、AGI 等）
→ 防禦 85% 仍可能擋下 15%
```

## § 7 敵方陰招菜單

> 鏡像玩家的赫克特服務（[blackmarket.md § 1](blackmarket.md)）。
> 敵方對玩家發動的陰招、形式跟玩家對敵方一樣。

### 7.1 普通陰招（個人 grudge 5-11、faction 8-14）

| 陰招 | 觸發時機 | 效果 | 來源敘事 |
|---|---|---|---|
| 🍞 **食物被下毒** | 派遣前一晚 | 出場 HP -15% / WIL -5 | 廚房有可疑人影、梅拉沒在 |
| 📢 **謠言被散播** | 派遣前一晚 | 主人好感 -3、玩家心情 -10 | 訓練所內外都聽得到 |
| 🤜 **假陪練偷襲** | 訓練時段 | 受傷一次（部位擲骰）| 對手訓練所派來的「黑手」 |
| 🪙 **偷竊嫁禍** | 場內事件、Day 25+ | fame -5、主人疑心 +5 | 你床下被人塞了不該有的東西 |

### 7.2 致命陰招（個人 grudge ≥ 12、立即報復 flag 同步解鎖）

| 陰招 | 效果 |
|---|---|
| ☠ **慢性毒** | 你睡前被人摸過水壺、X 天後（5-10）發作 → 重傷 + ailment |
| 🗡 **暗殺嘗試** | 夜晚事件、有人偷襲你回鋪位、戰鬥（你+1 vs 1-2 殺手）|
| 🩸 **鋼鞭** | 公開場合刻意打傷你（不殺、但留疤）|

### 7.3 失敗效果

陰招過了你的防禦判定 → 失敗。
**敵方不知道你擋下了**（增加玩家「我贏了卻不知為何沒事」的迷霧感）。
但訓練所內部會留下蛛絲馬跡（可選 storyReveal「你發現水壺有怪味、但你今天剛好沒喝」）。

## § 7B 戰中陰招菜單（戰鬥內事件）

> 跟 § 7 戰前陰招對等、發生在戰鬥**進行中**。
> 每場戰鬥**最多 1 次**陰招（不會連發）、看 personality 決定選哪個。

### 7B.1 陰招清單

| 陰招 | 觸發時機 | 命中效果 | 失敗效果 |
|---|---|---|---|
| 🏖 **撒沙** | 第 3-5 回合 | 玩家 ACC **-50%** 持續 **8 回合** | 對手 ACC -10%（手忙腳亂）3 回合 |
| ☠ **暗藏毒匕** | 對手第一次命中你時 | 中毒 5 回合（HP -3 / stamina -2 每回合）| 對手暴露毒武器、fame -3 |
| 🩸 **暗器投擲** | 玩家 HP < 50% 時 | 受傷一次（部位擲骰）| 對手體力 -10（自損） |

### 7B.2 對手 personality 決定選哪個

| Personality | 偏好陰招 | 為何 |
|---|---|---|
| `cruel` | 暗藏毒匕 / 暗器 | 想你死、不只贏 |
| `impulsive` | 撒沙 | 情緒型、抓到啥用啥 |
| `opportunist` | 休息室埋伏（見 § 7C）| 不在台上動手、保留可信度 |
| `prideful` | 多半不用陰招（有失身份）；用了必選最狠（暗藏毒匕）| 自尊高 |
| `patient` | 暗藏毒匕 | 精算過、一擊必殺 |

### 7B.3 觸發頻率

```
基礎觸發率 = 個人 grudge / 50 × 100%（上限 60%）
+ revenge_target_<oppId> flag → 強制 100% 觸發（仍走 § 6 防禦判定）
```

### 7B.4 陰招對白（命中 / 失敗）

#### 撒沙範例

**命中**：
```
對手：「吃這個！」
（他抓起一把沙——朝你眼睛撒過來。）
[shake-big]
主角：（——可惡看不到了。）
[玩家 ACC -50%、8 回合]
```

**失敗**（玩家防禦過）：
```
（他朝你撒了一把沙——）
（你側頭、剛好閃開。）
主角：（看見了。下作。）
[對手 ACC -10%、3 回合]
```

#### 暗藏毒匕範例

**命中**：
```
（他刀劃過你左肩——一陣異常的麻。）
主角：（這混蛋⋯⋯刀上有東西。）
[玩家中毒 5 回合]
```

**失敗**：
```
（他的刀劃過你護甲、你聞到怪味。）
主角：（毒。他在刀上抹了毒。）
[對手暴露、fame -3]
```

### 7B.5 陰招命中後對手反應（依 personality）

| Personality | 對白 + 額外效果 |
|---|---|

## § 8 玩家對自己人下陰招的後果

> 你提到的：未來如果開放「玩家對自己訓練所隊友下陰招」（例如要長官設陷阱、買通陪練弄傷某隊友）—— **要有明確後果**。

- **在場其他隊友 -10 affection**（他們不知道是你做的、但氣氛變調）
- **整體訓練所信任度 -5**（背景角鬥士 mumble 變調）
- **觸發 storyReveal 機率 ×0.5**（隊友變得不愛跟你聊）

### 8.2 被陰招的隊友的反應（看好感）
