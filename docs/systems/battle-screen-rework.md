## 規範

### 左右屬性欄（各 20%）
- 心情臉 💚🔵🟣
- 六維（STR / DEX / CON / AGI / WIL / LUK）
- 派生（ATK / DEF / SPD 等）
- 裝備摘要（武器 / 護甲）
- 特性列表（戰鬥相關的）
- 下方 `▼ 切換秀` 下拉 — 3v3 時可選哪個角色的屬性
- 同一個 NPC 只能在一個 slot（不重複）
- 主人指定的 slot 無法換（鎖定）

### 中央欄（60%）從上到下

1. **戰鬥時間條**（整場計時器，一條）
2. **Stage**（arena00 背景）
   - 玩家方 3 slot（1v1 只用中間）
   - 中間：攻擊視覺 + 對話泡泡
   - 敵方 3 slot
3. **主人觀眾區**（塔倫 / 阿圖斯 / 侍從，依 forcedNPCs）
4. **攻擊按鈕 4 格**
5. **戰鬥日誌**（獨立，不跟主畫面日誌共用）

### 攻擊按鈕四格
| 格 | 內容 |
|---|---|
| 1 | ⚔ 普攻 |
| 2 | ⚡ 技能1（gauge 滿時的招式：瞬刺/血怒/盾牆反擊）|
| 3 | 🛡 技能2（情境技：強架/斬首/饒恕）|
| 4 | ▶ 自動（自動戰鬥 toggle）|

### 每人 slot 設計
```
┌──────────┐
│ [頭像/首字] │
│  名字       │
│  HP: ▓▓▓░  │   ← HP 條
│  ATB: ▓▓░ │   ← ATB 條
└──────────┘
```

### 對話泡泡
攻擊時 stage 中央或 slot 上方跳出：
- 「看招！」「給你嘗嘗！」「呵！」（攻擊）
- 「閃得好！」「該死⋯」「啊啊啊！」（防守）
- 通用詞庫（可未來擴充個性化）

### 攻擊動畫
攻擊方 slot **lunge**（往目標方向撲）+ 目標 slot **shake**（震動）
→ 沿用現有 `bt-lunge-right` / `bt-lunge-left` / `bt-hit-shake` CSS

### 心情臉色
- 💚 綠 = 心情 ≥ 60 / 自信
- 🔵 藍 = 心情 30-60 / 專注
- 🟣 紫 = 心情 < 30 / HP 低 / 狂怒

**純視覺，不影響戰鬥數值**。
敵人也動態變色（血量低變紫）。

### 3v3 邏輯
- Slot 位置：左 / 中 / 右
- 1v1 時：只用中間 slot
- 3v3 時：三個都用
- **強制 slot**（主人指定）不可換
- **自由 slot** 可透過下拉切換
