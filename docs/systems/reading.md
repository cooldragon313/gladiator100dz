# 讀書系統（Reading System）

> 讀書 = 普世成長系統，不只是「拆傻福」的工具。
> 透過書本累積**見識**、獲得特性/技能/裝備配方/任務線索。
> 設計日期：2026-04-19
> 實作對應：Phase 1.5 起分階段上線

---

## 核心哲學

1. **讀書是普世機會，不分出身** — 所有 origin 都能讀書，只是**入口機會不同**
2. **書本 = 故事觸發器** — 書不只給數值，而是解鎖事件鏈
3. **不主動打破百日祭** — 再大的陰謀、再珍貴的藏寶，都不能提前終止 Day 100 角鬥士身份
4. **讓玩家決定傻福命運** — 讀書會影響傻福，但透過玩家**持續的選擇**而非機制強加
5. **稀缺資源管理** — 書櫃有限容量，玩家必須在「拿到新書時」做取捨

## 五大書本分類

| 類型 | 讀完效果 | 累積見識？ | 是否影響傻福？ |
|---|---|---|---|
| **識字本** | 見識 +1~3，基礎識字 | ✅ 主力 | ✅ 推進傻福衰退 |
| **傳記 / 哲學** | 獲得特性（勤奮 / 堅忍 / 狡智 / 明辨 / 勇敢 等） | ✅ 小量（+1） | ✅ 推進傻福衰退 |
| **技能秘術** | 永久獲得戰鬥技能（主動 / 被動）| ❌ 不累積 | ❌ 不影響傻福 |
| **藍圖** | 裝備配方 / 藥劑配方 → 觸發鍛造師 / 醫生事件 | ❌ 不累積 | ❌ 不影響傻福 |
| **藏寶圖 / 秘密** | 地點解鎖 / 陰謀任務鏈（側線） | ❌ 不累積 | ❌ 不影響傻福 |

### 設計中樞：**文字書 vs 非文字書**

- **文字書（識字 + 傳記）** → 吸收知識 → 推進見識 → 影響傻福
- **非文字書（技能 + 藍圖 + 地圖）** → 肌肉記憶 / 解讀符號 / 空間記憶 → 不推進見識

**意義**：傻福玩家可以安心讀技能書、鍛造藍圖、藏寶圖變強，而不失去福氣。
這給玩家策略選擇，不是被動承受。

## 見識（discernment）數值

### 累積與效果

| 見識 | 訓練 EXP 加成 | 累積條件 |
|---|---|---|
| 0 | ±0 | 預設 |
| 3 | **+10%** | 讀 1~2 本文字書 |
| 5 | **+20%** | 讀 3 本文字書 |
| 8 | **+35%** | 讀 5 本文字書 |
| 12 | **+50%** | 讀 8 本以上文字書 |

- **永久 buff**，時間不會消退（知識不會忘）
- 存在 `player.discernment`，隨讀書事件累加
- 每檔閥值跨過時跳 addLog 提示（「你感覺思路清晰了一些」）

### 見識 × 讀書速度（正反饋）

```
讀書所需晚數 = 基礎晚數 × (1 - 見識 × 0.03)
```

| 見識 | 速度倍率 | 讀 6 晚書變為 |
|---|---|---|
| 0（傻福 / 文盲）| ×1.00 | 6 晚 |
| 3 | ×0.91 | ~5.5 晚 |
| 8 | ×0.76 | ~4.5 晚 |
| 15 | ×0.55 | ~3.3 晚 |
| 20（極博學）| ×0.40 | ~2.4 晚 |

**敘事自然**：
- 傻福玩家看書：「字好多⋯看不懂⋯」慢慢磨
- 博學玩家看書：翻兩下就懂了

### 見識門檻 lock（某些書需要最低見識才能讀）

| 書類 | 見識門檻 | 傻福玩家（見識 0）能讀？ |
|---|---|---|
| 識字本 | 無 | ✅ 可讀（推進清醒）|
| 傳記 | **≥ 3** | ❌ 需先累積基礎見識 |
| 技能秘術 | 無（肌肉記憶）| ✅ 可讀 |
| 藍圖 | **≥ 5** | ❌ 機關圖看不懂 |
| 藏寶圖 / 秘密 | **≥ 8** | ❌ 密文 / 文字暗號全封鎖 |

**設計對稱**：
- 傻福玩家（低見識） → 技能書特攻，走武力路線
- 博學玩家（高見識） → 藍圖 / 密文特攻，走陰謀 / 裝備路線
- 中道玩家 → 兩邊都沾，最全能

## 書櫃系統

### 容量 5 本

- 玩家同時最多持有 **5 本未讀書**（代表奴隸藏書有限）
- 第 6 本進來時 → 彈出選擇「丟棄哪一本」
- 每本書讀完會從書櫃消失（消耗品）

### 資料結構

```js
player.bookshelf = [
  { id: 'children_reader', progress: 2, nights: 6 },   // 讀了 2 晚，總共需 6 晚
  { id: 'general_memoir',  progress: 0, nights: 8 },
  ...
]
player.discernment = 5
player.focusBookId = 'children_reader'  // 專心書（可 null）
player.readBooks    = ['children_reader', 'battle_scroll_v1']  // 已讀清單
```

### 睡前讀書自動觸發

每晚就寢前自動觸發「睡前讀書」事件：

1. 書櫃為空 → 不觸發
2. 書櫃 1 本 → 直接讀該本，進度 +1
3. 書櫃 ≥ 2 本且設了 `focusBookId` → 讀專心書
4. 書櫃 ≥ 2 本未設專心 → **隨機抽一本**讀
5. 書櫃 ≥ 3 本且未設專心 → 套用「貪多嚼不爛」debuff（進度 ×0.7）

### 專心書機制

- 玩家在角色頁「書櫃」區塊可以標記一本為**專心書**
- 專心書的進度每晚 ×1.0（滿速）
- 其他書不會被讀，但不會消失（等專心書讀完自動輪替）
- 可以隨時更換專心書

### 貪多嚼不爛（Distraction Debuff）

| 書櫃本數 | 未設專心時進度倍率 |
|---|---|
| 1 | ×1.0 |
| 2 | ×1.0 |
| 3 | ×0.7 |
| 4 | ×0.5 |
| 5 | ×0.4 |

**設計用意**：鼓勵玩家**主動選擇專注**，而非囤積所有書。拿到新書時要做取捨。

## 讀書入口差異化（由 NPC / 事件 / Origin 驅動）

### 出身驅動的書源

| Origin | 主要讀書路徑 | 主動度 | 觸發條件 |
|---|---|---|---|
| **貴族** | 主人塔倫主動提議培養「管帳奴隸」 | 高 | Day 10+ / 主人好感 40+ |
| **農家子弟** | 醫生老默（好感 50）或卡西烏斯教識字 | 中 | Day 40+ |
| **手工匠** | 卡西烏斯看出你有手藝根底，贈送《鐵匠筆記》 | 中 | Day 30+ / 卡西烏斯好感 40+ |
| **罪犯 / 乞丐** | 幾乎沒人教，靠撿殘卷或黑市 | 低 | Day 50+ 機率觸發 |
| **沒落騎士** | 起手《家傳劍譜》+ 識字能力 | 起手 | Day 1 |
| **信徒** | 神殿事件 / 聖典抄本 | 中 | Day 35+ / 觸發神殿事件 |
| **賭徒** | 黑市掮客遇見，販售禁書 | 低 | Day 40+ / 金錢門檻 |

### 特性驅動的書源

| 特性 | 觸發 | 說明 |
|---|---|---|
| **勤勉（diligence）** | 書櫃有書時每晚額外 30% 機率觸發「睡前翻書」小事件 | 勤奮玩家自發性讀書 |
| **識字（literate）** | 讀書基礎速度 ×1.2 | 貴族 / 沒落騎士 origin 自帶 |
| **傻人傻福** | 文字書讀書速度 ×1.0（無加成）+ NPC 額外警告對白 | 每到半醒閥值觸發敘事警告 |
| **神經質（neurotic）** | 讀書進度會倒退 1 的低機率事件（做惡夢忘內容）| 反映精神狀況 |

### 起手書（Origin 綁定）

| Origin | 起手物 | 效果 |
|---|---|---|
| 沒落騎士 | 《家傳劍譜》 | 讀完獲得「橫掃」主動技能 |
| 手工匠 | 《鐵匠筆記》 | 讀完獲得「武器保養」被動（武器耐久 +30%）|
| 信徒 | 《聖典殘頁》 | 讀完獲得「禱告」被動（mood 下限 +5）|
| 貴族 | 《貴族禮儀錄》 | 起手直接 +3 見識（已識字）|

農家 / 乞丐 / 罪犯 / 賭徒 **無起手書**，反映階級現實。

## 讀書事件流程

### 一般書本（普世流程）

```
Day N 玩家拿到書（事件 / NPC 贈 / 黑市購）
  → 加入書櫃（如滿 5 本，彈出丟棄選擇）
  → 書櫃 UI 顯示進度 0 / N 晚
  → 每晚睡前自動讀（+1 進度，套倍率）
  → 讀滿 N 晚 → 觸發「讀完」事件
    → 識字本 / 傳記：discernment +X，傻福推進檢查，可能獲得特性
    → 技能書：獲得技能，addLog 顯示
    → 藍圖：獲得配方 flag，觸發後續鍛造事件（N 天內）
    → 藏寶圖：獲得地點 flag，觸發後續探索事件鏈
  → 從書櫃移除
```

### 讀完事件 Stage 演出

用 `Stage.playEvent` 做小過場：
- **文字書**：油燈特寫 + 書頁翻動 + 「你讀完了 XXX」
- **技能書**：武者揮劍剪影 + 「你領悟了 XXX」
- **藍圖**：機關齒輪動畫 + 「你看懂了圖紙」
- **藏寶圖**：地圖展開 + 標記點閃爍 + 「你記住了路線」

## 傻福三階段交互

> 完整定義見 [traits.md](traits.md#傻人傻福漸進三階段)，這裡只列交互邏輯。

### 階段推進時機

讀完**文字書**後檢查 `player.discernment`：

| 見識 | 傻福階段 | 觸發敘事 |
|---|---|---|
| 0~4 | 完整傻福 | 無變化 |
| 5~9 | 半傻半醒 | NPC 警告「你變了」（卡西烏斯 / 老默）|
| 10+ | 清醒（傻福消失）| 最終敘事「你已不是那個孩子」 |

### 傻福玩家的 NPC 警告對白

- **卡西烏斯**（好感高）：「你一直笑嘻嘻的 — 再讀下去可能笑不出來了。還要繼續嗎？」
- **老默**（冷淡）：「你這種腦袋，讀書是好事也是壞事。自己決定。」
- **梅拉**（擔憂）：「孩子⋯⋯你是不是⋯⋯不一樣了？」

**技術實作**：
- 半醒觸發時（見識從 4→5），讀完書當晚追加 NPC 對白
- 清醒觸發時（見識從 9→10），當晚播放重大事件過場

### 傻福玩家**拒絕繼續讀**的路徑

在半醒狀態讀完任何文字書時，強制觸發選擇：

```
「你感覺腦袋越來越清楚。要繼續讀下去嗎？」
  [A] 「繼續，我想看見這個世界」 → 見識正常累積，最終變清醒
  [B] 「夠了，我這樣挺好」 → 書櫃所有文字書自動捨棄，見識凍結在當前，獲得「粗識文字」特性
```

「粗識文字」= 玩家的中間態終點：保留半醒的福氣減半效果，不再衰退。

## 與既有系統的整合

### D.15 跨系統整合清單

1. **好感度連動** ✅ — 書的獲取幾乎都靠 NPC 好感門檻
2. **特性共鳴** ✅ — 勤奮自發翻書 / 傻福 NPC 警告 / 神經質惡夢倒退
3. **故事揭露** ✅ — 卡西烏斯贈書觸發 storyReveal；老默醫書傳承
4. **資源物品** ✅ — 書本為新物品類別（`player.bookshelf`），有 storyTag:'book'
5. **Stage 演出** ✅ — 睡前讀書 / 讀完事件用 Stage.playEvent
6. **Origin 差異** ✅ — 7 出身書源與起手書完全不同
7. **觸發頻率** ✅ — 書為消耗品（讀完消失），見識永久累積，書櫃 5 本上限
8. **旗標鉤子** ✅ — `read_book_<id>` / `knows_blueprint_<id>` / `has_map_<id>` / `player.discernment`
9. **道德軸驅動** ⚠️ — 黑市禁書 / 毒術書讀完推進 mercy negative（讀完會想試試）

### 相關 Flags

| Flag | 寫入時機 | 用途 |
|---|---|---|
| `read_book_<id>` | 讀完一本書 | 避免再次分發 |
| `knows_blueprint_<id>` | 讀完藍圖 | 鍛造師事件讀取 |
| `has_map_<id>` | 讀完地圖 | 探索事件讀取 |
| `dullard_awakened` | 見識從 9→10 | 傻福清醒觸發點 |
| `refused_awakening` | 選擇 B 停留半醒 | 獲得「粗識文字」 |
| `literate_gifted_by_<npc>` | NPC 教識字完成 | NPC 故事記號 |

## 藏寶圖 / 秘密任務的「不破百日祭」守則

你在側線任務中可以：
- ✅ 揭發訓練師陰謀 → 換訓練師
- ✅ 主人兒子篡位 → 換主人或內鬥升級好感
- ✅ 主人暗殺計畫 → 警告主人暴漲好感 / 裝睡繼承新主
- ✅ 挖寶 → 獲得大量金錢 / 裝備
- ✅ 甚至密謀逃脫 → 劇情升溫，NPC 好感劇變

但你**不能**：
- ❌ 提前終止角鬥士身份
- ❌ 跳過 Day 100 百日祭儀式判定
- ❌ 直接離開羅馬城成為自由人（那是 Phase 3 劇情）

**通則：再大的側線陰謀都只影響周邊 NPC 命運，玩家該到 Day 100 就到 Day 100。**
