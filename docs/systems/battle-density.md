# 戰鬥密度規劃（100 天）

> 定義整個遊戲在 100 天內**會有多少場戰鬥**、分佈在哪些 Phase、每場戰鬥的性質。
> 討論日期：2026-04-18
> 最後更新：2026-04-18

---

## 總覽

預期**百日總戰鬥數：約 25-35 場**（視玩家選擇而定）。
類比：實況野球 / NBA 2K MyCareer — 除了訓練跟任務，動不動就有比賽。

---

## Phase 1（Day 1-30）：**小圈圈內戰鬥**（預計 5-7 場）

此階段**不出訓練所、不見外人**。所有戰鬥都在訓練所內部。
主人不會讓「沒訓練出屁」的投資品上競技場。

| 戰鬥類型 | 數量 | 例 |
|---|---|---|
| 長官測試 | 1 場（強制） | Day 5「沙洗」 |
| 隊友切磋 | 1-2 場 | 卡西烏斯 / 奧蘭 / 達吉邀請（可拒絕） |
| 隊友衝突 | 1-2 場 | 赫克托霸凌 → 升級實戰（可選） |
| 寵物馴服 | 0-1 場 | 抓老鼠（已做）/ 馴服小灰（失敗路線 = 打鬥） |
| 新人淘汰相關 | 0-1 場 | Day 10 衍生事件 |

### Phase 1 節奏
- Day 1-4：只有訓練 + 認識人
- **Day 5 沙洗** — 第一場大戰鬥（核心事件）
- Day 6-15：情緒恢復期 + 隊友事件浮出
- Day 12-20：第一次隊友切磋
- Day 15-25：赫克托衝突升級可能實戰
- Day 28-30：「考核期末」感，畢業前 NPC 關係浮出

---

## Phase 2（Day 31-100）：**戰鬥解放**（預計 20-28 場）

玩家有了初步名聲，主人開始賭你。

| 戰鬥類型 | 數量 | 觸發 |
|---|---|---|
| **主線固定戰** | 3 場 | Day 50 大型競技 / Day 75 宿敵會戰 / Day 100 萬骸祭 |
| **主人賭局** | 8-10 場候補、**預期 3-5 場**觸發 | 主人下注 → 強制戰鬥 |
| **隊友切磋** | 3-5 場 | 隊友實力成長後再邀請 |
| **衝突戰** | 1-3 場 | 赫克托（若 Phase 1 未打）/ 路遇挑釁 |
| **路人挑戰** | 3-5 場 | Day 50+ 外出任務遇強盜、刺客 |
| **野獸戰** | 1-3 場 | 陪主人打獵、困獸鬥表演 |
| **特殊路線** | 0-2 場 | 逃獄嘗試戰、隱藏 boss |

### Phase 2 節奏
- Day 31-45：賭局開始出現（主人覺得可以下注了）
- Day 40-50：主線之間頻率約 3-4 天一戰
- Day 50：**大型競技**（主線）
- Day 55-70：賭局密集期
- Day 75：**宿敵會戰**（主線）
- Day 80-95：可能有野獸戰、隱藏對決
- **Day 100 萬骸祭** — 一切結算

## 已決定的規則

### ✅ Phase 1 不出訓練所
- 符合「奴隸沒有主動權」哲學
- 玩家不信任主人、不信任世界，只有身邊幾個人
- 所有戰鬥都是「這個訓練所裡的人事物」

### ✅ Phase 2 戰鬥頻率 3-5 天一場
- 避免「一週無事」的空轉感
- 但也不到每天戰鬥（玩家需要訓練期）
- 訓練 → 賭局 → 恢復 → 下一賭局 → 主線 → 恢復 → 下一個

### ✅ 主人賭局 = 動態揭露
- 候補日固定（Day 35/42/48/55/62/70/80/88）
- 每個日 30-40% 觸發機率
- **觸發後立刻用 NPC 對話揭露**（侍從傳話）
- 百日條變「已揭露」狀態
- 參考 `docs/systems/timeline.md`

### ✅ 戰鬥都不可逆
- 輸了會有後果（重傷、名聲掉、主人評分掉）
- **只有 Day 100 真正會死**
- 其他戰鬥輸了 → 挨鞭、暫時失能、被瞧不起、流失隊友好感
