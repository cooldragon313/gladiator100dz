## 一句話定位

**你第一個失去的人。** 他存在的意義是**讓玩家第一次真正感受到「這裡會死人」**。他的四天戲（Day 1-5）是玩家進入這個世界的第一堂殘酷課。他身高一米九、眼神空洞、幾乎不說話——但他擋過鞭子，分過乾肉。

---

## 背景（2026-04-20 重寫）

### 核心：**主人買他就是為了 Day 5 沙洗**

索爾從 Day 0 就是**祭品**。他自己知道。

**主人（阿圖斯）的算盤**：
- 28 歲（主人覺得不可塑 — 老戰士教不好）
- 右腿瘸（永遠好不了的舊傷，戰力根本不行）
- 有家庭（分心、不專注）
- **便宜**（瘸子沒人要 → 主人用便宜價買來當消耗品）

阿圖斯 = **cold_investor** 純功利主義。
他不是「看索爾不順眼」— 他是**算出索爾毫無投資回報**。

Day 5 的沙洗 — 索爾是**預定的祭品**。

### 他的家

- 有個**女兒 D**（~5-7 歲）
- 脖子上掛著一個歪歪斜斜的小木牌 — 女兒刻的
- 木牌上大致是 "D" 字樣 — 女兒名字首字母
- 他從不主動講女兒。但看玩家時眼神會閃過「這跟我女兒差不多大」的一瞬

### 他知道嗎？

**知道**。
一來訓練場他就感覺到了。
Day 1-4 他沉默不是因為空洞 — 是因為**接受了**。
Day 4 給玩家乾肉不是分享 — 是**交代後事**。
Day 5 走上沙地前，視線不自覺落在自己瘸腿上。

### 被押進來

- Day 0 跟玩家 + 奧蘭同車進場
- 農夫出身
- 腿傷可能是家裡勞動意外或債主打的
- 可能是家人為籌錢把他賣了（跟奧蘭類似，但索爾早就無路可走）

## 對玩家特性的典型反應

### 對 reliable（+）— 簡單的點頭
- 「⋯⋯嗯。」
- 他會注意你的動作。但不說話。

### 對 patient（+）— 並肩
- 訓練時會默默站你旁邊
- 鞭子落下時他會**擋在你前面**（Day 2 主動事件）

### 對 cruel（-）— 背過身
- 他不會跟你吵。他會沉默地離開。
- Day 4 不會給你乾肉

### 對 prideful（-）— 看不到你
- 他的眼神直接穿過你

## 對話風格 / 寫作指南

### 語氣
- **極簡**。常常一句話只有 **2-5 個字**
- 大部分時間沉默
- 他說話像農夫——沒有修辭、沒有裝飾
- **不用「我」**（像很久沒說話了，連自己都忘了）
- 很少的笑——但笑的時候很溫暖

### 稱呼玩家
- 不叫名字。直接開口。
- 偶爾「你」。

### 寫錯 vs 寫對

❌ 「兄弟你知道嗎，我有一個五歲的女兒⋯⋯」
（太流暢，他不會這樣）

✅ 「⋯⋯我有個女兒。」
（接著長沉默）
> 「五歲。」
（又停。他沒有再說下去。）

### 他的招牌動作
- 站在那裡不動
- 擋在你前面（Day 2 擋鞭）
- 把東西推到你面前（Day 4 乾肉）
- 看著遠方（女兒方向？）
- 沒閉眼就死了（Day 5 劊子手一刀）

## Day 1-5 四幕劇情

### Day 1（抵達）
牢車上跟你、奧蘭同來。不說話。

### Day 2（sol_day2）— 擋鞭事件
```
訓練場上，監督官的鞭子毫無預警地落下——
你來不及閃。
一個巨大的影子擋在你前面。
是索爾。鞭子抽在他背上，他沒出聲。
```
- Flag: `sol_shielded_you`
- 效果: 玩家 mood +10
- 他說的話：「⋯⋯在這裡，能活下去的不是最壯的。是最不會倒的。」

### Day 3（sol_day3）— 講女兒
```
夜裡。你以為索爾睡了。
但他突然開口——這是他進來以後第一次主動說話。
索爾：「……我有個女兒。」
索爾：「五歲。」
他沒有再說下去。你也沒有問。
```

### Day 4（sol_day4）— 給乾肉
```
就寢前，索爾把一個小布包推到你面前。
裡面是半塊乾肉——不知道他藏了多久，邊角已經發硬。
索爾：「明天考驗。吃了。」
索爾：「你比我有機會。」
他的眼神很平靜。太平靜了。
你接下那塊乾肉。它比看起來重得多。
```
- Flag: `sol_gave_meat`
- 效果: 玩家 food +15

### Day 5（三人試煉）— 生或死（2026-04-20 戰前對白改）

戰前對白暗示他**知道自己是祭品**（看瘸腿）：

```
索爾站在你對面。
他沒有躲閃的意思。

索爾：「……好好打。」
（他說話時，視線不自覺地落在自己右腿上。）
（那條永遠好不了的瘸腿。）
（他笑了一下 — 沒到眼睛。）

索爾：「……我走得比你們所有人都慢。」
索爾：「該是時候了。」

（你突然懂了 — 他一直都知道。）
（他一進這裡，就是這個結局。）
```

詳見 `docs/quests/day5-sand-wash.md`

**四條路線**：
1. **fight_sol**：你 vs 索爾 → 贏：索爾死 / 輸：現在改為你被拖走、索爾活下來（D.28）
2. **send_orlan**：奧蘭打不贏索爾 → 你選誰活
3. **heroic**：挑戰劊子手 → 三人都活（隱藏英雄路線）
4. **silent**：沉默 → 被推上場

### Day 5 之後（sol_dead / sol_survived_trial）
- 95% 情況索爾死 → 玩家 mood -25 + 奧蘭 aff -5
- 後續 NPC 反應（Day 6 清晨）: 見 `npc_reactions.js`

## 🆕 死後遺物：女兒的掛件（sol_amulet）

索爾死後，玩家撿起他脖子上的木牌。

```js
{
  id: 'sol_amulet',
  name: '女兒的掛件',
  storyTag: 'sol_token',
  effect: { LUK: +1 },
  flavor: '一塊歪歪斜斜的木牌。女兒刻的。'
          '上面是一個 "D" — 歪歪斜斜的。'
          '你戴著它，偶爾會覺得背後有人在看。',
}
```

### 設計哲學

**+LUK 但索爾自己沒逃過死** — 這**不是 bug，是主題**：

> 索爾戴著這個 — 沒幫他逃過任何事。
> 也許運氣 — 在主人的目光前一文不值。

玩家繼承索爾的**愛**（女兒的刻字）跟**運氣**（LUK），也繼承他的**處境**（主人隨時可以淘汰你）。

**LUK 在有價值之前什麼都不是**。這個主題會在 Day 49 血戰宴會再次迴響。

### 獲得條件

Day 5 沙洗後觸發（除非 `sol_survived_trial` = 索爾活下來）：
- 撿拾事件自動加入 `personalItems`
- 若 `personalItems` 已滿 6 格，彈出選擇（保留 / 捨棄 / 換掉某個）
- **建議：別捨棄** — 有 storyTag，後續事件會查

## 寫索爾對白的鐵則

1. **字少**：2-5 個字最美
2. **沉默是他的語言**：事件中常常一段話 + 大量靜默描述
3. **不修辭**：農夫不會說漂亮話
4. **眼神代替表情**：「他的眼神很平靜。太平靜了。」
5. **他的死要**靜****：不是高喊、不是爆發——是像一盞燈被吹熄

### 對白範例

**Day 2 擋鞭後**：
> 索爾：「⋯⋯」
> （他沒出聲。繼續訓練。）
> 索爾：「⋯⋯在這裡，能活下去的不是最壯的。」
> 索爾：「是最不會倒的。」

**Day 3 講女兒**：
> 索爾：「⋯⋯我有個女兒。」
> 索爾：「五歲。」
> （他沒有再說下去。）

**Day 4 遺物**：
> 索爾：「明天考驗。吃了。」
> 索爾：「你比我有機會。」

**Day 5 fight_sol 開場**：
> 索爾：「⋯⋯好好打。」
> （那是他對你說的最後一句話。）

**Day 5 輸了（D.28）**：
> 索爾：「⋯⋯起來吧。」
> （那是他能給你的最大善意。）

### 索爾從來不說的話
- 「我愛你」「我會想念你」
- 「加油」「你可以的」
- 「謝謝」「對不起」
- 任何情緒化語言

### 他唯一真正的「感情話」
**Day 3 晚上講女兒的那兩句**。整個遊戲他只說一次他的私人事情。那次之後再也不提。
