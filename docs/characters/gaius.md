## 一句話定位

**橫向張力的源頭**。維努斯場規格高於阿圖斯場、近 5 年大會冠軍幾乎全包 → 阿圖斯一直在「被壓著」的位置 → 阿圖斯越來越想用陰招翻身 → 玩家被夾在兩個老人 30 年恩怨之間。

## 背景（CANON 級事實）

### 兩家對比

| 項目 | 阿圖斯場（玩家在這）| 維努斯場（蓋烏斯）|
|---|---|---|
| 規模 | 中型偏小（40 人） | 中大型（60-70 人）|
| 主軸 | 雜燴混合 | **力量主導 — 清一色 STR 帥哥** |
| 裝潢 | 樸素堪用、木樁石塊 | **大理石訓練場、鐵鑄重器、進口教練** |
| 戰績 | 中下、長期被壓 | **近 5 年大會冠軍幾乎全包** |
| 主人心態 | 阿圖斯：忌妒、不服、想用陰招翻身 | 蓋烏斯：得意、輕蔑、施捨式友好 |
| 玩家觀感 | 「我被分配到輸家窩」 | 「對面那群混蛋」 |

### 跟阿圖斯的關係（30 年）
- **童年好友** — 一起長大
- **18 歲後**變成終身競爭對手
- 公開姿態：「親兄弟」、宴會擁抱、互相敬酒
- 私底下：碎念、酸言、互看不順眼
- **共同利益時毫不介意合作**（兩家曾聯手坑過第三家）
- **對方倒楣時絕不出手**、可能落井下石

### 童年共同史素材池（5 條）

> 用來餵後續對白、揭露事件、不是一次倒出來。

| 階段 | 事件 | 後續影響 |
|---|---|---|
| 14 歲 | 阿圖斯搶了蓋烏斯喜歡的女孩 → 那女孩後來變成阿圖斯妻（已過世）| 蓋烏斯心結、看到 Livia 會閃眼 |
| 18 歲 | 蓋烏斯陰阿圖斯一場馬賽 → 阿圖斯爸氣死 | 阿圖斯心結、提到馬會冷臉 |
| 25 歲 | 兩家聯手坑了第三鄰居（後來那人當大臣）| 共同把柄、互相牽制 |
| 35 歲 | 阿圖斯老婆死、蓋烏斯「弔唁」順手買阿圖斯一塊地 | 阿圖斯酒後會罵 |
| 現在 | 兩人各擁訓練所、競技場排名互咬 | 賽前下注、賽後各種小動作 |

### 跟新領主提圖斯
- 拼命奉承（提圖斯剛上任、有實權）
- 但提圖斯軍人本性看不慣油腔滑調 → 反而對阿圖斯比較好
- 蓋烏斯察覺後越發焦躁、想用更狠的手段壓住阿圖斯場

## 對玩家特性的典型反應

### 對 prideful（玩家驕傲）— 欣賞、想挖角
- 「⋯⋯阿圖斯這小子、養出來個有種的。」
- 玩家好感降、但蓋烏斯個人對玩家好感升

### 對 reliable（玩家可靠）— 想利用
- 「⋯⋯這個鬥士有用。」
- 會找機會要玩家「替他帶話」（暗中傳信給阿圖斯）

### 對 cruel（玩家殘忍）— 害怕但裝不
- 「⋯⋯阿圖斯啊⋯⋯你這寵物、也太兇了。」
- 公開場合誇獎、私下會跟阿圖斯說「你管管那個瘋子」

### 對 merciful（玩家仁慈）— 嘲笑
- 「⋯⋯訓練場不需要慈悲。」
- 賽後會跟阿圖斯耳語「⋯⋯你這個太軟了」

## 對話風格 / 寫作指南

### 語氣
- **油滑、奉承、底下藏刀**
- 跟阿圖斯：「親兄弟」假腔調、但話裡有刺
- 跟提圖斯：拼命奉承、講話拉長尾音
- 對玩家：施捨式的稱讚、不直接看玩家眼睛
- 笑的時候**笑容滿臉、眼神冷**
- 喜歡用「⋯⋯啊」、「⋯⋯呢」這種拖音

### 稱呼
- 對阿圖斯：「阿圖斯啊」（拖長、不真誠）
- 對提圖斯：「領主大人」、「您」（恭敬）
- 對玩家：「這位鬥士」、「阿圖斯這寵物」

### 寫錯 vs 寫對

❌ 「混帳！我恨你阿圖斯！」
（蓋烏斯不會直接撕破臉、太掉價）

✅ 「⋯⋯阿圖斯啊⋯⋯這事辦得不夠周到呢。」
（拖音、笑著、底下是刀）

❌ 「我要殺了你！」
（蓋烏斯陰人不直接動手）

✅ 「⋯⋯德基烏斯、你晚上去『拜訪』那位一下。」
（用人、不沾手）

### 招牌動作
- 「他舉杯、笑著、酒沒喝。」
- 「他拍阿圖斯的肩、力道有點大。」
- 「他笑著看你、眼神飄到別處。」
- 「他低聲跟手下講話、聲音傳不到台下。」
- 「他離開的時候、回頭看了阿圖斯一眼、笑了。」

### 一定要避免
- 不要演成漫畫式邪惡反派
- 不要當面說刻薄話（他都背後做）
- 不要熱血、不要激動
- 他應該讓玩家覺得「這人很煩、但不能直接打他」

## 招敵變友機制（觸發蓋烏斯暗殺鏈）

> 詳見 [arena-events-roster.md § 5](../quests/arena-events-roster.md)

玩家若成功招走凱里烏斯 / 諾克斯 → 蓋烏斯震怒 → 暗殺鏈啟動：

| 階段 | Day 距觸發 | 內容 |
|---|---|---|
| 食物下毒事件 | +5 天 | 食物裡有人下毒、廚娘梅拉好感夠 → 提早警告 |
| 食物下毒追查 | +6 天 | 玩家追查毒源、可能是女僕 / 同訓練所敵視玩家的人 |
| 第一次暗殺 | +15 天 | 夜間刺客闖入訓練場 |
| 競技場下毒 | +25 天 | 比賽中對手武器塗毒 |
| **黑爪登場** | +35 天 | B5「黑爪」boss、Day 80 強制戰 |

避免方式：不撬人 / 撬人後賄賂蓋烏斯 500 金錢 / 蓋烏斯被殺失勢

## CANON 鐵則（寫蓋烏斯對白前必看）

1. **30 年青梅竹馬 + 競爭對手** — 不要演成純粹陌生敵人
2. **公開親兄弟、私下小動作** — 永遠不能當面撕破臉
3. **維努斯場規格更高、贏在前面** — 蓋烏斯是「贏家位置」、不是輸家報復
4. **現在巴結提圖斯但被冷待** — 焦躁感從這裡來
5. **油滑奉承、底下藏刀** — 講話永遠笑著、永遠拖音
6. **不直接動手陰人** — 都用德基烏斯 / 黑爪 / 別人

## 跟其他線的整合

- **阿圖斯線**：蓋烏斯是阿圖斯的「永遠贏不了的對手」— 阿圖斯越想用陰招翻身、玩家越被推去做髒事
- **提圖斯線**：蓋烏斯巴結被冷待、Day 80 領主夜宴上會看出端倪
- **凱德線**：凱德是另一家訓練所的、不是蓋烏斯的人 — 兩條線基本不交叉
- **大反撲**：Day 100 反撲時、蓋烏斯派的角鬥士會跟衛兵一起對付玩家陣營（但只是輔助、不是主力）
