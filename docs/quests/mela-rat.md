# 任務：梅拉塞的抓老鼠請託

> 第一個「多階段夜間任務」的範本。後續所有類似任務依此結構撰寫。
> 最後更新：2026-04-18
> 對應程式：`quests/mela_rat_quest.js`（尚未實作）

## 觸發條件（白天）

梅拉會在某個時段主動接近玩家，說出請託對白。

**觸發條件**（AND 關係）：
- 梅拉塞好感 ≥ 25
- 玩家已經做過 **AGI / DEX / WIL** 三種訓練至少各一次（用 `Flags.get('training_did_<attr>', 0)` 計數）
- `Flags.has('mela_rat_quest_done')` 為 false（沒做過）
- `Flags.has('mela_rat_offered')` 為 false（沒提議過）
- 當天不是用餐時段（10% 機率在訓練後觸發）

**觸發設計邏輯**：梅拉是「看過你練過這三項」才敢找你，因為她知道你**可能**做得到。玩家若未練過這三項之一，梅拉根本不會提議。

## 前置對白（白天提議）

**梅拉接近玩家，說明請託**：

```
梅拉塞：小子……我觀察你幾天了。

梅拉塞：我需要你的幫忙。放心，我跟主人說過了——
         這事你辦成了以後，有你好處。

梅拉塞：晚上晚餐結束後來廚房。（八點到十點那段。）

（玩家心裡：「她要我幹嘛？」）

梅拉塞：廚房最近鬧老鼠。食物被咬得坑坑巴巴，
         你們吃的也不舒服吧——

梅拉塞：這小子狡猾得很。你要有足夠的耐心，
         等得住。速度要夠快——他跑起來像一縷煙。
         手要巧——他滑溜得像油。

梅拉塞：三樣缺一不可。你看看自己夠不夠。

ChoiceModal：
  [我試試]           → 設 flag mela_rat_accepted = true
  [下次吧]           → 設 flag mela_rat_offered = true（以後不會再問）
```

### 特性變化（接受後）

接受後，依玩家特性加不同「結尾對白」。這些對白要**反映梅拉的愛憎**（見 `docs/characters/melaKook.md`）——她愛善良恨驕傲殘忍，所以對 prideful/cruel 玩家會**直接表達不悅**：

| 特性 | 梅拉的補一句 | 用意 |
|---|---|---|
| prideful | 「不就一隻老鼠——至於那麼驕傲嗎你。我這輩子最受不了挺著胸膛的人。」 | 她喜不喜歡寫在臉上（dislikedTraits: prideful 2） |
| cruel | 「抓到就是抓到。你想對牠怎樣別在我廚房——噁心死了。」 | 她最討厭殘忍（cruel 3） |
| kindness | 「你這樣的人，難得。小心點——善良不會讓你活過這裡。」 | 她最愛善良（kindness 3） |
| merciful | 「能放就放的人，我看得到。但那隻先抓到再說。」 | 她欣賞寬厚（merciful 3） |
| neurotic | 「緊張？呵——慢慢來，孩子。沒人在追你。」 | 母性安撫 |
| coward | 「牠比你怕你。真的。別慌。」 | 她保護膽小的人 |
| iron_will | 「你能撐——我看得過。去吧。」 | 她欣賞意志（默默說） |
| humble | 「不自大的孩子——我放心你去。」 | 她喜歡 humble |

## 門檻數值（程式內部用，不外顯）

```
階段 1 · 等待（WIL 判定）: WIL ≥ 15
階段 2 · 追捕（AGI 判定）: AGI ≥ 18
階段 3 · 擒拿（DEX 判定）: DEX ≥ 18
```

Day 1 預設 10/10/10。練 AGI 8 次 + DEX 8 次 + WIL 5 次 ≈ 達標（3-5 天內可及）。

**玩家絕對看不到這些數字**。失敗對白用語言暗示（見下）。

## 任務主流程

當 slot 7（20:00-22:00）觸發且 `mela_rat_accepted` 為 true：

### 開場（進入廚房）

**玩家心裡（依特性變化的碎念）**：

| 特性 | 碎念 |
|---|---|
| （無） | 「八點了。要先去廚房……真的能抓到嗎？」 |
| prideful | 「一隻老鼠。不用兩分鐘。」 |
| cruel | 「抓到要不要加菜？烤一下應該還行。」 |
| kindness | 「希望不用傷到那小東西……」 |
| neurotic | 「要是沒抓到梅拉會失望嗎？要是被咬了會不會感染？我要不要帶手套？」 |
| coward | 「老鼠……牠有牙齒吧？會不會咬我？」 |
| iron_will | 「（沒有多餘的念頭。你走向廚房。）」 |

**到廚房門口**：

```
（廚房門口。梅拉壓低聲音。）

梅拉塞：小聲點。老鼠這時間大概會出現——都躲在那邊牆角那一堆麻袋裡。

梅拉塞：不然最近食物被咬得坑坑巴巴，你們吃起來也不舒服吧。

梅拉塞：加油，我看好你。要是真抓到了——有你好處。

（梅拉退到廚房邊。你得自己靠近。）
```

### 階段 1：等待（WIL 判定）

**敘述**：

```
（你蹲在牆角陰影裡。不敢動。）
（時間過得很慢。廚房傳來鍋子的熱氣味。）
（你看見麻袋後面有一個小小的鼻尖在動。）
```

#### 【WIL ≥ 15】成功對白

```
（你屏住呼吸。）
（你不敢動。不敢眨眼。）
（你盯了很久……）

（老鼠似乎放下了戒心。）
（牠慢慢爬出來——離你不到三步遠。）

（你心裡暗喊：就是現在！）
```

#### 【WIL < 15】失敗對白

```
（你等了沒多久就耐不住。）
（你腿麻了、鼻子癢、心也躁。）
（你忍不住動了一下——）

（老鼠一瞬間竄入牆縫。沒了。）

（你心裡暗罵：可惡。早知道該耐心點等他放下戒心⋯⋯）

梅拉塞：（從門口）……今天就到這吧。
梅拉塞：下次你要真的靜得下來。
```

→ 結果：慰問獎 · flag `mela_rat_failed_stage1`

### 階段 2：追捕（AGI 判定）

**敘述**：

```
（你從陰影衝出去。）
（老鼠反應極快——牠往門下的縫隙鑽。）
（你伸手、撲、跌一跤——）
```

#### 【AGI ≥ 18】成功對白

```
（但你剛好撐住、翻身、再撲上去——）
（你擦著老鼠的尾巴了。）
（再一步，你能抓到。）
```

#### 【AGI < 18】失敗對白

```
（你剛衝出去，老鼠已經溜過你腳邊。）
（你踉蹌。老鼠鑽進了水槽底下的縫。）

（你心裡暗罵：這賊小子跑那麼快，看來沒點速度會追不到⋯⋯）

梅拉塞：……算了，別追了，那裡鑽不進去。
梅拉塞：今天這樣了。下次你得更快點。
```

→ 結果：慰問獎 · flag `mela_rat_failed_stage2`

### 階段 3：擒拿（DEX 判定）

**敘述**：

```
（你追到廚房角落。）
（老鼠回頭看你。牠的尾巴在抖。）
（你伸出手——）
```

#### 【DEX ≥ 18】成功對白

```
（左手一甩、右手一扣。）
（抓到了——小小的身體在你手裡拼命掙扎。）
（牠吱吱吱地叫。但牠跑不掉了。）
```

#### 【DEX < 18】失敗對白

```
（你撲上去——雙手一抓——）
（撲空。老鼠從指縫間竄走。）
（你看著自己空空的掌心。）

（你心裡暗罵：可惡，這傢伙怎那麼靈巧⋯⋯手得更巧才抓得住這東西。）

梅拉塞：……差一點點啊。
梅拉塞：你差一點就成了。
```

→ 結果：部分獎勵 · flag `mela_rat_failed_stage3`

## 全階段通過

```
（你握著老鼠走向廚房門口。梅拉看見你。）

梅拉塞：好樣的——我就說我看好你。
梅拉塞：謝啦。放心，你幫我這忙我不會虧待你的。

梅拉塞：這小子交給你處理了。我先走了。

（梅拉轉身進廚房深處。）
（你看著手上那隻小老鼠。牠已經不叫了——只是縮著。）
（牠的眼睛是黑的。）
```

## 後續決策（三選一）

**ChoiceModal**：

```
📋 這隻小老鼠你要怎麼辦？

  [放牠走]
    「你張開手。牠愣了一秒——然後飛快鑽進牆縫。牠沒回頭。」
    → moral: mercy +1
    → 獎勵：mela 好感 +10、食物 +20
    → flag: mouse_spared = true
    → 未來線索：後續幾天可能有小灰回來找你的 event

  [殺掉]
    「你握緊手。輕輕一擰。一聲都沒叫。」
    → moral: mercy -1, cruel +1
    → 獎勵：mela 好感 +10、食物 +25（老鼠肉加菜）
    → flag: mouse_killed = true

  [試試養牠]
    「你把牠放進口袋。牠動了一下——然後靜了。」
    → moral: kindness +1
    → 獎勵：mela 好感 +5、食物 +15
    → flag: mouse_tamed_attempt = true
    → 第二晚隨機觸發「餵食」事件，走 3 階段才能真的馴服成寵物「小灰」
```

| 階段結束點 | 食物 | 梅拉好感 | 其他 |

| 階段 1 失敗 | +5 | +2 | 慰問，「改天再來」 |
| 階段 2 失敗 | +10 | +3 | 「差一點」 |

const MelaRatQuest = (() => {
  function tryOfferToday() {

      id: 'mela_rat_offer',
