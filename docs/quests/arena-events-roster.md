## § 0. 設計核心人物：蓋烏斯（隔壁訓練所主人）

### 角色卡

| 欄位 | 內容 |
|---|---|
| 名字 | **蓋烏斯·維努斯**（已敲定 8-1） |
| 年齡 | 50+（與阿圖斯同齡） |
| 關係 | 阿圖斯**青梅竹馬 + 30 年競爭對手** |
| 訓練所 | 「維努斯場」（與阿圖斯場同城、長期壓著阿圖斯） |
| 公開姿態 | 「親兄弟」、宴會擁抱、互相敬酒 |
| 私底下 | 碎念、酸言、互看不順眼 |
| 利益對齊時 | 合作分贓毫不介意 |
| 對方倒楣時 | 絕不出手、可能落井下石 |
| **訓練所地位** | **維努斯場規格高於阿圖斯場、近 5 年大型賽事幾乎全包冠軍** |

### 兩訓練所實力對比（重要！整篇敘事張力來源）

| 項目 | 阿圖斯場（玩家） | 維努斯場（蓋烏斯） |
|---|---|---|
| 規模 | 中型偏小（40 人） | 中大型（60-70 人） |
| 主軸 | 雜燴混合（什麼都有） | **力量主導 — 清一色 STR 帥哥** |
| 裝潢 | 樸素堪用、木樁石塊 | **大理石訓練場、鐵鑄重器、進口教練** |
| 競賽戰績 | 中下、長期被壓 | **近 5 年大會冠軍幾乎全包** |
| 主人心態 | 阿圖斯：**忌妒、不服、想用陰招翻身** | 蓋烏斯：**得意、輕蔑、施捨式友好** |
| 玩家觀感 | 「我被分配到輸家窩」 | 「對面那群混蛋」 |

### 阿圖斯的「玩陰」傾向（從上述地位差衍生）

> 因為長期被壓 → 阿圖斯越來越想抄捷徑、玩陰招翻身

- 競賽前要玩家**下毒手斷對方派來對手的手腳**（道德選擇 — mercy + reliability 軸）
- 賽後可能要玩家**對外裝沒事、對內回報細節**（情報戰）
- 蓋烏斯陰阿圖斯/玩家時、阿圖斯反應不是保護玩家而是「**你怎麼會被陰**」（責怪）
- 玩家若拒絕做髒事 → 阿圖斯失望、好感 -、可能轉而依靠其他奴隸做
- 反過來：玩家主動建議陰招 → 阿圖斯好感 + 但 reliability 軸 -

### 童年/共同史（storyReveal 素材池）

> 用來餵後續對白、揭露事件，不是一次倒出來。

| 階段 | 事件 | 後續影響 |
|---|---|---|
| 14 歲 | 阿圖斯搶了蓋烏斯喜歡的女孩 → 那女孩後來變成阿圖斯妻（已過世） | 蓋烏斯心結、看到 Livia 會閃眼 |
| 18 歲 | 蓋烏斯陰阿圖斯一場馬賽 → 阿圖斯爸氣死 | 阿圖斯心結、提到馬會冷臉 |
| 25 歲 | 兩家聯手坑了第三個鄰居（後來那人當大臣） | 共同把柄、互相牽制 |
| 35 歲 | 阿圖斯老婆死、蓋烏斯「弔唁」順手買了阿圖斯一塊地 | 阿圖斯酒後會罵 |
| 現在 | 兩人各擁有訓練所、競技場排名互咬 | 賽前下注、賽後各種小動作 |

### 在遊戲中的功能

- 所有「跨訓練所事件」的源頭

## § 2. 跨訓練所事件（橫向、頻繁）

### 2.1 友鄰切磋（隨機、每 7-10 天）

- 蓋烏斯派 1 個小角來打、或玩家被派過去
- 對手屬性對齊玩家當期 ±3、非死戰
- **獎勵**：贏 → 金錢 30 + 名聲 5；輸 → 名聲 -3 + 對方碎念
- **道德軸**：可選放水 / 不放水 → 觸發 pride

### 2.2 雙主人合作場（Day 35）

- 兩家共同派人對戰第三家（如外地訓練所）
- 玩家 + 蓋烏斯家 1 角 → 2v2
- **彩蛋**：賽前隊友（蓋烏斯家那位）可能私下碎念蓋烏斯
- **獎勵**：贏 → 大筆金錢 + 蓋烏斯好感（少量）
- **後續鉤子**：這個對方訓練所 NPC 可能變成「招敵變友」候選

### 2.3 雙主人陰招場（Day 60）

- 蓋烏斯偷偷下藥傷玩家對手 / 派假觀眾干擾比賽
- 賽後玩家發現端倪
- **道德選擇**：
  - 告發 → 阿圖斯 +好感、蓋烏斯敵意 +
  - 沉默 → 蓋烏斯好感 +、reliability 軸 -
  - 反向利用 → 玩家也學陰招 → 觸發新特性「兩面刀」

### 2.4 互換新兵（Day 50 + Day 80）

- 阿圖斯把玩家「借」給蓋烏斯一週
- 那週訓練場景換到維努斯場（NPC 全換、不同氣氛）
- 蓋烏斯人設：比阿圖斯更精明、更冷血、但對玩家試探性的好
- 結束後回阿圖斯場、可能帶回對方秘密
- **獎勵**：見識 + 維努斯場某 NPC 的好感（後續招敵用）

### 2.5 公開宴會撕逼（Day 75）

- 兩主人在宴會上明面友好、暗中針鋒相對
- 玩家是隨從、聽到他們的對話
- **不是戰鬥事件**、純劇情、storyReveal 重要節點
- 揭露其中一段「童年陰影」（從 § 0 表中挑一個）

## § 3. 場內事件（隨機、每 7-15 天）

### 3.1 訓練所人口設計（解答 E 的疑問）

> 目前 ~10 名 NPC + 隊友 6 + 觀眾 3 ≈ 看起來訓練所 < 15 人，但羅馬中型 ludus 應該 30-80 人。

**建議擴充**：

| 類型 | 數量 | 說明 |
|---|---|---|
| 命名 NPC（有故事） | 10-12（現狀） | orlan / hector / cassius / sol / mela / officer / artus 等 |
| 背景命名 NPC（少故事） | 8-10 | 出現在背景、有名字、偶爾有八卦 |
| 純背景人口（無故事） | 15-20 | 隨機名字、隨機死/賣/到 |
| **總計** | **30-40** | 中型 ludus 規模 |

**流動率**：每 7-10 天觸發一次「人口流動事件」之一：
- 「XX 在競技場上沒回來」(死訊)
- 「XX 被賣去外地」(消失)
- 「來了個新人 XX」(新人事件鉤子)

**附帶事件：抬屍體 + 清房間**（核心 immersion 機制）

> 死訊發生隔天、玩家可能被叫去處理後事 — 加深「輸了就掰」的無奈感

- 監督官派玩家（+1-2 個 NPC）去搬死者屍體出去
- 對白：「⋯⋯昨天的、抬出去燒。」
- 清房間：把死者的東西打包、給其他人分掉、床板擦乾淨給新人
- 偶爾玩家會撿到死者的小物（木牌 / 半本日記 / 一塊麵包）→ storyReveal
- mood -5、reliability + 1
- **隨機死訊可以包含名字 NPC**（包括奧蘭、赫克特等、機率低但存在）→ 「不是每個 NPC 都保證活到 100 天」

**有名字 NPC 死亡規則**：
- 奧蘭、赫克特、卡西烏斯等核心 NPC：死亡機率低（< 5% per arena loss event）、且需玩家好感 < 30 才能觸發（高好感保護）
- 不是 100% 安全 → 維持「每個人都可能死」的不適感
- 死訊事件會用 DialogueModal 強化、不只是 log

### 3.3 兩派系選邊戰（核心新機制）

> 訓練所內**兩個招牌戰士**為派系領袖、玩家選邊累積派系認同

#### 兩派定義

| 派系 | 領袖 | 風格 | 主張 |
|---|---|---|---|
| **赫克特派**（陰招派） | 赫克特 | 敏捷 + 出陰招 + 為了贏不擇手段 | 「規矩是給死人立的」、「贏才是真的」 |
| **卡西烏斯派**（傳統派） | 卡西烏斯 | 力量 + 技藝 + 講信義 | 「角鬥士也有尊嚴」、「不靠陰招贏的才是贏」 |

> 兩位都是現有 NPC、不需要新增。

#### 選邊事件（Day 30+ 開始觸發）

- 第一次群毆（INTRA_BRAWL）就是選邊起點
- 後續事件玩家可選某邊立場（背書 / 出手 / 沉默）
- 多次選邊累積派系認同
- 同邊好感 **+30**、對立邊 **-30**（一次定型、不要每次都這麼大）
- 訓練場每邊大約 4-5 個 NPC 跟隨（影響 6d.2 大反撲時誰會跟你）

#### 派系認同對遊戲影響

- 訓練協力時同派系 NPC 提供額外加成
- 大反撲時同派系 NPC 較易加入（好感門檻低）
- 對立派最終可能站到敵方、或退場
- 結局影響：完全偏向赫克特派 → 玩家走「陰招生存」結局（更接近反撲）；偏向卡西烏斯派 → 玩家走「傳統榮譽」結局（更接近標準過關）

### 3.4 新人池伏筆 NPC（隨機出場時機）

> 設計目的：「**有時候你後期才見到他、就再也走不完他的故事**」 — 製造重玩價值

#### 設計

- 名字：**狄圖斯**（暫名、待敲定）
- 設定：知道某段重要秘密的奴隸（例如：知道某個訓練所主人偷稅、或知道領主某段醜聞、或攜帶傳家秘寶藏在身上）
- 出場時機：**隨機 Day 25-80 任何一次「新人到」事件**
- **早出（Day 25-50）**：玩家有 30+ 天可累積好感、走完故事、拿到大獎勵（特殊裝備 / 隱藏結局解鎖品 / 大筆金錢）
- **中出（Day 50-70）**：剛好夠走故事、但獎勵會打折
- **晚出（Day 70-80）**：故事走不完、可能他先死了、獎勵拿不到 → 玩家會悔
- 第二週目可能完全不出（< 100% 出場率、增加重玩動機）

#### 故事鏈（多階段）

| 階段 | 觸發條件 |
|---|---|
| 初遇 | 新人到、玩家可選關心 / 不理 |
| 試探 | 好感 30 → 他開始暗示「我有個秘密」 |
| 揭密 | 好感 60 → 把秘密告訴你 |
| 行動 | 好感 80 → 玩家可選擇用秘密幹件大事（勒索 / 揭發 / 賣情報） |
| 結局 | 走完整鏈 → 他活著陪你到 Day 100 / 沒走完 → 他可能在某次競技場死 |

> 此 NPC 走完的玩家 ≈ 30%、其他人會留下「⋯⋯狄圖斯到底想說什麼」的伏筆感

## § 5. 招敵變友機制（已更新 8-3）

### 5.1 完整鏈條（4 步 + 暗殺反噬）

> 不是「直接邀請」、而是「先打贏 + 放過 → 種下種子 → 後續邀請」

1. **前期遇到（弱化版）** — Day 25-50 之間第一次遇到目標 NPC（友鄰切磋 / 合作場 / 互換新兵）
   - 此時對方屬性比一般同 tier 弱（玩家有機會打贏）
   - 玩家若打贏 → 兩個選項：
     - 「補刀殺死」→ 蓋烏斯場戰力 -、後續招敵失敗、無報復
     - **「放過」→ 觸發 5.2 種子事件**

2. **種子事件（放過後）** — 對方賽後私下找玩家
   - 道謝 + 簡短自介
   - 留下「我欠你一次」的種子
   - Flag 設 `vesnus_npc_{id}_seeded`
   - 累積互動 / storyReveal（每位 2-3 條）

3. **正式邀請（Day 60+）** — 累積條件達成後觸發 ChoiceModal
   - 條件：好感 60+ / 完成身世 storyReveal / 給觸發品
   - 玩家選「跳過來吧」→ 機率判定
   - 成功率：看好感 + 對方對阿圖斯場觀感（高位 = 看不起 / 低位 = 觀望）

4. **後果分歧**
   - **成功**（NPC 轉到阿圖斯場）：
     - ✓ 該角色從敵方 boss 名冊消失
     - ✓ 阿圖斯場多一個強角色（可選為觀眾/隊友、提供協力）
     - ✓ 維努斯場戰力下降
     - ✗ **蓋烏斯震怒 → 5.4 暗殺鏈啟動**
   - **失敗**（NPC 拒絕 + 告密）：
     - ✗ 蓋烏斯知道你想撬人、5.4 暗殺鏈也啟動（更早）

### 5.2 種子事件設計範本

> 玩家放過 NPC 後、賽場散場時的私下對話

```
（你站起身，他卻從沙地裡撐起。）
（他看了你一眼，沒道謝，只是把武器丟到一邊。）

盧基烏斯：「⋯⋯你大可一刀。」
盧基烏斯：「為什麼？」

[選擇]
- 「我贏夠了。」      → reliability +
- 「下次我可不放手。」 → pride +
- 「你看起來還能打。」 → mercy +
（任一選項都觸發種子 flag）

盧基烏斯：「⋯⋯記住這個。」
盧基烏斯：「我欠你一次。」
（他走進維努斯場的人群裡。）
```

### 5.4 蓋烏斯暗殺鏈（招敵副作用）

| 階段 | Day 距觸發 | 內容 |

### 6.3 「無名者」凱德（最終版：BOSS + 故意輸 + 死前促成反撲）

> 2026-05-01 四修（最終）：使用者**最終定案** — 凱德**仍是 Day 100 BOSS**
> 但**戰鬥中故意輸給玩家**、躺在地上、用「死」促成反撲時機
> 動機**不是反抗體制的英雄**、是「**自由身但不知道要幹啥、回不去正常生活、把這條命給你用**」

#### 角色卡

| 欄位 | 內容 |
|---|---|
| 名字 | **凱德**（玩家叫他「凱哥」） |
| 出身 | 跟農家 origin 玩家**同村**、年紀大玩家幾歲 |
| 過去 | **幾年前**被賣（不是童年）、進別家訓練所、為主人賺大錢、早就是自由人 |
| 現狀 | 自由人、但除了競技戰士也沒別的能力 → **選擇繼續打** |
| 戰績 | 30 連勝零敗、競技場圈內傳奇、人稱「無名者」（他自己拒絕公開本名） |
| 角色定位 | **終極 BOSS + 故意輸** — Day 100 萬骸祭最終戰對手、戰鬥中故意敗給玩家、用「躺下」拉領主走下來、製造反撲時機 |

#### 悲劇核心（為什麼是盟友、動機是無路可回）

- 凱德這些年靠主人寄錢回家鄉、主人都說「有寄、家人都好」
- 跟玩家相認後、玩家告訴他**家鄉幾個月前被滅村**（偽旗事件）
- 凱德崩潰、知道這幾年勞動全是養了騙他的主人
- **更深的崩潰**：他早就是自由人、但自由身對他沒意義
- 他不知道自由人能做什麼 — 不會種田、不會做生意、不會當官
- 除了打架什麼都不會、所以才繼續留在競技場打
- **他已經回不去正常生活**

#### 為什麼故意輸（核心動機）

- 不是熱血反抗（不是英雄）
- 不是復仇（他自己親人也死了、但情緒已經麻木了）
- 是**「我已經沒有別的事可以做了」** — 把這條命給你用、好歹結束有意義
- 這比反抗體制的英雄動機**更悲劇** — 是失敗者最後的一搏
- 玩家會理解：凱德不是來救他、是把自己當墊腳石讓玩家殺到領主

#### 重要：凱德故意輸的設計妙處

- **戰鬥本身有戲劇張力**（玩家還是要打、不是純看劇情）
- **凱德倒下 → 領主走下來給最後決意 → 領主拉近** = 暗殺時機自然
- 沒有「凱德跟玩家並肩」的英雄式畫面、是凱德犧牲式的悲劇
- 玩家獨自踩在凱德身上接受歡呼的視覺衝擊強

### 6b.5 衝動 vs 冷靜分支（核心新設計）

> Day 65 觸發相認後、玩家當下情緒由**特性**決定走哪條

#### 分支判定（觸發瞬間）

```js
// 觸發 farmboy_recognized_lord 後立即判定
if (玩家有 衝動/暴怒/不屈 等特性 OR 道德軸 pride 累積 ≥ 60) {
  → 6b.5a 衝動分支（自動衝出去）
} else if (玩家有 冷靜/沉著/堅忍 等特性 OR 道德軸 patience 累積 ≥ 60) {
  → 6b.5b 冷靜分支（自動忍下來、接 § 6d 大反撲）
} else {
  → 6b.5c 中性分支（玩家自選 ChoiceModal）
}
```

#### 「衝動 vs 冷靜」是怎麼來的（雙軌制 — 答 8-23）

**A. 出生特性（開場決定、少數人有）**：
- 出生擲骰可能拿到「**易怒 / 暴躁 / 不屈**」（衝動類）
- 或拿到「**冷靜 / 沉著 / 堅忍**」（冷靜類）
- 多數玩家**沒有**這類特性、走 B 路線決定

**B. 100 天事件累積（核心）**：
- 100 天中**很多事件**都是「忍 vs 衝」測試
- 每次玩家選衝動 → pride 軸 +1
- 每次玩家選忍 → patience 軸 +1
- 累積夠了（pride ≥ 60 或 patience ≥ 60）→ 自動分流
- 這是 [docs/systems/moral.js](../../src/core/moral.js) 的 D.19 道德軸已實作的

**現有可累積的事件清單（已實作或待實作）**：

| Day | 事件 | 衝選項 | 忍選項 |
|---|---|---|---|
| 1 | 開場被監督官鞭打 | 反抗（pride+） | 低頭（patience+） |
| 5 | 沙洗（已實作） | 拒絕 | 配合 |
| 12 | 友鄰切磋第一次被嘲諷 | 罵回去 | 沉默 |

| 30 | 鐵骨阿巴賽前嘲諷 | 嗆聲 | 不理 |

| 60 | 雙主人陰招場發現有貓膩 | 當場揭發 | 隱忍 |
| 65 | **領主訪場被打** | （已是觸發點本身） | （已是觸發點本身） |

**設計目的**：玩家到 Day 65 時、pride/patience 累積已經反映他這 65 天怎麼活的、自然分流不勉強。沒累積到的 → 中性分支讓玩家當下選擇 — 也是一種 immersion。

|---|---|
| 屬性遠不夠（合理場景）→ 死 | **「無名死」結局**：玩家當場被斬殺、沒人知道為何忽然爆走、阿圖斯隔天罵「沒用的東西」、訓練所恢復正常、家鄉冤魂無報 |

#### 為什麼是這四個破綻（設計理由）

- **講錯具體鄰居名字**（最尖銳、最先觸發）：把玩家的家鄉記憶直接放進來、是「**只有當事人才看得出**」的破綻 — 玩家認識真的養豬戶、瓦倫編的名字對不上 → 整個故事立刻倒

- **手**：沒繭、指甲修過 → 同樣沒辦法假造
- **講話風格**：背過的台詞跟真倖存者的破碎話差很遠、玩家有自己村人講話的記憶當對照

- 也會出席夜宴（如果 fame 夠）
- 聽到瓦倫的演戲、會覺得「⋯⋯這人講話不像農夫」、但不會觸發復仇 flag
- 純粹敘事氛圍 — 認識到領主的虛偽
- 影響 patience / pride 道德軸（小幅）

| **B. 大反撲** | Day 100 打贏最終戰後、**領主致詞時**玩家主動丟武器 / 撲領主 → 群戰 | 5 個新分支結局（自由人 / 血祭 / 殉道者 / 無名死 / 沉默自由） | **15-25%**（看玩家好感與道德軸） |
| **C. 提早溜走** | Day 80 跟比拉斯走地道、不參加萬骸祭 | 1 個新結局（逃亡者） | **5-10%** |
| 死亡 | Day 100 戰敗 / 反撲失敗 / 逃亡失敗 | 既有死亡結局 / 新分支死亡結局 | — |

### 6d.1 為什麼這樣分

- **標準過關是核心**：絕大多數玩家會走這條，這條已有完整 8 結局系統，不動
- **大反撲是「真正想完成什麼的人才會選」**：需要好感累積、需要劇情觸發（凱德 / 農家死仇 / 厭倦體制）
- **提早溜走是「不演了」的玩家路線**：放棄名聲、活下來
- 三條互斥、一個 run 只能走一條

### 6d.2 大反撲結局：殘血群戰機制（B 路徑、opt-in、最終定案）

#### 觸發條件（必須**主動**選、不會自動觸發）

- **時機**：Day 100 萬骸祭打完凱德 wave 5、**領主走下競技場**、走到玩家面前 3 步
- **入口**：ChoiceModal「最後決意」（見 § 6c1.3）— 玩家可選暗號發動
- **盟友條件**：選擇反抗時、若累積好感 80+ 的 NPC < 5 名 → 系統跳警告「你只有 X 個盟友、必死」、可反悔回 A 路徑
- **無農家死仇也可選**：理由不必是復仇、可以是「我不想再演了」、「凱德把命給我了」、「為了死去的兄弟」
- **重點**：選 A（接受加冕）= 直接觸發既有 ending.js 結局判定、跟現在沒差

#### 戰場狀態（核心改動：**全員殘血**、不是 buff 強化）

> 凱德剛死、玩家剛跟凱德打完一場硬仗、其他盟友都剛剛從別的戰鬥（沙地外圍 / 觀眾席暴動）扛過來
> 不是強化版玩家陣營、是**殘破之師對精銳衛兵**的悲壯場景

| 角色 | HP 狀態 | 說明 |
|---|---|---|
| 玩家 | ~50% | 5 wave 打完、傷沒治、武器是訓練用沒帶完整裝備 |
| 凱德 | 死 | 剛剛被玩家殺了、躺在場中央、不能加入 |
| 赫克特（若加入） | ~50% | 剛從場外衛兵殺出、扛傷 |
| 奧蘭（若加入） | ~40% | 沒戰鬥訓練、勉強跟上 |
| 招敵 NPC（盧基烏斯/諾克斯）| ~60% | 從觀眾席衝下來 |
| 其他好感 80+ NPC | ~50% | 跟著起鬨、扛傷 |
| 觀眾席暴動奴隸（弱） | ~30% | 一波就倒、純人海 |

**敵方陣營**（滿血、武裝齊全）：
- 領主衛兵 30 人（基本 stat、長矛短劍、列陣）
- 提圖斯本人（中強、有 4 衛兵護衛、距離玩家 3 步）
- 維努斯場奉命的角鬥士 10-15 人（若蓋烏斯仍在場）

#### 戰鬥節奏：殘血硬拼 + 斬首機會

| 階段 | 設計 |
|---|---|
| **斬首 1**（最關鍵） | 玩家先機 — 領主在 3 步內、玩家衝刺斬殺 — 若殺到 → 提圖斯死、戰場大亂、敵方戰意大跌 |
| 階段 2 | 衛兵反應、玩家陣營 5-20 人 vs 衛兵 30 人、雙方各死人 |
| 階段 3 | 衛兵援軍到 10 人、玩家方 NPC 一個個倒下（演 NPC 名字 + 死法） |
| 階段 4 | 殘餘玩家陣營硬撐到底、看誰先力竭 |

#### 數值設計（殘血劣勢、非滿血 buff）

> 不是「人多就贏」、是「人多硬撐、看能不能撐到把領主殺了」

- 玩家自己 HP ~50%、武器訓練用、屬性正常
- 每位盟友 = 半血戰士、屬性正常但 HP 50%
- **斬首戰**：玩家 vs 領主（領主滿血、4 衛兵保護、距離 3 步）
  - 殺到一招 → 後續變主帥已死的混戰
  - 沒殺到 → 衛兵圍上、進階段 2
- **計算（粗）**：盟友總 HP × 0.5 vs 衛兵總 HP × 1.0
  - 5 名盟友 = ~750 HP vs 30 衛兵 ~3000 HP（很難贏）
  - 10 名盟友 = ~1500 HP vs 30 衛兵 ~3000 HP（拼一拼）
  - 20 名盟友 = ~3000 HP vs 30 衛兵 ~3000 HP（人海可能贏）
- **斬首戰若成功**：衛兵戰意 -50%、敵方總 HP 變 1500 → 玩家陣營容易翻盤
- **核心張力**：玩家斬首這一刀**是凱德犧牲換來的**、是這場反撲能不能成的關鍵

#### 結局分支

| 戰況 | 結局 |
|---|---|
| 大勝（5+ 盟友、玩家活、提圖斯死） | **「自由人」結局**：城市動盪、玩家逃出、變地下傳奇 |
| 小勝（玩家活、提圖斯死、盟友幾乎全滅） | **「血祭」結局**：成功但孤獨、所有人為你死 |
| 兩敗（玩家死、提圖斯死） | **「殉道者」結局**：你死了、但揭發成真、家鄉冤魂安息 |
| 失敗（玩家死、提圖斯活） | **「無名死」結局**：失敗、但凱德活下來繼續抗爭 |
| 拒絕（玩家認慫、回去當奴隸） | **「沉默自由」結局**：得自由人身份、但每晚都做夢 |

### 6d.3 提早溜走結局：比拉斯地道

#### 角色設計：「想逃的隊友」

| 欄位 | 內容 |
|---|---|
| 名字 | **比拉斯**（暫名、已敲定 8-13） |
| 設定 | 訓練所內較年輕、被擄不久 |
| 個性 | 怕死、機靈、整天看牆找縫 |
| 公開人設 | 大家都覺得他軟弱、會哭、訓練混 |
| 暗面 | 他在挖**地道**、已經挖了好幾個月、就快通了 |

#### 觸發鏈（修正：好感從 20 → 60、加屬性門檻）

> 比拉斯不是隨便信人 — 他要**確認玩家可靠 + 不是冷血殺手**才會帶你

| Day | 事件 | 觸發條件 |
|---|---|---|
| Day 25 | 比拉斯訓練混被監督官打、玩家路過、可選表情（看 / 同情 / 無視） | 無 |
| Day 45 | 比拉斯被同訓練所人欺負、玩家可選介入 | 無 |
| Day 55 | 比拉斯主動找玩家、隨便聊、試探玩家口風 | 玩家好感 ≥ 30 |
| Day 65 | 比拉斯酒後說漏「⋯⋯如果有辦法走、你會想嗎？」、玩家答 | **好感 ≥ 60** |
| Day 70 | 比拉斯確定玩家信任值、帶玩家看地道、邀請 | **好感 ≥ 60 + 道德軸滿足以下任一**：<br>• reliability 累積值高（玩家可靠、不會出賣他）<br>• mercy 累積值高（玩家不喜歡這個殺戮的世界） |
| Day 80 | **逃跑之夜**（玩家選擇接受） | 上一階段達成 |

#### 道德軸門檻（重要）

> 為什麼要 reliability OR mercy ？

- **reliability 高** → 比拉斯知道玩家不會臨陣脫逃、不會出賣計畫
- **mercy 高** → 比拉斯知道玩家對這個體制有反感、有逃離動機
- **兩者都低** → 比拉斯判斷玩家「就是個殺手、會把我賣給主人」、不會邀請
- **兩者都高** → 觸發加碼對白（比拉斯說「我等你很久了」）、逃跑成功率 +20%

#### 逃跑之夜：殺出血路關卡

- 4-6 場小型戰鬥（巡邏衛兵、夜班守衛、追兵）
- 每場 1-3 個敵人、屬性中等
- 過程中比拉斯可能受傷 / 被抓 / 死亡（依玩家配合度）
- 最終出城：**「逃亡者」結局**

#### 後續

- 玩家不再回訓練所、Day 81-100 不會發生
- 失去萬骸祭、失去名聲
- 但活著、自由
- 可能多年後路上偶遇凱德 / 奧蘭、他們提到「萬骸祭那年發生了什麼」（傳聞性後話）

### 6d.4 三條結局線的選擇張力
（背包帶子勒進肩膀的痛感、突然像第一次。）
- **A 標準過關** = 中風險中回報、活下來看屬性/特性決定怎麼活、預設路線
- **B 大反撲** = 高風險高回報、史詩、可能死、改變世界
- **C 提早溜走** = 低風險低回報、平凡、活著、不改變世界
- 玩家的道德軸（pride vs patience / loyalty vs reliability）會影響哪條更自然
- **絕大多數玩家走 A**、深度玩家會試 B、厭世玩家會試 C
- **重點：A 路徑用既有 8 結局系統、不重做**

### 6d.5 跟既有 ending.js 的整合

> [src/ending.js](../../src/ending.js) `Endings.pickAndPlay(survived)` 已實作 8 結局判定

- A 路徑：直接呼叫 `Endings.pickAndPlay(true)`、不變
- B 路徑：新加 `Endings.playRebellion(allyCount, lordKilled, playerSurvived)` → 5 個分支
- C 路徑：新加 `Endings.playEscape(escapeNightSurvivors)` → 1 個結局
- 死亡：A 死直接 `Endings.pickAndPlay(false)`、B/C 死走專屬死亡結局
