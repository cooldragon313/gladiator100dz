# 跑腿外出事件系統（errand-outings）

> 玩家**短暫離開訓練場**的事件群。為遊戲帶入「外面的世界存在」、給玩家一個喘息點、也是劇情伏筆的觸發場域。
>
> 設計原則：每次外出都有可能觸發**意外事件**（撞小孩、聽鄉音、扒手等）。事件本身比跑腿任務還重要 — 跑腿是藉口、外面才是戲。
>
> 最後更新：2026-05-06（第一版設計、撞小孩 3 路徑 + 護衛分支 + 護身符後續）

## § 0. 設計目的

1. **建立「外面世界」的存在感** — 訓練場很悶、外面有風 / 鳥 / 老婆婆 / 商販 / 衛兵
2. **逃跑伏筆** — 玩家確實有機會逃、但理性上不會（家鄉沒了、城裡守衛多）→ 為未來「同伴邀請逃跑」事件鋪墊
3. **特性驗證點** — 撞小孩等事件依玩家 traits 走不同路徑、把玩家的「自己選的」特性驗證在行為上
4. **道具觸發鏈** — 外出帶回的道具（如護身符）變成後續 NPC 事件的鑰匙

## § 1. 邏輯前提（CANON 級）

### 玩家能逃跑嗎？

**理論上能、實際上不會**：
- 主人**不直接武裝看管** → 用「同奴陪同 + 都市社會壓力」防逃
- 玩家**有逃跑動機**（不開心 / 家鄉夢）— 為未來「同伴邀請」事件保留可能性
- 但目前沒人幫他逃 / 家鄉沒了 / 城裡守衛多 / 沒身份沒錢沒地方躲 → 理性上不會自己跑

### 為什麼能跟同奴出門？

主人逐步建立**信任**：
- 主人好感 ≥ 30 → 第一次差事（最簡單、近距離）
- 主人好感 ≥ 50 → 中距離差事（可能跟其他訓練所打交道）
- 主人好感 ≥ 70 → 信任型差事（單獨送信、處理小事）

僕人傳話會明說：
> 「主人說、相信你不會幹傻事。城裡守衛很多。準時回來。」

這是**社會契約**而不是武裝威懾 — 比起被銬著、這種「我相信你」反而更有束縛力。

## § 2. 觸發系統 — 4 個跑腿源頭

### 觸發判定（每天上午 / 訓練前掃描）

```js
function tryErrand() {
  if (Flags.has('errand_today_done')) return;
  if (玩家好感主人 < 30) return;
  if (Math.random() > 0.10) return;   // 每天 10% 機率出現
  // 滾動 4 個源頭、抽一個條件符合的
  const source = pickAvailableSource();
  if (!source) return;
```

## § 3. 中段隨機事件池

> 跑腿途中 30% 機率觸發一個。事件本身比跑腿還重要。

### 已設計（本輪實作）
- ✅ **撞小孩事件** — 詳見 § 4

### 待擴充
- 🔲 **聽到鄉音** — 農家 origin 玩家聽到家鄉口音、回憶湧上、或問話
- 🔲 **路人施捨** — 老乞丐塞玩家半個麵包、玩家給或不給
- 🔲 **目擊扒手** — 看到扒手偷有錢人、選擇舉報 / 沉默 / 當場制止
- 🔲 **舊識** — 玩家家鄉認識的人擦肩、可能認出 / 不認出
- 🔲 **黑市攤販** — 偷偷賣禁書 / 偷渡管道 / 可疑藥物（觸發逃跑相關）
- 🔲 **打架圍觀** — 兩個醉漢打架、選擇插手 / 沉默
- 🔲 **賤民被追打** — 衛兵追打一個亂民、玩家可選介入

每個事件都應該：
- 依玩家 traits / 好感分支
- 有明確 stat 獎懲
- 帶回**道具**或**flag** → 後續 NPC 對白觸發

## § 4. 撞小孩事件（核心、本輪實作）

### 觸發
- 任一跑腿途中、30% 機率擲骰
- 一個 run 最多觸發 1 次（doneFlag: `errand_child_bump_done`）

### 開場（共通）

```
（人潮擁擠、你低著頭走、肩膀不時擦過陌生人。）
（——震！）
（一個小小的身影撞上你的腿、跌坐在地。）
小孩：「嗚⋯⋯哇——!」
（小孩沾著沙、嘴角發抖、抬頭看你。）
```

### 護衛分支判定（50% 機率）

```js
const hasGuards = Math.random() < 0.5;
const isVesnus = hasGuards && Math.random() < 0.4;   // 護衛 → 40% 機率是維努斯場小孩
```

### 路徑 1：無護衛、玩家依特性走（merciful / cruel / neutral）

#### **merciful**（玩家有 `merciful` / `kindness` 特性）

```
（你蹲下、把他扶起來、拍他衣服上的灰。）
（一個女人從人群中擠出來、衝過幾個攤販。）
母親：「不要——! 不要打他!」
（她伸手要拉走小孩、整張臉是恐懼。）
（她看見你的姿勢——你的手只是扶著、不是抓著。）
（她愣住了。）
（——你看到她從圍裙口袋裡掏出一塊布包。）
母親：「⋯⋯這個給你。」
（她塞進你的手裡、不等回應就拉著小孩離開。）
小孩：「⋯⋯謝謝叔叔。」
（你打開布包——裡面是一塊小銅飾、繡著鳶尾花紋。）
（你把它收進懷裡。）

[同行者] ：「⋯⋯走吧。耽誤太久主人會問。」
```

**獎勵**：
- mood **+25**
- food +20（差事還是有買到）
- kindness EXP **+20** / mercy 軸 +
- **獲得「鳶尾紋飾」personal item**（LUK +1、storyTag）
- 主人 **-1**（拖延、回家晚了 5 分鐘）

#### **cruel**（玩家有 `cruel` / 已推 pride 軸）

```
（你冷冷看著地上的小孩。）
（——你沒蹲下。）
（你瞪了他一眼、轉身要走。）
（一個女人從人群中擠出來。）
母親：「不要——!」
（她衝過來抱住小孩、整個人縮在小孩前面。）
（她不敢抬頭看你。）
（小孩在她懷裡發抖、嘴角的血沒擦。）
（——你走了。）

⋯⋯

（你頭也沒回。）
（但你聽見小孩在哭。）
（內心對白：「⋯⋯我看起來像那樣？」）
（內心對白：「⋯⋯以後做夢、會看到那雙眼睛嗎？」）

[同行者] ：「⋯⋯這就對了。主人不會等你。」
```

**獎勵**：
- mood **-10**
- food +20
- pride 軸 + / reliability 軸 -
- 主人 **+1**（準時到家、護衛回報「沒讓主人丟臉」）
- **觸發內心對白：「以後夢裡會看到」**（埋伏筆）

#### **neutral**（無特性傾向）

```
（你愣住了。）
（小孩在腳邊哭、沙土沾在他臉上。）
（——你的手抬到一半、又放下。）
（一個女人衝過來抱住小孩、看了你一眼、拉他離開。）
（沒有人說一句話。）

（你站在原地。）
（——我什麼都沒做。）
（——但好像有什麼錯過了。）

[同行者] ：「⋯⋯人擠人、走快點。」
（你跟上去。）
```

**獎勵**：
- mood **0**（沒變化、純劇情）
- food +20（差事完成）
- **無屬性獎懲**
- 內心對白留下空虛感

> Neutral 路徑核心精神：失去機會的空虛。沒做壞事、也沒做好事、像沒發生 — 但玩家自己知道有什麼錯過了。

### 路徑 2：有護衛在場（小孩家有錢）

撞到小孩瞬間、看到旁邊兩個壯漢立刻擋下：

```
（——震！）
（小孩跌坐在地、哭出來。）
（兩個壯漢從人群中分開、擋在你前面。）
護衛：「⋯⋯你瞎了？」
護衛：「⋯⋯小公子、有沒有事？」
（小孩在哭、母親從後方走出來、皺眉看你。）

[ChoiceModal: 你怎麼處理？]
```

**ChoiceModal 三選一**：

| 選項 | 條件 | 走向 |
|---|---|---|
| **道歉** | 無條件 | 護衛冷冷哼一聲、放你走、無事 |
| **幹架** | STR ≥ 20 / 玩家有 cruel 或 prideful | 戰鬥 1v3 中階護衛、或 1v1 vs 高階招牌 |
| **逃** | AGI ≥ 25 | 跑、可能被追、可能被抓 |

#### **道歉**

```
（你低頭。）
玩家：「⋯⋯我沒看到。對不起。」
（護衛冷哼一聲。）
護衛：「⋯⋯下次小心點。」
（他們帶著小孩跟母親離開。）
（——你看到母親回頭看了你一眼、欲言又止。）

[同行者] ：「⋯⋯走吧、回家了。」
```

獎勵：
- mood -3（被罵、不爽）
- pride 軸 -（低頭）
- 無其他懲罰

#### **幹架（無特性 / cruel / prideful 才出現）**

```
（你抓緊腰間。）
玩家：「⋯⋯誰瞎了？」
（護衛眼睛一瞇。）
護衛：「⋯⋯有趣。」
（旁邊的人群開始散開。）
護衛：「跟我走一趟。後巷說。」

[戰鬥開始 — 看小孩身份]
```

**判定戰鬥規模**：

| 小孩身份 | 戰鬥配置 | 戰勝獎勵 |
|---|---|---|
| 普通有錢人家 | 1v3 中階護衛（STR/CON ~30）| 1 件護衛護甲（leather/chainmail）+ 金錢 30 |
| **維努斯場主人家**（40% 機率）| 1v1 vs 「布魯圖」或「奎因圖斯」（招牌、STR 50+）| 對方招牌的武器（T2 級）+ 大筆名聲 +20 + 大事件 |

> 維努斯場小孩 = 蓋烏斯的私生子 / 親戚 → 跟 [arena-events-roster.md § 5 招敵變友](arena-events-roster.md) 的暗殺鏈整合

**戰鬥流程**：玩家被護衛帶到後巷、戰鬥（用 Battle.startFromConfig）、戰勝後拿戰利品、護衛/招牌投降或撤退

**戰勝獎勵**：
- 護衛裝備 1 件（不剝對方光、就是戰利品掉落）
- mood **+15**
- pride 軸 +
- fame +10（普通）/ +30（維努斯場招牌）
- 主人事後**會知道**（公然鬥毆藏不住）→ 反應分歧：
  - **隨機 / 主人好感 ≥ 50** → 「我家小子打贏 X 家的招牌」高興 +5 / 額外賞 +20 金錢 + 1 天休息
  - **主人 < 50** → 不高興 -3 / 罰款 -50 金錢 / 鞭打 + mood -10

**戰敗**：
- 重傷退場（HP -50 / 體力 -50 / mood -30）
- 主人破費保你出來、主人 -10 / 罰金 -100

#### **逃（AGI ≥ 25）**

```
（你轉身、衝進人群。）
護衛：「站住——!」
（你衝進巷弄、攤販的東西被你撞翻、有人罵。）

[判定逃跑成功率：AGI 越高越成功]
```

**成功（AGI ≥ 40）**：
- 跑掉、沒被抓
- mood -5 / pride 軸 -
- 同行者：「⋯⋯你瘋了。下次別這樣。」（不打小報告 — 同奴有點良心）
- 主人**不知道**

**失敗（AGI < 40）**：
- 被衛兵抓回、丟回主人家
- 主人 **-10** / 罰款 -100
- 鞭打 + 失眠 trait 風險

## § 5. 「鳶尾紋飾」護身符 — 道具系統

### 道具卡

| 欄位 | 內容 |
|---|---|
| 名字 | **鳶尾紋飾** |
| 圖標 | 🌸（藍紫銅徽）|
| 類型 | personal item / amulet |
| 效果 | LUK +1、戰鬥 EVA +2 |
| 描述 | 一塊小銅飾、繡著鳶尾花紋。市集那位母親從圍裙口袋裡掏出來塞給你的。看起來不像一般人會有的東西。 |
| storyTag | `iris_charm` |
| 來源 | 撞小孩事件 merciful 路徑 |

### 真實身份（揭露）

不是「老婆婆做的」、是**某沒落貴族家的家徽**：
- 鳶尾花是某舊貴族「**德拉格家族**」的家徽（暫名）
- 該家族曾是城裡權貴、近年沒落、被擠出政壇
- 家族遺老仍在、留有一些舊符號 / 信物
- 那位母親可能是該家族的**遠親 / 私生子分支 / 落魄後人**
- 小孩繼承家徽飾品、母親拿來保命用 / 給你當謝禮

### 訓練所 3 NPC 認出對白（核心鉤子）

當玩家穿戴或攜帶「鳶尾紋飾」時、特定 NPC 對話會多一段：

#### **老默**（醫生、見多識廣）

> 觸發條件：玩家攜帶 + 老默好感 ≥ 30 + 隨機晚上事件

```
老默：「⋯⋯那塊鳶尾。」
（他看著你的飾品、眼神不太一樣。）
```
