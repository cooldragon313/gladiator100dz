> 開放空手系（unarmed）戰鬥流派。透過 Forum 邊緣的退役角鬥士盧基烏斯傳授。
> 設計初稿：2026-04-25
> 對應程式：尚未實作 / 預計 `src/npc/lucius_events.js` + `src/npc/npc.js` 加 NPC 定義

## 🏛 羅馬背景對齊

- **拳法 ≠ 中國武俠**：羅馬時代有 **pankration**（混合格鬥摔技）+ **pugilism**（拳擊）兩種傳統
- **沒有秘笈 / 武功秘籍**：羅馬世界沒這個概念。改用「**師徒傳承**」(transmission by training)
- **盧基烏斯只會 3 招** — 因為**他師父也只教到第三招就被殺**。傳承斷在這
- **沒有 T4** — 是設計的一部分、不是缺陷。「這條路就到這裡」

## 🎭 角色設定

### 斷腳的盧基烏斯 (Lucius "the Cripple")

- **本名**：Lucius（盧基烏斯）— 羅馬最常見的男子名、刻意平凡
- **外號**：「斷腳的」(`The Cripple`) — 大多數人只記得這個
- **年齡**：30+ 歲（看起來更老）
- **背景**：曾是邊緣訓練所的角鬥士。約 8 年前在主人安排的「賭局送頭戰」中**奇蹟存活但失去左腳**。沒被救回、被丟出訓練所自生自滅。靠乞討跟偶爾教點拳法為生
- **位置**：**露天交易市場外的牆角**（Forum 邊緣巷弄）— 羅馬城邦商業中心的邊緣地帶、人來人往但沒人看他
- **外觀**：左腳膝下截斷、用拐杖、灰白頭髮糾結成塊、眼神異常清醒
- **性格**：話少、不抱怨、看人準、不對任何人懷有惡意（連對主人也不）

### 跟巴爺主線的對位
| | 巴爺（Babrius） | 盧基烏斯 |
|---|---|---|
| 結局 | 被「救回」、留在訓練所當監督官 | 被丟出體制、流落街頭 |
| 體制視角 | 招牌、看板 | 失敗品、垃圾 |
| 心境 | 不知道自己被算計過 | 早就知道、接受了 |
| 火種 | 體制裡的徒勞 | 體制外的傳承 |

**設計意圖**：玩家透過盧基烏斯看見**巴爺主線的另一個結局** — 「沒被救的人是這樣的」。同時學的拳法是**反系統的符號**（沒武器、不靠裝備、靠技巧活下去）。

### favoredAttr / 愛憎特性
- `favoredAttr: 'AGI'` — 機動性是空手存活的核心
- `likedTraits: { reliable: 2, humble: 2, kindness: 1 }` — 老實、謙卑、有善意的人他願意教
- `dislikedTraits: { cruel: 3, opportunist: 2, prideful: 2 }` — 殘忍 / 投機 / 驕傲的人他連話都不說

## 📅 觸發鏈（4 階段）

### 段 1：暗示（Day 30-50）
玩家在以下任一既有事件中聽到盧基烏斯的存在（自動補一句）：

#### 卡西烏斯補刀（已實作）— 加一句
> 「⋯⋯巴爺算幸運的。沒被救的人多了去、市場外那個瘸子、聽說以前也是好手⋯⋯」
> （他喝一口水。）「⋯⋯算了。命就是命。」

#### 老默接話（已寫資料、缺 hook 待補）— 加一句
> 「上一個跟你一樣表現的人⋯⋯被退役了。沒死、但也沒回訓練所。」
> （他繼續包紮。）「⋯⋯Forum 那邊偶爾還能看到他。」

**任一觸發** → set flag `heard_about_cripple`。

### 段 2：相遇 — 閃避判定（Day 30+ 起）
跑腿事件中（`errand_market_*`），玩家走過 Forum 邊緣時、盧基烏斯的拐杖突然絆出來。**用閃避判定篩掉低屬性玩家**：

#### 觸發條件
- `Flags.has('heard_about_cripple')` ✓
- 跑腿事件中
- 35% 機率
- 還沒觸發過 `met_lucius` 或 `lucius_brushed_off_rude`

#### 閃避判定
依玩家 AGI + DEX 的 effective 值：
- `Stats.eff('AGI') >= 18 && Stats.eff('DEX') >= 15` → **閃避成功**分支
- 其中一項不夠 → **撞牆失敗**分支

#### 分支 A：撞牆失敗（屬性不夠）
- 視覺：**全螢幕震動 + 紅光閃** + `Stage.popupBig({ icon:'💥', title:'撞上！', color:'red', shake:true })`
- 數值：HP -10 / mood -8
- 對白：
  > （你正想閃過、但腿沒跟上反應。）
  > （肩膀重重撞上市場柱子、你倒退兩步。）
  > **主角**（內心）：「⋯⋯（鍛鍊還是不夠。反應慢了。）」
  > （你扶著牆站起來、沒看清是什麼絆到你。）
  > （獨腳老人縮回拐杖、低頭沒說話。）
  > （你拍拍灰、繼續走。）
- **設計暗示**：玩家直接感受「我屬性不夠」、知道要再練 AGI/DEX
- set flag `lucius_brushed_off` + `lucius_last_attempt_day = p.day`、**3 天 cooldown** 才能再次觸發
- **不**設 `met_lucius`（沒真的相遇）

#### 分支 B：閃避成功（屬性夠）
- 視覺：**輕微震動 + 慢動作感** + `Stage.popupBig({ icon:'✦', title:'閃過了', color:'gold' })`
- 對白：
  > （你眼角餘光掃到拐杖橫出來——）
  > （你下意識側身、繞過。）
  > （你停下、轉頭看。一個獨腳老人坐在牆角、剛才差點撞到他。）
  > **盧基烏斯**：「⋯⋯」
  > （他抬頭看你。眼神異常清醒。）
- → 進入 ChoiceModal「道歉並幫助」分支（見下）

#### 道歉幫助選擇（分支 B 接續）
**ChoiceModal**（標題：「他看著你」）

| 選項 | 條件 | 結果 |
|---|---|---|
| 「對不起，我沒看到你。」+ 給點錢 | 玩家有錢 ≥ 5 | -money 5 / set `met_lucius` / set `lucius_kindness_1` / **不漲 affection 數值** |
| 「對不起。沒事吧？」+ 給食物 | 玩家有食物 ≥ 10 | -food 10 / set `met_lucius` / set `lucius_kindness_1` / **不漲 affection 數值** |
| 「⋯⋯沒事吧？」（純關心、沒給東西） | 任何人都能選 | set `met_lucius` 但**不**設 `lucius_kindness_1`（下次相遇要重做）|
| 「閃開、別擋路。」 | 任何人都能選 | set `lucius_brushed_off_rude` / Moral.push('mercy', 'negative') / **永久封印**盧基烏斯線（玩家自選）|

### 🎨 好感邏輯特例 — 「結緣」非「買感」

盧基烏斯**不用 affection 數值**，改用「**緣分階段**」(`lucius_bond_stage` 0-5) 推進。

設計理由：他不是「愛被人施捨」、**他願意接受你的好意 = 他覺得有緣**。給東西 = 結個善緣、不是買感情。

> 「本來就要給他東西、就當他不喜歡白收、也看少年的有眼緣吧。」

機制：
- 給東西不漲 affection、改進 `lucius_bond_stage` 階段
- 每次相遇 +1 stage（最多 5）
- 每階解鎖下一段對話 / 教招
- 卡片顏色就照階段對應（從灰 → 暖橘 → 金）

#### 後續觸發限制 + 善意獎勵階梯
**門檻**：4 善意特性（`humble` / `merciful` / `kindness` / `reliable`）中**至少有 1 個**才會解鎖第 2 次相遇對話。
- 沒有任一個 → 即使第 1 次給錢、盧基烏斯也不會主動開口（只是繼續沉默乞討）
- 設計理由：拳法傳承不傳「沒善意的人」、跟他 dislikedTraits 連動

**🆕 善意獎勵階梯**（每次見面都結算 — 善意越多陪你練越久）：

| 善意特性數 | 對白變化 | 每次見面 EXP 獎勵 |
|---|---|---|
| **1 個（門檻）** | 「⋯⋯小子。坐。」基本對話 | 無 bonus |
| **2 個** | 「⋯⋯你比我想的好一點。陪你練一下。」 | **DEX +200** |
| **3 個** | 「⋯⋯你寬厚、又謙虛。這樣的人路難走。罷了、多幫你一點。」 | **DEX +200 / AGI +200** |
| **4 個（全滿）** | 「⋯⋯（他第一次正眼看你。）我看你真的⋯⋯難得。多陪你練幾下吧。」 | **DEX +200 / AGI +200 / WIL +200** |

設計理由：
- DEX / AGI / WIL **正好是空手系核心三屬性** — 盧基烏斯「看見好人就送他們需要的根基」
- **每次見面都給** — 後期屬性升級越來越貴（expToNext 指數成長），這個 EXP 是穩定 income
- 玩家如果一直陪盧基烏斯（10 次相遇）= 4 善意 → 累積 DEX +2000 / AGI +2000 / WIL +2000
- **善意可動**：玩家若中途流失某特性（道德軸轉向）→ 下次見面獎勵變少。反之亦然
- 不需要 `lucius_kindness_bonus_paid` flag、每次都結算

### 段 3：學習（多次相遇）

| 次序 | 觸發 | 內容 |
|---|---|---|
| **第 1 次相遇** | 段 2 閃避成功 + 給東西 + 有任一善意特性 | 對話建立連結、未教招 |
| **第 2 次相遇** | 至少隔 3 天 + `lucius_kindness_1` + 有善意特性 | 他開口：「⋯⋯小子。我看你也快變成我這樣。」要不要學一招？選擇「願意聽」→ 教 **赤手奪刃 T1** |
| **第 3 次相遇** | 隔 5 天 + 玩家在戰鬥中受過重傷 | 「你撐到現在還站著、不容易。」教 **借力反摔 T1** |
| **第 4 次相遇** | 隔 7 天 + 玩家曾使用前兩招 ≥ 5 次 | 「⋯⋯這招很危險。對自己也是。」教 **要害打擊 T1** |
| **第 5 次相遇（隱藏）** | 巴爺主線完成 + 對盧基烏斯提巴爺 | 他眼神變了。「⋯⋯哦。他活著。」 |

### 段 4：T2-T3 自悟（沒有師父教、靠經驗）
| 階 | 條件 |
|---|---|
| 任一招 T2 | 該招在戰鬥中使用 ≥ 8 次 + AGI ≥ 25 + 付額外 EXP |
| 任一招 T3 | T2 招使用 ≥ 12 次 + AGI ≥ 30 + 付更多 EXP |

### 🆕 段 5：T4 自創招式（屬性自悟 + 玩家命名）
- **條件**：T3 全 4 招都會 + AGI ≥ 35 + DEX ≥ 30 + WIL ≥ 25
- **觸發**：在戰鬥中使用 T3 招達 N 次後、戰鬥結束自動 popup
- **內容**：系統按玩家 build 風味決定**效果**、由**玩家文字輸入命名**

#### 流程
1. 戰鬥結束 → DialogueModal：「（你倒在沙地上喘氣。腦中突然出現一個畫面——這個動作、你從來沒做過、但你確定它能用。）」
2. 系統判定主導屬性（最高的）：
   - 高 AGI 主導 → 效果：被攻擊時 50% 機率讓敵人下回合 miss + 自身 EVA +20 持續 3 回合
   - 高 DEX 主導 → 效果：戰鬥開場第 1 回合、必中暴擊（無視所有防禦）
   - 高 WIL 主導 → 效果：HP < 30% 時 ATK +30%、CRT +20、SPD +10（持續至戰鬥結束）
   - 三者均高（差距 ≤ 3） → 效果：上述三招各 60% 強度的綜合
3. **prompt 玩家輸入招式名**（中文 / 英文皆可、最多 6 字）
   - 顯示效果說明、讓玩家根據效果命名
   - 預設值給空白、玩家可選按 Enter 跳過 → 自動命名「無名」
4. 永久取得後在角色頁顯示為**「✦ 自創 T4：[玩家命名]」**、特殊金色邊框

#### 隱藏小事件
玩家下次見盧基烏斯、可選「我自己創了一招、你要看嗎？」
- 他看完：「⋯⋯哦。」沉默許久。
- 「⋯⋯【玩家招名】啊。」（他唸出玩家取的名字）
- 「⋯⋯這招我師父沒教過。」
- 「⋯⋯也許你比我們都強。」
- 然後他笑了。「⋯⋯傳承沒斷。」
- set flag `lucius_saw_t4_creation`、解鎖隱藏結局線

**未來 Phase 2 可能補**：找到盧基烏斯師父的另一個徒弟（散落他處的倖存者）→ 解 T4。**現階段不做**。

## 🥊 4 個招式（羅馬風重設計）

### 1. 赤手奪刃 (Bare-Hand Disarm)
- **羅馬語感**：pankration 的「武器奪取」技巧
- **效果 T1**：1 回合內被攻擊有 **60% 完美格擋**（無傷）
- **效果 T2**：80% 完美格擋 + 格擋成功時敵人 SPD -10%（1 回合）
- **效果 T3**：100% 完美格擋 + 格擋成功時敵人下回合無法用特技
- **cost T1**：AGI 100 + WIL 50（req AGI 18 + WIL 14）
- **cd 4 / sta 12**

### 2. 借力反摔 (Leverage Throw)
- **羅馬語感**：pankration 摔技 — 利用對方力量摔倒
- **效果 T1**：被攻擊時、把傷害的 **70% 反彈**給敵人
- **效果 T2**：85% 反彈
- **效果 T3**：100% 反彈 + 敵人下回合無法行動（重摔倒地）
- **cost T1**：AGI 80 + WIL 50 + DEX 70（req AGI 20 + DEX 18）
- **cd 5 / sta 18**

### 3. 要害打擊 (Vital Strike)
- **羅馬語感**：希臘羅馬醫學「動脈 / 神經叢」概念
- **效果 T1**：命中後、敵人下 **2 回合**無法用特技
- **效果 T2**：3 回合 + 命中時 30% 機率連帶讓敵人 ATK -20%（3 回合）
- **效果 T3**：4 回合 + 50% 機率 + 敵人 SPD 也 -20%
- **cost T1**：DEX 100 + WIL 50（req DEX 20 + WIL 18）
- **cd 4 / sta 15**

### 4. 關節破 (Joint Breaker)
- **羅馬語感**：解剖學 — 攻擊膝肘等關節
- **效果 T1**：對 DEF ≥ 8 的敵人忽略 **一半 DEF**
- **效果 T2**：忽略所有 DEF
- **效果 T3**：忽略所有 DEF + 30% 機率讓敵人「斷手」（ATK -50% 持續至戰鬥結束）
- **cost T1**：STR 50 + DEX 100（req STR 14 + DEX 20）
- **cd 3 / sta 10**

### 設計取捨
- **沒有純高傷招** — 空手就不該打高傷
- **4 招涵蓋四個情境**：防禦 / 反擊 / 控制 / 破甲
- **EXP cost 平均偏低**（拳法應該便宜），但**屬性門檻嚴格**（不是隨便能學）
- **AGI 需求都很高** — 強化「閃避流」的 build identity

## 🎮 對抗匕首流的設計

匕首流（focus route）：高 SPD、高 CRT、高 EVA、輕甲。
玩家空手對上匕首：

### 解法：赤手奪刃 + 集中槽
- 匕首流 focus 槽累積快（命中 +6、暴擊 +12、閃避 +8）
- 玩家如果也走 focus（拳法走 focus 路？） — 兩邊鬥誰槽先滿
- **建議：拳法走 focus route** — 跟匕首流對等
- 集中槽滿時觸發特技 **「絕對瞬息」** — 1 回合 100% 命中 + 暴擊
- 玩家用此 1 回合 + 赤手奪刃 (T3 100% 格擋) → 完美防禦反擊機會

### 高級反擊概念（你提的「集中槽滿時短時間 100% 反擊」）
這個其實是 focus route special「**絕對瞬息**」的延伸版。可以加碼：
- 標準絕對瞬息：1 回合命中+暴擊
- 拳法版（手持 fists 時）：1 回合 100% 反彈傷害（強化版赤手奪刃 + 借力反摔合一）

寫進 testbattle.js 的 SPECIALS 配置即可、不需新事件。
