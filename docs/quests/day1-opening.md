## 完整流程

### 🌑 場景 1：牢房裡（黑幕中、剛被吵醒）

**純感官建立 — 夏日清晨**（打字機速度）：

```
（雞啼。遠遠一聲。）
（夏天的悶——空氣還沒動。）
（你的後頸都是汗。）

（你舔了舔嘴唇——甜的。是血。）
（你的牙齒有一顆在晃。）
（你翻身——身上每個關節都痛。）
（昨晚的瘀青還在，有些還腫。）

（天色才剛亮。不知為何就要起床了。）
（你的心情——連罵人的力氣都沒有。）
```

這段純粹是：熱、痛、血、晃的牙、夏天的悶。**沒有敘述者、沒有外面訓練聲**（剛天亮、玩家還在牢房裡、武器沒在響）。
建立：**身體已經被操過一晚了**的疲憊感 + 夏季感。


### 🎯 場景 3.5：**特性即時互動** 🆕（關鍵）

走出牢房往訓練場的路上，經過一個小男孩。他比玩家年輕，臉上有淚痕，腳下是破鞋。他抓著自己的手腕在發抖。

**依特性自動觸發不同互動**：

#### kindness 玩家（預設所有玩家都有）
```
（你停下腳步。）
（那男孩看起來比你小得多。他的眼睛在哭。）
（你沒多想——伸手把他扶起來。）

玩家：⋯⋯深呼吸。你站得起來。

男孩（小聲）：⋯⋯謝謝。
男孩（小聲）：我叫⋯⋯達吉。

（你聽見後面有腳步聲。是一個捲髮的年輕人——他剛好看到這一幕。）
（他對你笑了一下。眼睛先彎的那種笑。）

→ 效果：dagiSlave 好感 +15、orlan 好感 +10、flag 'day1_helped_dagi'
```

#### cruel 玩家（少見，通常要從事件累積）
```
（那男孩擋在你路上，縮著身子哭。）
（你沒思考——踢了他一腳。）

男孩：嗚啊——！

（他摔在地上。獄卒在遠處看見了，笑了一聲。）
（捲髮的年輕人也看見了。他的眼神變了。）

→ 效果：dagiSlave 好感 -20、orlan 好感 -15、overseer 好感 +3、flag 'day1_kicked_dagi'
```

#### baseline（沒 kindness 也沒 cruel — 預設都有 kindness，所以這很少發生）
```
（那男孩擋在你路上。你繞過去，沒停。）
（他看了你一眼。你沒看他。）

→ 效果：dagiSlave 好感 -3
```

**這段設計意圖**：
- 玩家**看得到自己特性的第一次作用**
- 奧蘭跟達吉的**起始好感從這裡來**，不是憑空 25/10
- 未來遊戲裡 NPC 對你的反應 → 玩家記得這個瞬間
