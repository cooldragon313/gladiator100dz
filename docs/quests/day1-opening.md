## 完整流程

### 🌑 場景 1：牢房裡（黑幕中、剛被吵醒）

**純感官建立 — 夏日清晨**（打字機速度）：

```
（雞啼。遠遠一聲。）
（夏天的悶——空氣還沒動。）
（你的後頸都是汗。）

（你舔了舔嘴唇——甜的。是血。）
（你的牙齒有一顆在晃。）
（你翻身——身上每個關節都痛。）
（昨晚的瘀青還在，有些還腫。）

（天色才剛亮。不知為何就要起床了。）
（你的心情——連罵人的力氣都沒有。）
```

這段純粹是：熱、痛、血、晃的牙、夏天的悶。**沒有敘述者、沒有外面訓練聲**（剛天亮、玩家還在牢房裡、武器沒在響）。
建立：**身體已經被操過一晚了**的疲憊感 + 夏季感。

### 🌆 場景 3.5a：石走廊（Spartacus 氛圍）🆕

**寫作原則（v3 調整）**：
- 用**玩家看到的具體細節 + 內心疑問**，不用敘述者的形容詞
- 疤、假手 = 視覺伏筆（未來玩家自己也會有）
- **赫克特第一次出聲** — 假裝友善接近新人（反差伏筆）

```
（你走出牢房門。）
（眼前是一條長長的、陰暗的石廊。）
（空氣比牢房裡還悶。汗馬上又流出來了。）
（牆上每隔幾步有一根火把。光線跳動。）

（走廊裡已經有一些人——不是新進的。）
（他們站著、靠牆、或拿著東西經過。）
（他們盯著你。）
（⋯⋯這些都是誰？）

（你不自覺放慢了腳步。）
```

#### 沿途 NPC（視覺細節 + 內心疑問）

**① 疤臉站牆邊（視覺伏筆：疤痕系統）**
```

（他的左臉有一條從眉骨到下巴的疤——又深又粗。）
（⋯⋯這疤看起來真恐怖。是怎麼來的？）
（他看你的時候連眼皮都沒眨。）
```

**② 木頭假手（視覺伏筆：受傷後果）**
```

（木頭的指節還會動。他跟你四目相交——你先移開。）

（一個人走過。）
（他戴著一條黑鐵鑄的頸圈。）

（旁邊有人小聲說：）

**⑥ 赫克特：假裝的友善** 🆕（反差伏筆）
```
（再走幾步，一個笑嘻嘻的男人靠過來。）
（他的手拍了一下你肩膀——有點太熟了。）

赫克特：哎喲，新來的。
赫克特：別緊張嘛。我叫赫克特。
赫克特：這裡不難混——只要跟對人。

讓玩家**相信**赫克特是好人——反差留給未來的事件揭露。

#### 這段設計意圖
- **Spartacus 氛圍**：具體視覺細節（疤、假手、火把、石廊）建立世界
- **疤 + 假手** = 疤痕系統的視覺伏筆（未來玩家自己也會有）
- **「血狼」伍爾克** — Day 1 埋下的前十種子鬥士伏筆
  - 未來可能是 Day 75 宿敵會戰或 Day 100 萬骸祭對手
  - **玩家不提「百日祭」**（Day 1 還不知道）
- **赫克特假善意第一次出場** — 反差設計
  - 玩家的「暖」是錯覺
  - 赫克特 baseAff 是 -10，他是反派
  - 未來玩家發現他不是好人時會想起這一刻

### 🎯 場景 3.5b：**特性即時互動** 🆕（關鍵）

繼續往訓練場走，經過一個小男孩（達吉）。他比玩家年輕，臉上有淚痕，腳下是破鞋。他抓著自己的手腕在發抖。

**依特性自動觸發不同互動**：

#### kindness 玩家（預設所有玩家都有）
```
（你停下腳步。）
（那男孩看起來比你小得多。他的眼睛在哭。）
（你沒多想——伸手把他扶起來。）

玩家：⋯⋯深呼吸。你站得起來。

男孩（小聲）：⋯⋯謝謝。
男孩（小聲）：我叫⋯⋯達吉。

（你聽見後面有腳步聲。是一個捲髮的年輕人——他剛好看到這一幕。）
（他對你笑了一下。眼睛先彎的那種笑。）

→ 效果：dagiSlave 好感 +15、orlan 好感 +10、flag 'day1_helped_dagi'
```

#### cruel 玩家（少見，通常要從事件累積）
```
（那男孩擋在你路上，縮著身子哭。）
（你沒思考——踢了他一腳。）

男孩：嗚啊——！

（他摔在地上。獄卒在遠處看見了，笑了一聲。）
（捲髮的年輕人也看見了。他的眼神變了。）

→ 效果：dagiSlave 好感 -20、orlan 好感 -15、overseer 好感 +3、flag 'day1_kicked_dagi'
```

#### baseline（沒 kindness 也沒 cruel — 預設都有 kindness，所以這很少發生）
```
（那男孩擋在你路上。你繞過去，沒停。）
（他看了你一眼。你沒看他。）

→ 效果：dagiSlave 好感 -3
```

**這段設計意圖**：
- 玩家**看得到自己特性的第一次作用**
- 奧蘭跟達吉的**起始好感從這裡來**，不是憑空 25/10
- 未來遊戲裡 NPC 對你的反應 → 玩家記得這個瞬間

### 🌅 場景 4：走向訓練場（睜眼）

**[Stage.openEyes() — 黑幕掀開]**

**寫作原則（v3）**：用具體訓練器材（沙袋、木樁、兵器架）建立訓練場感。

```
（你走進訓練場。）
（腳底是沙——早晨的沙還是涼的，但空氣已經悶了。）

（場邊堆著幾個沙袋——有些已經破了，沙從裂口漏出來。）
（一排木樁立在中央。每根木樁上都有被砍出來的凹痕，密密麻麻。）
（兵器架上掛著練習武器——鈍的木劍、缺角的盾、無刃的斧。）
（有些柄上的木紋已經被磨得光滑。）

（你聞到血味、汗味，還有什麼東西在燒——是早餐的爐子？）
（一排比你先到的人站在場中央——他們的眼睛裡沒有顏色。）
（角落裡有人在練劍。每一下都像在砍一個他恨的人。）
（蟲在某個地方叫。夏天的蟲。）
（你站到隊伍後面。沒人看你。）
```

**器材細節的意涵**：
- 破沙袋、凹痕木樁、磨光的柄 = 「**這裡已經有人練了很久**」
- 木劍、缺角盾、無刃斧 = 新人練習器材簡陋
- 器材狀態暗示**訓練所**的歷史與規模
