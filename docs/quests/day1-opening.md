## 完整流程

### 🌑 場景 1：牢房裡（黑幕中、睜眼前）

**純感官建立**（文字顯示在黑底 + 打字機速度）：

```
（遠處有武器敲打的聲音。鐵錘落在鐵上。）
（更遠處，有人在吼——聽不清內容，只聽得出節奏。）
（你聞到身上的鐵鏽味。）
（你舔了舔嘴唇——甜的。是血。）
（你牙齒有一顆在晃。）
（雞啼。一聲。天要亮了。）
```

這段**沒有敘述者**、**沒有玩家聲音**。純感官。
建立：**這是角鬥場 / 監獄 / 有武器 / 有訓練**——但玩家自己拼湊。

### 💥 場景 2：獄卒踢門

```
砰——

（鐵門撞開。聲音在石牆之間彈了好幾下。你的耳朵嗡的一聲。）

獄卒：起床！臭奴隸！
獄卒：還在睡？太陽都曬屁股了！
獄卒：給我滾起來——去訓練場集合！
獄卒：今天起你是貨，不是人——懂嗎？

（他踢了你一腳。你沒感覺痛——背已經在痛了。）
```

### 💭 場景 3：玩家內心（依 origin）

依玩家 origin 輸出不同內心獨白：

#### 農家子弟（farmBoy，有 `diligence`）
```
（你沒吭聲。）
（你記得村子燒那天——獄卒也是這樣踹門的。）
（沒什麼新鮮的。）
```

#### 貴族（nobleman，未實作但預留）
```
（你被推倒的時候還以為自己穿著絲綢。）
（但你手底下的石地讓你想起來——那個世界已經沒了。）
```

#### 其他 / baseline（所有玩家都有 `kindness`）
```
（你站起來。）
（旁邊還有幾個跟你一樣的人。你看見其中一個在哭。）
（你沒說話——但你想說點什麼。）
```

### 🌅 場景 4：走向訓練場（睜眼）

**[Stage.openEyes() — 黑幕掀開]**

純感官描述 + 走路過程：

```
（陽光刺眼。你瞇起眼。）
（你腳底是沙。你聞到血味、汗味、還有什麼東西在燒。）
（訓練場比你想的大。）
（一排比你先到的人站在場中央——他們的眼睛裡沒有顏色。）
（角落裡有人在練劍。每一下都像在砍一個他恨的人。）
（你走進去，站到隊伍後面。沒人看你。）
```

### 🎭 場景 5：高台上的長官（資訊全從對話）

```
（一個人從高台走下來。他手裡拿著鞭子——但沒在用。）

???：都到齊了？

（他環視一圈。停在你身上半秒。——就半秒。）

塔倫長官：好。
塔倫長官：我叫塔倫。從今天起——你們是我的。
塔倫長官：我只說一次規則。
塔倫長官：吃早飯。訓練。吃晚飯。睡覺。再來一次。
塔倫長官：簡單得跟母牛一樣——
塔倫長官：——但能撐住的人不多。

（他停了一下。他的嘴角挑了一下。）

塔倫長官：五天後，有個「沙洗」。
塔倫長官：別問那是什麼——到那天你自己會知道。
塔倫長官：反正——三個人進去，兩個人出來。
塔倫長官：出來的那兩個人，希望有你。

（幾個人倒抽一口氣。你也是。但你沒出聲。）

塔倫長官：給我拿出吃奶的力氣練。五天。
塔倫長官：不然的話——
塔倫長官：——呵呵。

（他沒說完。但所有人都懂。）

塔倫長官：散了。
```

**關鍵資訊都從塔倫口裡**：
- ✅ 他叫塔倫
- ✅ 日常節奏（吃/練/吃/睡）
- ✅ 5 天後有「沙洗」（神秘名字）
- ✅ 「三個人進去，兩個人出來」— 生死暗示但沒明說
- ✅ 威脅（「呵呵」的停頓）

### 💭 場景 6：散場內心（依 origin）

#### 農家子弟
```
（你沒動。你腦子裡還在回想「三個人進去兩個人出來」。）
（你以前看過類似的場面——穀倉裡殺豬，兩隻活一隻死。）
（這裡大概也是。）
（你聽見旁邊有個年輕人小聲問同伴：）
???：那是什麼意思？

（你回頭——那是一個跟你差不多大的男生。捲髮。眼睛還亮。）
???：你好。我叫奧蘭。
奧蘭：以後⋯⋯請多指教。
（他伸出手。你看見他的手指在抖。）
```

#### baseline
```
（你站在原地。）
（五天。「沙洗」。三個人進去兩個人出來。）
（你腦子裡有聲音——但你分不清是你在說話，還是別人。）

（你感覺有人碰了你一下。你轉頭。）
???：你好。我叫奧蘭。
奧蘭：以後⋯⋯請多指教。
（他伸出手。你看見他的手指在抖。）
```

### 🎯 場景 7：與奧蘭第一次接觸（遊戲開始）

```
（你看著那隻手——關節粗糙、但不像訓練過的手。像個文員的手。）
（你伸手握了一下。他的手比你想像的還冷。）

奧蘭：⋯⋯那個「沙洗」——你知道是什麼嗎？
玩家：（無法選擇 — 你搖頭）

奧蘭：我也不知道。
奧蘭：但塔倫長官看起來不是開玩笑的人。
（他苦笑了一下。眼睛先彎起來的那種笑——但笑沒到眼睛。）

奧蘭：我們⋯⋯一起活下去吧？
```

**ChoiceModal**（或直接推進）：
- **「嗯。」**（握他的手）→ 奧蘭 aff +5、flag `orlan_day1_oath`
- **（不說話，點頭）** → 奧蘭 aff +3
- **「⋯⋯我還不認識你。」** → 奧蘭 aff -2、flag `orlan_day1_rejected`

完成 → 進入 Day 1 正常流程（早餐、訓練等）。

## 對比現有版本

### 現有問題
- ❌ 有敘述「眼前是訓練場。沙地。器械。還有一排跟你一樣眼神空洞的人。」— 旁白味太重
- ❌ 塔倫長官直接說「每天四個時段——訓練、吃飯、訓練、睡覺。」— 太 meta，像在讀遊戲手冊
- ❌ 「在這裡，你只有一個身分：活著的投資」— 好但太直白
- ❌ Day 5 只有「基礎考驗」這種遊戲感名字
- ❌ 沒有奧蘭 Day 1 首次接觸場景

### 新版改善
- ✅ 感官先行（鐵鏽、雞啼、血味）建立世界
- ✅ 塔倫的話接近「長官在訓話」的口氣，不像遊戲解說員
- ✅ 「沙洗」的神秘感 + 三人進兩人出的暗示
- ✅ 長官自報姓名（塔倫）
- ✅ 奧蘭 Day 1 首次握手 — 情感基底一開場就建立
- ✅ 依 origin 內心獨白 — 玩家感受到自己的身份

## 實作規劃

### Step 1：重寫 `_playDay1WakeUp`（main.js）
- 分成 7 個小段落（對應上面 7 場景）
- 每段用 DialogueModal.play，鏈式 onComplete

### Step 2：加 origin 檢查
- `Stats.player.origin` 或 `traits` 裡有什麼
- 依此挑選內心獨白版本

### Step 3：連結奧蘭 Day 1
- 現在好像還沒有奧蘭 Day 1 握手事件（or 有但很簡陋）
- 新版把這段接上去

### Step 4：Day 1 塔倫訓話結尾 → 設 timeline flag
- `Flags.set('timeline_sand_wash_revealed', true)` 讓百日條顯示 Day 5「沙洗」
- 同步百日條資料結構改動（見 timeline.md）

### Step 5：音效 / Stage 配合
- 獄卒踢門用 `sound.js` 的音效
- 雞啼可以用現有 Stage.playMorning 音效
