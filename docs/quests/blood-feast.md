## 👤 對手：快步馬爾斯（Mars）

### 角色設定

```
年齡：60 歲
出身：20 年前本城第一不敗角鬥士
現在：養在主人家的老名字，只拿出來展示
性格：認命、安靜、尊嚴
```

### 為什麼被挑

跟索爾同主題 — **沒價值**：
- 60 歲（遠超過角鬥士平均壽命）
- 腿速早就不行了（「快步」是諷刺）
- 主人當成「吹牛本錢」— 每次貴族來就拿出來說
- 用完即丟

### 戰鬥難度

- 比玩家弱（Day 49 玩家應該足夠強）
- 目標：**讓玩家贏**（非真考驗）
- 若真輸 → 主人震怒 → 另外懲罰路線（見後）

### 戰前對白（展示給貴族看）

```
主人阿圖斯站在貴族面前。

阿圖斯：「各位 — 今日讓你們看看『競技場的快步馬爾斯』。」
阿圖斯：「20 年前他是本城第一個不敗的。」
阿圖斯：「今日 — 讓他陪新人練練。」

（你看了馬爾斯一眼。）
（他站在那裡 — 鬢角全白。腿稍微拖著。）
（他對你點了一下頭。沒有敵意。）
（你突然懂了 — 馬爾斯不是威脅。他是祭品。）
```

## ⚔ 戰鬥流程

### Stage 1：你 vs 馬爾斯

- 戰鬥應該不難 → 玩家正常打贏
- 馬爾斯倒下時不求饒，靜靜躺著
- 你正想幫他止血

### Stage 2：貴族介入（關鍵）

```
貴族（聲音從觀眾席飄來）：
「這傢伙的『快步』——就這樣？名不副傳嘛。」
（他笑。但沒人笑。）
貴族：「腿還有用嗎？」

阿圖斯看你。
阿圖斯：「去。」
阿圖斯：「把他的腿砍下來。拿給大人看。」

（所有人看你。馬爾斯躺在地上 — 他聽見了。）
（他的眼睛看向天空。）
```

**玩家震驚點**：你不是打贏了嗎？為什麼還要砍他的腿？
**答案**：這是貴族的**娛樂**。馬爾斯的用途從一開始就不是打，是**被處決**。

## 🎯 玩家選擇

### 選項 A：切

```
你走到馬爾斯身邊。
他抬頭看你。

馬爾斯：「⋯⋯痛快點。」
（他的眼睛沒有恨 — 只有疲憊。）

（你揮下。）

馬爾斯：「⋯⋯謝。」
（他閉上眼。）

（你把腿提到貴族面前。血順著你的手滴。）

貴族鼓掌：「好魄力。」
貴族：「上次那案子 — 後天我那簽。」

（主人阿圖斯微笑。）
（你第一次看見他笑。）
（你懂了 — 這整場戲是為了這筆生意。）
（馬爾斯的腿 = 貴族的簽名。）
```

#### 效果
- 主人好感 +15
- 貴族好感 +10（隱藏 NPC，未來事件用）
- fame +5
- mood **-30**（情緒崩潰）
- **moral mercy negative weight:3 lock:true**（永久烙印）
- 特性獲得：`bloody_hands`（雙手染血）
- flag: `cut_mars_leg`

#### 在場 NPC 反應

- 奧蘭（若在場）：**好感 -20**，看你的眼神變了
- 梅拉：**好感 -15**，之後不再多給你麵包（暗中）
- 老默：**好感 -10**，冷淡但繼續治療
- 葛拉：**好感 -8**，沉默（他理解你為什麼做，但失望）
- 卡西烏斯：**好感 -15**，後續 storyReveal 對白有冷意

---

### 選項 B：拒絕 / 求情

```
你停下。你的刀在手裡。

你：「⋯⋯他已經輸了。」
你：「沒必要。」

（全場安靜。貴族看你。主人看你。）

阿圖斯：「你說什麼？」
（他的語氣沒有變 — 但整個空氣降了幾度。）
```

**接下來根據主人好感 + fame 分岔**：

#### B-1：不足（aff < 50 OR fame < 30）→ 被打到昏倒

```
阿圖斯：「監督官。」
（監督官走過來。鞭子在他手裡。）
阿圖斯：「讓他想起自己是什麼。」

（第一鞭。你跪下。）
（第二鞭。你倒下。）
（第三鞭。你開始咳血。）
...（連續 5-8 鞭）

貴族笑了。
貴族：「嗯 — 大人。那腿⋯⋯」
（主人點頭。監督官把你拖開 — 另一個侍從去切馬爾斯的腿。）

（你在地上看著。你聽見馬爾斯的慘叫。）
（你聽見貴族說：「後天我那簽。」）
（你昏過去前最後看見的 — 是阿圖斯轉身走了。）
```

**效果**：
- HP **-50**（重傷）
- **軀幹重傷**（wounds.torso.severity=3）
- 全身**輕傷** × 3（arms/legs/head 各輕傷）
- mood **-40**
- 主人好感 **-20**
- fame **-5**（在貴族面前失態）
- 貴族看你的眼神：嫌惡
- **moral mercy positive weight:3**（你守住底線）
- **moral reliability negative weight:2**（你破壞主人的局）
- flag: `refused_blood_feast` + `punished_by_overseer`

#### B-1 後續：職業 NPC 協助復原（有好感才給）

- **梅拉（好感 ≥ 40）**：「孩子⋯⋯吃。」深夜送食物 → food +30
- **老默（好感 ≥ 50）**：「你這種傻子⋯⋯」連續 3 天急救 → 各部位 severity -1
- **奧蘭（永遠）**：每晚坐你床邊 → mood 恢復 +5/天

**但要痛感** — 協助不是魔法：
- 你還是會痛
- 還是會夢到馬爾斯的慘叫
- 還是會記得主人轉身的樣子

#### B-2：足夠（aff ≥ 50 且 fame ≥ 30）→ 主人救你（但代價）

```
阿圖斯沒動。
（全場靜。你低著頭等。）

阿圖斯（轉向貴族）：「大人 —」
阿圖斯：「讓他們留著玩。」
阿圖斯：「我另有好東西給您看。」

（他拍手。侍從端上酒跟水果。）
（貴族還在笑，但眼神已經涼了。）
貴族：「⋯⋯也行。」

（你聽見阿圖斯走近你身邊。他的聲音沒有變。）
阿圖斯：「下次 — 你最好不要再浪費我的時間。」

（他走開。）
（貴族也轉開視線去吃東西。）
（你站起來。馬爾斯還躺在那裡。）
（他看你。他的眼裡有淚。）

馬爾斯：「⋯⋯謝謝。」
（這是他這一生最後說的話。）
（但那晚他睡著了 — 沒醒來。）
```

**效果**：
- 主人好感 **-10**（大降）
- 貴族好感 -8
- fame ±0
- mood -15
- **moral mercy positive weight:3 lock:true**（永久烙印）
- **moral reliability positive weight:2**（你敢對抗）
- flag: `saved_mars_but_paid` + `master_annoyed`

**馬爾斯隔夜死**（自然死亡 / 可能傷太重 / 或主人秘密處決）：
- flag: `mars_died_peaceful`
- 你獲得一小件馬爾斯遺物：`mars_token`（老舊的青銅護腕，+ACC 2）

**後續主人態度變冷**：
- 下次獎勵延遲
- 某些主人相關事件冷處理
- 但**你沒完全失寵** — 他還是算計過的
