> 「賭博一直是羅馬軍營跟競技場後台最常見的娛樂。三個杯子、一顆鉛球、看你眼力夠不夠快。」

## 📅 觸發機制

### 條件
- 夜間 slot 7（睡前那段）
- Day ≥ 8（先讓玩家熟悉訓練）
- 該日玩家還沒做夜間任務
- 至少 1 個 teammate 在場
- **基礎 12% 機率** + 玩家 LUK 高 → 提升（更多人來邀）
- **5 天 cooldown**（避免天天有人邀）

### 找你的人
- 隨機抽 1 個在場 teammate
- recruit（無名 NPC）也可邀
- 永駐 NPC（orlan / cassius / hector）對白會更個性化

### 入場 Stage 對白
```
（夜了。其他人圍在火堆邊。）
（XXX 走過來、手裡轉著三個破陶杯。）
XXX：「來嘛、賭一把睡覺前。」
XXX：「三杯一球、一場 5 銅錢。猜對你拿、猜錯我拿。」
XXX：「玩三場、看誰幸運。」
```

ChoiceModal：
| 選項 | 條件 | 結果 |
|---|---|---|
| 「賭。」 | 玩家 money ≥ 15（賭金 + 流動性）| 進入小遊戲 |
| 「不睡了我累。」 | 任何人 | 退出、不影響任何狀態 |
| 「⋯⋯我沒錢。」 | 自動顯示（money < 15）| **NPC 嫌棄、aff -1**：「⋯⋯廢物。連賭都沒本錢。」 |

## 🥚 三杯藏球小遊戲 — UI 流程

### 可實作版本
1. **三個杯子並排**（HTML div + CSS）
2. **球放進第 X 個杯**、玩家看到（球在杯下動畫）
3. **杯子蓋下、開始洗牌動畫**
4. **洗牌速度依對手 DEX**（DEX 越高、洗越快）
5. **洗完 → 玩家點擊一個杯子**
6. **開蓋 → 球在/不在**
7. **進下一場**（best of 3）

### 洗牌速度公式（初版、實測再調）
```
shuffleDuration = max(800, 2500 - oppDEX * 80) ms
shuffleCount    = min(15, 3 + Math.floor(oppDEX * 0.3))
```

| 對手 DEX | 洗牌時間 | 洗牌次數 | 玩家難度 |
|---|---|---|---|
| 10 | 1700ms | 6 次 | 容易 |
| 20 | 900ms | 9 次 | 中等 |
| 30 | 800ms | 12 次 | 困難 |
| 40+ | 800ms | 15 次 | 靠運氣 |

### 公平機制（球真的會跟著移動）
- 每次洗牌 = 隨機 swap 兩個杯子（程式追蹤實際球位）
- 球真的會跟著移動、不作弊、玩家可以追
- 玩家 DEX 高 → 動畫播 0.85x slow（眼睛跟得上）
- DEX < 15 → 動畫正常速
- DEX ≥ 25 → 0.85x slow
- DEX ≥ 35 → 0.7x slow（明顯放慢）

→ 雙方 DEX 對撞、玩家高 DEX 抗高 DEX 對手

## 🎰 三場制 + 結算

### 場 1（最簡單）
- 對手 DEX -10 buff（笨手笨腳熱身）
- 全速 1.5x（更慢）
- 基本上會贏

### 場 2（中等）
- 對手 DEX 原值
- 正常速度

### 場 3（困難 — 防刷）
- 對手 DEX +5 buff
- 全速 0.8x（更快）
- 就算高 DEX 玩家也只有 ~50% 猜中

### 結算

| 結果 | 賭金結算 | 額外獎勵 |
|---|---|---|
| **全勝（3:0）** | +15 銅錢（全贏）| **LUK +1**（永久）+ mood +8 + NPC aff +3 + 對白「看來我挺幸運」 |
| 2 勝 1 敗 | +5 銅錢 | mood +3 |
| 1 勝 2 敗 | -5 銅錢 | mood -3 |
| 0 勝（3 敗）| -15 銅錢（全輸）| mood -8 + NPC aff -1（被笑）|

**賭金規則**：
- 每場固定 5 銅錢
- 猜中：對手付你 5 銅錢
- 猜錯：你付對手 5 銅錢
- 三場結算 = 淨輸贏

## 🌟 連勝獎勵 — 幸運之星

### 累計全勝次數
玩家全勝 = 3:0 那次。**累計達 5 次** → 自動授予「**幸運之星**」技能。

```js
luckyStar: {
  id: 'luckyStar', name: '幸運之星',
  type: 'passive',
  desc: '累計賭局全勝 5 次。LUK 永久 +5。',
  storyOnly: true,
  grantedBy: 'gambling_5_perfect_wins',
  passiveBonus: { LUK_BASE: 5 },   // 永久基礎屬性加成（非派生）
}
```

授予時對白：
```
（你連著贏五場了。）
XXX：「⋯⋯小子。你他媽的是幸運之神看上的吧。」
XXX：「⋯⋯算了。我輸得心服口服。」
（你心裡也不知道是不是真的幸運。）
（但你開始覺得——也許運氣是站你這邊的。）
✦ 你獲得了【幸運之星】 — LUK +5。
```

設計意圖：
- 純粹獎勵堅持玩 LUK build 的玩家
- 5 次全勝 = 至少 25 天遊戲時間（5 天 cooldown）
- 不是隨便能拿、要技巧 + 屬性 + 運氣
