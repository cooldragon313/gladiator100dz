## 📌 核心定位（2026-04-20 重新定位）

### 主題

**震撼教育 — 讓新人「忘了過去，在這重新做人（戰士）」**
- 命是主人的
- 有沒有價值 — 由主人定
- 主人要誰死，誰死

### 場景本質

**公開處決儀式**：
- 所有奴隸圍觀（殺雞儆猴）
- 順便清掉幾個老鳥（沒價值的）
- 指定這批新人中一個淘汰
- **阿圖斯首次登場**（從陽台，不下來）

### 為什麼是索爾

**索爾是主人刻意買來當祭品的**：
- 28 歲（不可塑，教不好新戰士）
- 右腿瘸（戰力根本不行）
- 有女兒（分心、不專注）
- 便宜 → 主人用低價買來當消耗品

**阿圖斯 = cold_investor**，純功利。**沒有投資價值 = 淘汰**。

### 情緒目標

玩家**五分鐘後仍在想**：
- 「索爾一來就知道自己是祭品⋯⋯」
- 「下次沒價值的就是我⋯⋯」
- 「女兒 D 以後怎麼辦⋯⋯」

## 🎬 儀式開場（2026-04-20 新版 — 公開處決）

```
Day 5 清晨。訓練場中央。

（你跟奧蘭、索爾被推到中央。）
（但不只你們三個。）
（場邊還有 5-6 個老鳥被押過來，身上有傷 — 昨晚被打過。）
（其他奴隸幾十個圍成一圈，靜默。）

塔倫長官站在中央。

塔倫長官：「今天是新人的第五天。」
塔倫長官：「阿圖斯大人說 — 沒有人有資格混飯吃。」
（他的目光掃過全場。）
塔倫長官：「每過一段時間，我會淘汰不值得養的。」
塔倫長官：「今天，輪到這幾個。」

（你這時才懂 — 你們是被當例子的。）
```

### 阿圖斯首次登場（陽台）

```
（你第一次抬頭看陽台。）
（他站在那裡。）

（一個高大的身影。手裡拿著酒杯。）
（臉看不清 — 陽光太強。）
（他沒說話。只是看著。）

塔倫長官：「大人說 — 這幾個老的，時間到了。」
（5-6 個老鳥被侍從帶走。沒有掙扎。他們早就知道。）

（你聽見遠遠的木牌撞擊聲。索爾的掛件。）
（他的手在胸前握緊那塊木牌。）

（然後塔倫轉向你們。）
```

**設計重點**：
- 阿圖斯臉**看不清**（陽光 / 距離）→ 保留神秘
- 老鳥先清 → 讓觀眾看到「淘汰不是新人特惠」
- 索爾**握緊木牌**（女兒的） → 伏筆：他知道自己輪到了

### 塔倫宣判索爾

```
塔倫長官：「新人 —」
（他抬頭看陽台，等指示。）

（阿圖斯的目光從酒杯上方掃過你們三人。）
（停在索爾身上 — 1 秒。）
（然後移開。）

塔倫長官：「索爾。出局。」

（沒有理由。沒有解釋。）
（大人判斷 — 這個不值得繼續養。）

（索爾沒反應。他早就知道。）
（他低頭看自己的瘸腿一眼 — 像是跟它道別。）

塔倫長官：「剩下兩個 — 你們決定誰動手。」
塔倫長官：「打贏的留。打輸的 — 也一起走。」
塔倫長官：「這是給所有人看的。」

塔倫長官：「你們的命，是主人的。」
```

**主題句**：「你們的命，是主人的。」→ 會在 Day 49 血戰宴會再次迴響。

## 🆕 C-輸 路徑完整設計（本文件核心）

### 情感核心

> **你盡力了，但世界不給你機會。**

跟 A-輸的差異：
- **A 輸**：索爾因你弱而贏 → **索爾替你求情**
- **C 輸**：你獨自扛起全部 → **無人替你承擔** → 悲壯

這是「英勇的失敗者」原型 — 玩家會記一輩子。
