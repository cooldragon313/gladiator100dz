## 設計決策變更紀錄

### 2026-04-15 — DESIGN.md 大型重構

**背景**：DESIGN.md 累積到 6403 行，多處 checkbox 狀態漂移（例：Phase 1-E.2 已實作但 E.10/E.11 仍寫「待實作」），新協作者難以理解專案全貌。

**決定**：
- 按功能模組重組為九章節（技術架構 / 核心系統 / 人物 / 世界觀 / UI / 開發規範 / 設計哲學 / 內容模板 / 路線圖）
- 頂部新增 TO DO / CHECK LIST（動態焦點）和 TL;DR（新協作者 5 分鐘入門）
- 已實作細節與廢棄設計移到本 `HISTORY.md`
- 內容創作模板（D.11）遷移到獨立 `docs/CONTENT-TEMPLATES.md`
- 建立「進度單一事實源」規則：進度只以頂部總表為準，其他章節若有 checkbox 都是歷史快照

**遷移進度**：
- [x] 第一輪（2026-04-15）：新 DESIGN.md 主體 + HISTORY.md / CONTENT-TEMPLATES.md 骨架
- [ ] 第二輪：D.11 模板完整遷移到 CONTENT-TEMPLATES.md
- [ ] 第三輪：Phase 0/1/D.7 等實作細節完整遷移到 HISTORY.md
