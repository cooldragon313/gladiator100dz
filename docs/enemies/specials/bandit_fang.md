# 毒牙（強盜團首領）

> 外出押運任務遇到的強盜團首領。粗暴、髒、嘴上掛著笑的那種。
> 對應程式：`TB_ENEMIES.bandit_fang`

## 背景 / 形象

- **出身**：流民出身，靠搶劫維生
- **戰績**：殺過至少 15 個商人、侍從、護衛
- **形象**：滿臉橫肉、歪牙（所以外號「毒牙」）、身上永遠有酒味
- **性格**：**嘴巴不停**，一邊打一邊罵、嘲諷
- **可否說服**：❌（太餓了）

### 為什麼他會出現
- 玩家的**名聲** ≥ 20 後，主人會派你陪侍從外出運貨
- 毒牙專搶貴族商隊
- 這是玩家第一次真正**走出訓練所**（Phase 2 中段）

## 戰鬥對白

### 戰前（遇到強盜）
```
（樹林裡傳來一聲口哨。接著是幾個人影從四面走出。）
（領頭的那個有歪牙。他一邊走一邊啃著什麼。）

毒牙：喲——？這輛車看起來有油水啊。
毒牙：把東西留下。人可以走。

（侍從已經嚇得發抖。他看著你。）
毒牙：欸欸欸，這個小角鬥士不錯嘛。
毒牙：不過呢——
毒牙：一個人打六個——還是算了吧。
（他笑得露出那顆「毒牙」。）
```

### 戰場開場
```
毒牙：來吧，角鬥士。
毒牙：讓我看看阿圖斯花錢買的玩意兒——
毒牙：值不值他的銅板。
```

### 戰中（他血量 ≤ 50%）
```
毒牙：呸！你這小子真會躲！
毒牙：但你別得意——
毒牙：我腳底下的血，還沒乾過呢！
```

### 戰中（他血量 ≤ 20%，被動觸發）
```
毒牙：啊啊啊啊啊——！！
（他的眼睛紅了。他的斧頭亂揮。他不在乎命中了。）
```

### 戰後 — 玩家贏
```
（毒牙倒在地上。他看著你，那個嘴還在嘻笑。）
毒牙：操……
毒牙：……還是被小鬼打死了。
（他閉上眼。沒再動。）

（手下們四散逃走。你轉頭——侍從在發抖。）
侍從：……大人會記得這件事的。
```

### 戰後 — 玩家輸
```
（毒牙笑著走過來。他的斧頭靠在肩上。）
毒牙：嘖——真遺憾。
毒牙：本來想留你一條命讓你回去跟阿圖斯哭的。
（他抬起斧頭——）

（但侍從大聲呼救，遠處有馬蹄聲。毒牙罵了一聲，帶手下跑了。）
（你被侍從拖回訓練所。重傷但還活著。）
```

## 相關事件 / 觸發條件

- Day 55-70 外出押運任務（fame ≥ 20 解鎖）
- 觸發 flag：`encountered_bandit_fang`
- 贏了：+25 銅、主人好感 +15、fame +5
- 輸了：HP 大掉、fame -3、主人好感 -5

### 衍生
- **贏了毒牙**：後續外出任務「路人」品質會升級（更強的對手出現）
- **毒牙的餘黨**：Day 80+ 有可能出現復仇事件
