## 背景 / 形象

- **出身**：不明。可能是「血環」公會成員（暗殺者工會）
- **形象**：全身黑、面罩只露眼睛、戴黑色指套
- **聲音**：低沉、不帶情緒
- **性格**：冷漠、專業、不閒聊
- **可否說服**：💡 **可以**（高 WIL + 特定對話選項可以說服他「你不是他的目標」）

### 為什麼他來殺你
- fame ≥ 30 後觸發
- 主人在商業圈樹敵了 — 敵人派刺客警告阿圖斯（殺他的新投資品）
- **玩家要推理出**：是誰派的？（中期懸念）
- 可能選項：莫拉斯？海龍幫？還是訓練所內部？

## 戰鬥對白

### 戰前（夜間事件，玩家剛睡著）
```
（你以為你在做夢。）
（但夢裡不會有金屬的冰冷。）
（你睜開眼——一把短劍抵在你喉嚨上。）

夜鷹：別動。
夜鷹：誰付錢的，別問。
夜鷹：閉上眼——痛就一下。
```

### 戰場開場（玩家反抗成功）
```
（你滾下床。他的劍劃過枕頭。）
（他沒生氣。他笑了一下——但嘴是閉著的。）

夜鷹：好。有反應。
夜鷹：那就——認真打。
```

### 戰中
```
夜鷹：你的訓練不錯。
夜鷹：——但我不是你該打贏的對手。
```

### 說服路線（戰前，高 WIL 選項）
```
玩家：你不知道是誰付的錢，對不對？
夜鷹：……
玩家：付錢給你的人——今天派你殺我。明天會派別人殺你。
玩家：你們這種人活得久嗎？

（他沒回答。他的劍停在你脖子上很久。）
夜鷹：（最後）……這次我沒看見你。
（他跳出窗外消失。你聽見遠處有貓頭鷹叫。）
```

### 戰後 — 玩家贏
```
（夜鷹躺在地上。面罩歪了。你看見那是一個女人的臉。）
（她年紀不大。眼睛是綠的——像你認識的某個人的眼睛。）
夜鷹：……別告訴阿圖斯。
夜鷹：別讓他知道我失敗了。
（她的意思你聽不懂。但她閉了眼。）
```

### 戰後 — 玩家輸
```
（夜鷹的劍停了。她沒殺你。）
夜鷹：……你運氣好。
夜鷹：今晚我突然不想殺人了。
（她跳出窗。你聽見遠處有貓頭鷹叫。）
（你醒來時已經是早上。沒人信你的話。）
```

## 相關事件 / 觸發條件

- Day 65+ 夜間（fame ≥ 30）
- 一次性，觸發後 flag `assassin_visited` 設定
- 觸發 flag：`nighthawk_met` / `nighthawk_beaten` / `nighthawk_spared`

### 衍生
- **說服或放走她** → 後期（Day 90+）她會出現作為盟友（隱藏結局條件）
- **殺死她** → 血環公會聽到消息，Day 90+ 可能派更強的人
- **被殺** → Game Over 試玩途中（但 Day 100 前都不會真死，改為重傷）

### 懸念（未解）
- 她是誰派的？（劇情留白）
- 她為什麼沒殺？（可以透過後期揭露）
- 她的綠眼睛讓玩家想到誰？（劇情 hook）
