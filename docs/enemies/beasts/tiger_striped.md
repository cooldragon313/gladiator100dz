## 背景 / 形象

- **種類**：山林大虎（成年公虎）
- **體型**：站立時超過成年男人
- **條紋**：橘底黑紋，左眼上有老傷疤（暗示牠跟人打過且活下來）
- **可否說服**：❌（牠是虎）
- **可否逃**：❌（跑不贏牠）

### 為什麼會遇到
- 主人愛面子，喜歡帶幾個角鬥士去「狩獵」
- 名義上是「娛樂」，實際上是炫耀武力
- 但森林是真的森林 — 不受控制
- 這場戰鬥**沒觀眾**（連侍從都躲樹上）

## 戰鬥對白（旁白形式）

### 戰前（主人打獵隊）
```
（樹林很深。空氣有土味和松針味。）
（主人的馬在前，你跟另一個角鬥士在後。侍從拿著弓。）

阿圖斯大人：這一帶的山林裡有大物。
阿圖斯大人：你們兩個——看表現。

（你們展開陣形。你聽見腳步——是什麼很重的東西。）
```

### 遭遇（第一聲）
```
（左邊的樹叢晃了一下。）
（一個很大的影子跳出來。）

另一個角鬥士（驚呼）：虎！！

（那是虎。條紋清清楚楚。黃眼睛盯著你。）
（牠比你想像的大得多。牠沒叫——只是盯著。）
```

### 戰中（HP ≤ 50%）
```
（牠喘得很重。但牠沒退。）
（左眼上的疤讓牠看起來像在瞇眼。）
```

### 戰中（HP ≤ 20%，牠快敗了）
```
（牠退了一步。）
（你聽見遠處有馬蹄聲——主人走了？）
（牠看了你一眼——然後跳入樹林消失。）
（你沒追。你沒力氣追。）
```

### 戰後 — 玩家打退牠（不殺）
```
（牠走了。沒人追。）
（你倒在地上喘氣。你的肩膀有四道爪痕，很深，但不致命。）

另一個角鬥士：……你還活著？
另一個角鬥士：我以為我要替你收屍了。

（你笑不出來。你聽見馬蹄聲——主人回來了。）

阿圖斯大人：……牠跑了？
阿圖斯大人：算了。你們還能走回去就行。
```

### 戰後 — 玩家殺死牠
```
（虎倒下。牠的黃眼睛還睜著。）
（你坐在地上，手還在抖。）

另一個角鬥士：……你殺了牠。
另一個角鬥士：用那把劍？

（阿圖斯大人策馬過來。他沒下馬。）
阿圖斯大人：把虎皮剝了。
阿圖斯大人：那是我的獎盃。

（他走了。虎的血在你手上乾。）
```

### 戰後 — 玩家輸
```
（虎跳上你。你覺得身體斷了。）
（突然有箭射過——侍從的箭，射中虎的脖子。）
（虎吼一聲跳走。）
（你倒在地上。侍從慌忙跑下樹抓住你。）
侍從：大人！大人！還有氣！
（你聽見阿圖斯大人的聲音從遠處傳來——很冷。）
阿圖斯大人：……抬回去。看看能不能用。
```

## 相關事件 / 觸發條件

- Day 70+ 陪主人打獵（fame ≥ 40）
- 觸發 flag：`encountered_tiger_striped`
- 贏了（殺死）：+40 銅、fame +10、主人好感 +20、獲得「虎皮」道具
- 贏了（打退）：+20 銅、fame +5、獲得「虎爪」道具（可打造武器）
- 輸了：HP 歸 1、獲得「爪痕」疤痕（永久 CON -1）、主人好感 -10

### 衍生事件
- **獲得虎皮** → 可以跟葛拉鐵匠換高級武器
- **獲得虎爪** → 可以打造特殊武器「斑爪短劍」
- **爪痕疤痕** → 結局敘述會引用（「你肩膀上那四道疤再也沒消過」）

---

## 設計意圖

- 玩家第一次面對**無法溝通的對手**（不是人類戰鬥邏輯）
- 虎的戰鬥節奏不同 — STR 18 + 撲擊技能需要新戰術應對
- 「**可以打退不需要殺**」的選擇 — 尊重生命 or 贏取獎盃
- 為 Phase 3 後日談的「困獸鬥」事件鋪墊

## 注意事項

- 虎的武器 `claws` 跟 `fangs` 要新增到 `TB_WEAPONS`
