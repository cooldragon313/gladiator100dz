## 背景 / 形象

- **出身**：海龍幫（不是訓練所的角鬥士，是黑幫代理人）
- **形象**：永遠戴黑布面罩 + 黑色寬鬆衣。連身高都看不準
- **戰績**：不詳 — 官方紀錄為 0，因為他的戰鬥都不在正規場
- **性格**：完全沉默。整場戰鬥不說話。只在少數情況會出聲
- **可否說服**：❌（他是合約殺手，不受情感左右）

### 為什麼他來打你
- 阿圖斯大人跟海龍幫有一筆私下交易
- 主人下注玩家贏 — 因為他想知道玩家能不能對抗「非正規選手」
- **玩家第一次面對風格完全不同的對手**（跟正規鬥士完全不一樣的戰鬥感）

## 戰鬥對白

### 戰前（侍從傳話）
```
侍從：大人要你去打一個……不是訓練所的人。
侍從：對方叫「影子」。海龍幫的。
侍從：規則跟平常不太一樣——
侍從：沒觀眾。沒計分。沒救援。
侍從：如果你贏了——大人很高興。
侍從：如果你輸了——大人也不會賠太多。
```

### 戰場開場
```
（你從來沒看過他的臉。他的身形在沙地上像一道黑痕。）
（他沒說話。只是舉起雙匕。）
```

### 戰中（血量 ≤ 30%，他的）
```
影子：……
（他第一次發出聲音。那像是笑。）
```

### 戰後 — 玩家贏
```
（你揭開他的面罩——發現裡面沒有你想像中的「一張臉」。）
（那是一個年輕得不像樣的男孩。可能比你小。）
（他還有氣。）
影子：你……下次小心。
（他的眼神關閉了。）
```

### 戰後 — 玩家輸
```
（影子把匕首從你喉嚨邊移開。）
（他沒殺你。）
（他把一個小東西塞進你手裡——是一顆糖。）
（他消失在沙地上。）
（你醒來時是在醫療區。老默看著你沒說話。）
```

## 相關事件 / 觸發條件

- Day 55 或 Day 62 賭局候補
- 高風險高回報：贏了 +30 銅、輸了 +HP 大掉
- 觸發 flag：`timeline_bet_shadow_triggered`

### 特殊路線
- **輸了但拿到糖** → 觸發懸念線（糖是誰給的？海龍幫為什麼？）
- **贏了且揭開面罩** → 觸發「他是誰」的內心獨白 + 晨思系統引用
- **Day 80+ 重逢**：如果贏了，影子可能再次出現，這次沒契約（他自己想再打）

---

## 設計意圖

- 給玩家「**外面的世界有比訓練所恐怖的對手**」的第一次震撼
- 對比鐵臂（堂堂正正）vs 影子（速度+神秘）— 戰鬥風格完全不同
- 黑幫勢力是玩家「看不見但真實存在」的世界 — 為 Phase 2 後期劇情鋪路
