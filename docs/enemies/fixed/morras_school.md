## § 1 莫拉斯訓練所概要

- **主人**：莫拉斯大人（與阿圖斯大人是「友好的死對頭」）
- **資歷**：比阿圖斯訓練所資深（可能歷史更久）
- **風格**：正統派角鬥士訓練，重視技藝傳承
- **跟玩家關係**：主要競技對手，**不是敵人**——比賽結束就散
- **與阿圖斯的關係**：商業競爭 + 私交不差，賭局頻繁

**戰士只在場外競技出現**（孤兒、賭局、對抗賽），**不會出現在場內 Day 5 沙洗等內部事件**。

## § 2 戰士池（8 位）

```
    ┌─ 鐵臂 (③ 中等 boss) ────兄弟/師父──── 蝗蟲 (① 初等)
    │     (30 歲, 當家)                        (16 歲, 學徒)
    │
    └─ 老跛 (⑤ 中等) ──────亦師亦友──── 夜鶯 (④ 中等)
          (40+ 歲, 老兵)                    (25 歲, 女性)

    大石頭 (② 初等)          獨立個體，敬重前輩

    阿圖斯影 (⑧ 精英 boss)    ⚠️ 來自阿圖斯訓練所 — 主人線伏筆
```

**精英 tier · Day 60-80 派遣主 boss**

- 35 歲、**曾是阿圖斯訓練所的戰士**
- 武器 `warHammer` / 護甲 板 T4（鐵板甲）
- 性格：沉默、眼神複雜
- 🔗 **重劇情錨點**：認識葛拉、可能認識塔倫、跟主人有未解恩怨

**屬性**（AGI 從 22 → **45**、DEX 從 20 → **28** 補重武器速度）：
```

```

**派生重點**：ATK 77、DEF 97（含板 T4）、PEN 21%、HP 186（爆肥）、**~2400ms/擊**
→ 每擊 30+ 傷害，爆擊 200+（一擊要人命）

**Boss 被動**：`remembers_training`
- 戰鬥中隨機時刻說出一句話，揭露他知道塔倫 / 葛拉 / 訓練場的事
- 對白例：「這個動作⋯⋯塔倫沒改過。」

```

```

```
阿圖斯影：……轉告他。
阿圖斯影：我沒忘記我欠他的。
```

**被殺後果**：🚫 **重要訊息永遠不會揭露**（葛拉過去、主人過去）

## § 3 對手池使用（OPPONENT_POOL）

派遣事件從此 pool 按 tier 抽人：

```js
OPPONENT_POOL = {
  elementary: ['morras_locust', 'morras_boulder'],         // 蝗蟲、大石頭
  intermediate: ['morras_ironarm', 'morras_nightingale', 'morras_limpwise'],  // 鐵臂/夜鶯/老跛
  advanced: ['morras_doubleface', 'morras_coldmoon'],      // 雙面/冷月
  elite: ['morras_shadow']                                  // 阿圖斯影
};
```

**程式實作**（對應 arena-system.md § 1.5）：
```js
function pickOpponent(tier) {
  const pool = OPPONENT_POOL[tier];
  const candidates = pool.filter(id => !MORRAS_POOL[id].dead);
  if (candidates.length === 0) {
    // 全都被殺光 → 招替補新人（tier -1）
    return generateReplacement(tier);
  }
  return candidates[Math.floor(Math.random() * candidates.length)];
}
```

### Pool 狀態欄位（搭配 arena-system.md § 3.4b）
```js
{
  id: 'morras_locust',
  dead: false,            // 被殺後 true
  wounds: [],             // 持久傷勢
  metPlayer: false,       // 見過玩家
  playerSparedMe: false,  // 玩家曾放過他
  encounterCount: 0,      // 對戰次數（1st/2nd/3rd 對白不同）
}
```


### 冷月 2nd 次
```
（他看你一眼。）
```

- 鐵臂：「上次你給我留了條命。今天我也會。」（WIL -3 戰力減弱 — 他打得沒以前兇）
- 老跛：「那份情我記得。打吧，孩子。」

## § 5 替補新人系統（全部死光時）

當 pool 某 tier 全部 `dead:true` → 觸發 `generateReplacement`：
```
侍從：上一個……在你手上死了。
侍從：這是莫拉斯新找來的。
（一個生面孔走上來。他看起來比之前那個弱一點。）
```

**替補屬性**：對應 tier 範圍**下限**（比原本弱 15-20%）、tier 標「新人」
- 例：鐵臂死 → 中等 tier 新人 STR 18、HP 60（弱於鐵臂的 25/75）

→ **殘忍路線的隱性優勢**：殺光對手池 = 後期越打越輕鬆
→ **代價**：失去所有故事線 + 仁慈軸全無正向累積

## § 8 備忘：boss 速度方案採納（2026-04-23）

針對「中等 boss 鐵臂 3.75s/擊、精英 boss 阿圖斯影 5s/擊」太慢問題，**採納 A 方案（高 AGI 補速）**：

- 鐵臂：AGI 25 → **35**（職業戰士合理範圍）→ **~2400ms/擊**
- 阿圖斯影：AGI 22 → **45**、DEX 20 → **28** → **~2400ms/擊**

保留原武器 swingTime 不變（不做 boss 專屬武器變體）。
