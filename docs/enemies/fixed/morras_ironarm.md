## 背景 / 形象

- **出身**：莫拉斯大人名下的角鬥士，比你早 3 年進場
- **戰績**：跨訓練所 22 勝 4 敗（場內聲望高）
- **形象**：三十出頭，左臂有一條從手腕到肘窩的舊傷疤（所以外號「鐵臂」）
- **性格**：沉默寡言但不冷酷。打贏會跟對手說「下次記得格擋」這類的
- **可否說服**：❌（他是專業鬥士，不受情感左右）

### 為什麼他是第一個對手
- 莫拉斯大人跟阿圖斯大人是「友好的死對頭」（商業競爭）
- 鐵臂是莫拉斯的驕傲 — 阿圖斯故意安排你跟他打，如果你贏了「他臉上好看」
- 第一次賭局通常阿圖斯下注「鐵臂贏」（保險做法）— 玩家是冷門下注

## 戰鬥對白

### 戰前（侍從傳話）
```
侍從：大人跟莫拉斯老爺打了個賭。
侍從：七天後——你跟他家的「鐵臂」打一場。
侍從：大人下注鐵臂贏。
侍從：但如果你贏了——大人會記得你的臉。
侍從：練熟一點。
```

### 戰場開場（鐵臂看著你）
```
鐵臂：新來的。
鐵臂：打起來不要裝死。
```

### 戰中（血量 ≤ 50%）
```
鐵臂：不錯。
（他沒有多餘的話。但他的眼神變了。）
```

### 戰後 — 玩家贏
```
鐵臂：……下次記得格擋。
（他沒倒下。他站著被侍從攙扶走。）
```

### 戰後 — 玩家輸
```
鐵臂：活下去就好。
（他沒有補刀。他的手伸下來扶你起來。）
（那個握手比他的槌子還重。）
```
