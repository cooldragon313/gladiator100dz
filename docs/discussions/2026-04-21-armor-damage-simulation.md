### Scenario A 觀察

✅ **優點**：
- 武器 PEN 對板甲差異**已經有體現**（長槌/長槍對板甲比短劍明顯有效）
- 重武器（長槌/重斧）穩定 11 擊左右殺死板甲，合理

⚠️ **問題**：
- 匕首對鏈甲以上 = **1 傷害（被 floor 擋住）**，等於廢物
- 短劍對板甲 = 1 傷害，也廢
- 輕武器玩家對重甲**毫無希望**，沒有「我準備對了就能贏」的感覺
- 板甲 vs 輕武器 = 純耐力戰，沒戰術

→ **現有系統「重武器強壓輕武器」太線性，沒有 Outward/BB 的策略感**。

## § 2 Scenario B — 加 armor HP 雙血條（推薦）

### 新加機制

1. **每件護甲有 armorHP 池**（獨立於身體 HP）
2. **每次攻擊同時造成**：
   - 身體傷害（原公式）
   - 護甲傷害 = `ATK × 武器類別倍率`
3. **armorHP 歸零後**：armor.DEF 失效、匕首觸發 puncture（身體傷害 ×2）

### 武器類別對 armor 倍率

| 武器類 | armor 倍率 | 備註 |
|---|---|---|
| 鈍器（槌/長槌）| ×1.5 | BB 式破甲專長 |
| 斧（heavyAxe）| ×1.2 | 劈裂 |
| 劍（短劍/長劍）| ×0.8 | 砍甲效率差 |
| 長槍（polearm）| ×1.0 | 平均 |
| 匕首 | **×0.3** | 破不動，但破後 puncture ×2 |

### 模擬：vs 板甲（DEF 14, armorHP 120）

**Phase 1**（甲完好）：身體傷害照原公式

**Phase 2**（甲破）：DEF -14 = 30（base CON+STR）；匕首 puncture ×2 身傷

## § 4 結論 / 建議

### 現有 armor.DEF 數字（0/4/8/14）直接沿用

- 加上 armorHP 池後，**這組數字剛剛好**
- 不需要改 DEF，也不需要用 armors.js 那套（12/22/38 太大）
- `src/content/armors.js` 的顯示數字需要**跟著改成 0/4/8/14**以同步

### 加 armorHP 後的戰術感

- 匕首玩家對板甲不再是**完全廢**，是**打持久戰 + puncture 爆發**
- 鈍器玩家對板甲**保持碾壓**（設計意圖）
- 劍類玩家是**中庸派**（哪都能打但沒特別強）
- **玩家準備對（武器 vs 敵甲）= 戰鬥時間減半**

### 三類護甲 × 5 tier 擴充（給葛拉線用）

基於 A/B 模擬，建議的 5 tier 數值：

| Tier | 布 | 皮 | 板 |
|---|---|---|---|
| T1 | DEF 1, armorHP 8 | DEF 3, armorHP 25 | DEF 10, armorHP 80 |
| T2 | DEF 2, armorHP 12 | DEF 5, armorHP 40 | DEF 13, armorHP 110 |
| T3 | DEF 3, armorHP 18 | DEF 7, armorHP 55 | DEF 16, armorHP 145 |
| T4 | DEF 4, armorHP 25 | DEF 10, armorHP 75 | DEF 19, armorHP 185 |
| T5 | DEF 5, armorHP 35 | DEF 13, armorHP 100 | DEF 22, armorHP 240 |

**設計邏輯**：
- 布：DEF 很低，但有 armorHP 當緩衝（輕但不是無用）
- 皮：對應現有 leather 數字（T1-2 附近）
- 板：T1 起跳就是現有 chainmail 附近，T5 比 ironPlate 高
- 每 tier 約 +30-40% armorHP、+2-3 DEF

### 副屬性（SPD / EVA / AGI penalty）

沿用之前提案：
- 布 T1-5：SPD +1 到 +4，EVA +2 到 +8
- 皮 T1-5：SPD 0 到 -2，EVA 0 到 +2
- 板 T1-5：SPD -3 到 -15，AGI 0 到 -3，EVA -3 到 -12
