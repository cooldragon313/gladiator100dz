| 項目 | 內容 | 狀態 |
|---|---|---|

### 1.2 帝國的「個性」🔒

**外表富裕、實質腐爛**（晚期羅馬風）。

- **重利不重義** — 帝國擴張靠功利計算
  - 偽旗滅村被當成「平亂」嘉獎、不是醜聞
  - 領主升任只看戰功、不看手段
  - 對奴隸制視為理所當然的經濟基礎
- **公開殘酷的儀式感** — 暴力被儀式化、變成全民娛樂
  - 競技場是**全國共有的文化制度**、不是某個變態主人的私癖
  - 處決犯人、奴隸對打、人虎獸鬥都是**公開節目**
  - 平民百姓週末就去看 — 跟現代去看球賽一樣自然
- **表面榮耀、底下腐朽** — 元老院 / 最高層**裝著公正**、底下全是政治算計
  - 「為民除害」「平亂有功」這類詞**只是包裝**、底下都是利益交換
  - 30 年前的政變、幾個月前的偽旗、都是同一套邏輯不同手法
- 整體美學：金碧輝煌的領主府、繁華的市集、底下卻是奴隸經濟、政治謀殺、儀式化暴力

### 1.3 為什麼人民愛競技場 🔒

競技場 = **發洩 + 社交 + 社會刑罰** 三合一。

| 階層 | 為什麼來看 |
|---|---|
| **平民** | 唯一能合法觀看「強者生死」的場合、發洩日常被欺壓的怒氣、買票或免費入場的全民娛樂 |
| **商人 / 中產** | 應酬場合、跟同階級認識的人約在這、賭金錢輸贏 |
| **貴族 / 主人們** | 政治社交場、**看誰下注押對人 = 誰有眼光 = 誰值得結盟** |
| **領主 / 高官** | 收買民心的工具、舉辦大會 = 展示「我管的城市熱鬧繁榮」|
| **奴隸（玩家）** | 唯一可能爬升的窄門 — 打贏 100 天 → 自由身 / 名聲 / 重新做人 |

**核心定位**：競技場 = **帝國的暴力閥**。把街上隨機暴力擠壓進這個密閉場、變成可賣票、可下注、可娛樂的演出。

#### 「社會刑罰」這層的重要性 🔒

**競技場同時是執行死刑的方式之一**。罪犯、戰俘被丟進去 = 帝國合法處決機制。觀眾看死人不只是娛樂、也是**司法儀式**（死的人「該死」、為觀眾洗罪惡感）。

→ 帝國不必設斷頭台 / 絞刑場 — 一切刑罰都導向競技場。
→ 訓練場業者**有經濟動機**去採購戰犯 / 罪犯：他們是可參賽的合法消耗品 + 主人賺名聲跟金錢的素材。

### 1.4 奴隸來源四類 🔒（從 1.3「社會刑罰」延伸）

訓練場主人不是隨便抓人，是**有市場機制**的採購：

| 類型 | 說明 | 代表 NPC |
|---|---|---|
| **戰俘** | 對外戰役後產生、被歸類為「敵國俘虜」 | **玩家**（被當「敵國軍隊抓的俘虜」、實際是提圖斯部下偽旗事件後抓他賣到本城）→ 是冤獄式假戰俘 |
| **罪犯** | 司法判決送進競技場 = 合法處決方式 | Hector 可能是這類（背後一條長疤、砍他的人沒活下來）|
| **債務奴 / 平民賣身** | 破產 / 欠債 / 父母為籌錢賣孩子 | 索爾（推論為農夫債務奴）/ 凱德（成年後被村莊賣的）|
| **主人內部處置** | 主人家僕人犯錯、降級到訓練場 = 自家內部的奴隸調動 | 奧蘭（主人家傭人 6 年 → 偷錢 → 降級訓練奴）|

🔥 **玩家身份的雙重諷刺**：他是「假戰俘」 — 被自己國家滅村然後抓來歸類為「敵國俘虜」、賣到本城。Day 65 認出提圖斯臉時、不只是「殺我家鄉的人」、還是「**把我抓來當奴隸的人也是他**」。

⚠️ 待確認：

- 「**宗教意涵**」要不要加？古羅馬 munera 原本有「祭祖」意義（後來才商業化、變娛樂）。這個遊戲帝國的競技場是純世俗 / 還是有「祭神」儀式包裝？Livia 信奉某女神是埋的線索、可能要往這方向接。

### 2.3 阿圖斯立場 🔒

**純冷血商人、不參與政治派系**。
- 30 年前政變對他 = 政權更迭、跟我做生意無關
- Day 65 巴結提圖斯 = 純粹商業（新領主有實權、不得罪）
- 庫房就算有德拉格家東西也是商品 / 轉手戰利品、不是政治信物
- 他**從不在意**誰當領主 — 「誰當都要找我訓練奴隸」
- 他跟蓋烏斯 30 年情義是**個人**恩怨情仇、不牽政治

**對玩家的劇情含意**：阿圖斯不是「直接反派」、是「結構共犯」。他從不下手害你、但他這個生意機器**就是這套體制的根基**。玩家恨提圖斯比恨阿圖斯容易、但實際上兩個人都讓這套帝國運轉。

## § 7. NPC 愛恨情結地圖

### 7.1 訓練所內的「好人圈」vs「冷血派」

```
好人圈                          冷血派
───────────────                ───────────────

塔倫長官 ─┬─ 暗中獻計把巴爺送進險境 ─┐
          │                          │
          └─→ 「救」巴爺 → 升長官    │
                                     ▼
                              巴爺重傷退役
                                     │
                                     ▼
                          至今真心感激塔倫（不知真相）
```

**巴爺 v10 主線**：玩家逐步接觸 → 偷聽密謀 → 巴爺喝酒透漏 → 玩家手上有抉擇（揭穿 / 沉默）

### 7.3 跨陣營：奧蘭家庭真相

```
奧蘭被賣 ← 父親決定（為了妹妹的藥）
   │
   ▼
主人家 6 年 ← 偷 Marcus 錢 ← 妹妹病加重
   │
   ▼
Marcus 暴怒要殺 ← Livia 求情 ← 阿圖斯判「下訓練場」
   │                                   │
   ▼                                   ▼
降級訓練奴 ← 玩家 Day 0 同期 ← Day 60+ 玩家漸漸拼出真相
```

**Livia 暗助網絡**：透過梅拉「多塞一塊麵包」、寄藥給奧蘭妹妹、傳家書 — 阿圖斯**知情但默許**。

### 7.4 三代家族陰謀（30 年史）

```
30 年前政變 → 提圖斯爺爺輩 / 父輩搞掉德拉格家 → 政變派接領主
                    │
                    ▼
        阿圖斯 ⚠️ 立場待定（B 共謀 / C 收容倖存）
        蓋烏斯 ⚠️（同上）
                    │
                    ▼
2-3 個月前偽旗滅村 → 提圖斯這代執行 → 升領主
                    │
                    ▼
玩家家鄉被滅 ← 凱德家鄉被滅 ← 鳶尾紋飾流落
                    │
                    ▼
            玩家 Day 0 進訓練場
```

⚠️ **這條線一旦完整，玩家百日的每件事都被「**家族世代謀國**」串起來** — 提圖斯家族兩代用屍體鋪路、阿圖斯家族站在哪邊 = 玩家最後復仇的層數。
