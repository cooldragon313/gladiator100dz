## § 1 核心哲學

**赫克托是個狡猾生物**：
- 他不是「純惡霸」也不是「暗黑朋友」——他是**評估你能不能利用**的人
- 新人出現 → 他先試探看能不能拉攏
- 成功拉攏 → 你是他生意圈的一員（他示好、但底下計算）
- 拉攏失敗 → 你是他要打壓的新人（騷擾、嫁禍）

**玩家主動選路**：Day 1 遇到他時，**玩家選要不要接受拉攏**。這讓玩家對後續關係**有知情權**。

## § 4 笑臉路線

### 4.1 示好事件池（Day 2-12 隨機）

**觸發率**：赫克托在場 × 25%/天（最多 1/天）
**效果**：mood +5 ~ +8、赫克托 +5（+6 for 帶遊戲教學功能的）

**池子（帶遊戲教學暗示）**：

| # | 對白 | 教學內容 |
|---|---|---|
| 1 | 赫克特湊過來小聲：「長官在的時候別偷懶——他是會抽人的。」 | 偷懶懲罰系統 |
| 2 | 赫克特經過：「場上那個姓卡的（卡西烏斯），別惹他。老兵。」 | 隊友關係暗示 |
| 3 | 赫克特：「廚娘給的飯多留兩口，晚上會餓。」 | 食物管理 |
| 4 | 赫克特：「武器壞了找葛拉，別自己修——手藝不到位反而更糟。」 | 鐵匠系統 |
| 5 | 赫克特：「傷成那樣還練？老默那邊有藥。付錢就是了。」 | 治療系統 |
| 6 | 赫克特：「主人最近盯著誰，你沒發現？多抬頭看看。」 | 社交觀察 |
| 7 | 赫克特：「競技場前一晚睡飽。上次有人沒睡好，第一回合就躺下了。」 | 休息重要 |
| 8 | 赫克特（壓低聲音）：「隔壁莫拉斯訓練所最近要來人。提前準備。」 | 莫拉斯對打預告 |

**其中 1 種（10% 機率）特別**：
- 赫克特直接摸進你袋子塞了顆糖：「謝了，下次記得我。」（money +3、赫克托 +8）

### 4.2 好感累積 → 暗示

**觸發條件**：赫克托好感 ≥ 15（從 baseline -10 起算 = 淨 +25）
**時機**：下次赫克托在場

```
赫克特：嘿，新人——不，該改個稱呼了。
赫克特：看你也差不多習慣了。
赫克特：記得——真的有困難可以來找我。
赫克特：我消息挺靈通的。
（他拍拍你肩膀走了。）
```

Flag：`hector_offered_intel`

### 4.3 賣情報事件（莫拉斯對打前觸發）

**觸發條件**：
- 有 `hector_offered_intel` flag
- 派遣系統決定下一場是**莫拉斯訓練所**對手
- 前一天赫克托主動來

```
赫克特：下一場是莫拉斯那邊的人吧？
赫克特：我昨天聽到消息。
赫克特：對手的弱點，我知道。
赫克特：二十個銅幣，換一條命——划算。
```

**玩家選擇**：

| 選項 | 效果 |
|---|---|
| 買（20 金）| 依好感決定真假（見下）|
| 不買 | resultDialogue: 赫克特：「行，你自己看著辦。」 |
| 揍他 | （不給這選項——笑臉路線不出現這種選項）|

**真假判定**：
- 赫克托好感 ≥ 30 → **真情報 90% 機率**
- 赫克托好感 15-29 → 真情報 50% 機率
- 赫克托好感 < 15 → 真情報 20%（基本是假的）

**真情報對白**：
```
赫克特（湊近）：對手是熊羆赫魯斯。
赫克特：他的弱點——怕穿甲。
赫克特：右手握柄特別鬆，試試第三回合抓他換招的空檔。
（他收了錢走了。）
（心想：這情報貌似有用，看來下一場妥了。）
```
Flag：`hector_tip_real_<opponentId>`

**假情報對白（好感低時）**：
```
赫克特（眼神有點閃）：對手⋯⋯他怕重擊。
赫克特：揮斧、揮錘，直接砸。
（他收了錢走了。）
（你總覺得哪裡不對勁。）
```
Flag：`hector_tip_fake_<opponentId>`（不影響戰鬥，但後續會有「你被坑」log）

### 4.4 戰鬥中生效

`battle.js` 戰鬥開始時讀 flag：

**真情報**：
- 開場顯示：「你知道 [對手] 的弱點：[weakness]」
- 對應屬性加乘（+25%）：
  - PEN 弱點 → 玩家 PEN × 1.25
  - CRT 弱點 → 玩家 CRT +15
  - ACC 弱點 → 玩家 ACC +10
  - SPD 弱點 → 玩家 SPD +8
- 清除 flag（一次性）

**假情報**：
- 無加乘
- 戰鬥結束後如果失敗 → log「⋯⋯那情報根本是假的。赫克托賺了你的錢還賣給對手。」
- 赫克托好感 -10

### 4.5 交易 2 次 + 好感 80 → 黑市暗示

**觸發條件**：
- `hector_tip_*` flag 數量 ≥ 2
- 赫克托好感 ≥ 80
- 玩家有永久傷或疤痕（可選，進階解鎖）

```
赫克特（小聲）：聽說你那個手（或腿/胸/頭，依永久傷位置）沒好過？
赫克特：我認識人。
赫克特：不是什麼好醫生，是——能讓你重新跑的那種。
赫克特：但要錢。不少錢。
（他給你一個地址。）
```

**效果**：
- 設 flag `hector_black_market_contact`
- 加個人物品「黑市暗號」
- 後續與密醫系統串接（`got_black_doc_contact` 互通）

## § 5 臭臉路線

**觸發率**：赫克托在場 × **20%/天**（比原 35% 降，符合 audit 建議 B）
**反抗選項** 40% 機率彈 ChoiceModal（新增）

**池子 5 種 + 新增 1 種**（原 5 種數值打折，見 audit B 提案）：

| # | 文本 | 原效果 → 新效果 |
|---|---|---|
| 1 | 麵包不見了 | food -10, mood -5 → food -5, mood -3 |
| 2 | 踢翻沙袋 | mood -5 → mood -3 |
| 3 | 背後撞你 | mood -8 → mood -5 |
| 4 | 「不夠格」嘲諷 | mood -5 → mood -3 |
| 5 | 水壺加沙 | mood -8, food -5 → mood -5, food -3 |
| 6（🆕）| 故意踩你腳 | mood -3 + 40% 觸發反抗選項 |

**反抗選項彈窗**（40% 機率，新增）：
```
[短對白：赫克托剛踩了你的腳]
（他還在笑。整個訓練場都在看。）

▸ 回瞪他 → mood +3、hector -3（小反抗，道德 pride positive）
▸ 忍下來 → 無額外變動（道德 patience positive）
▸ 把他推開 → hector -8, stamina -5（升級衝突）
```

### 5.2 仇恨撞擊（新增）

**觸發條件**：
- `hector_hostile_path` flag
- 赫克托好感 ≤ -25（累積仇恨）
- 赫克托在場
- 隨機 8%/天

**效果**：
```
（你正在訓練——）
（一個肩膀重重撞上你！）
（震動 + 紅光 + sword_swing 音效 + HP -5）
赫克特：喔，沒看到你。
（他沒停腳，繼續走。）
```

**30% 機率附加**：
- 輕傷（對應部位隨機 arms/torso/head）
- severity 1
- 顯示「你的手臂有點撞瘀了」

（這是你說的「仇恨度到了偶爾撞一下，隨機給個小輕傷」）

### 5.3 玩家介入他欺負別人（新增）

**觸發條件**：
- `hector_hostile_path`
- 同場有其他 NPC（優先選曾被欺負的，例奧蘭、達吉）
- 隨機 10%/天

**事件**：
```
（赫克托堵住了達吉——又一次。）
達吉（壓低聲）：不是⋯⋯我⋯⋯
赫克特：閉嘴。錢拿出來。
（達吉看見你。他眼裡有一絲希望。）
（旁邊沒別人。）
```

**玩家選擇**：

| 選項 | 效果 |
|---|---|
| 介入維護達吉 | 達吉 +15、赫克托 -8、mood +5、moral:mercy:positive |
| 威脅赫克托「換個目標」 | 赫克托 -5、玩家被登記為下次目標 |
| 袖手旁觀 | 達吉 -10（他記住你沒幫）、moral:reliability:negative |
| 跟著起鬨 | 達吉 -20、赫克托 +8、moral:cruel:positive |

### 5.4 嫁禍事件（改版）

**觸發條件**：
- `hector_hostile_path`
- 赫克托好感 ≤ -20
- Day ≥ 20
- （**移除** `day25_done` 依賴）

**事件**（原對白保留，但選項改）：
```
回到牢房。
侍從攔住你。
侍從：你的床上發現了這個。
（不是你的錢袋。）
侍從：這是誰的？
你看見赫克托站在走廊盡頭。他在笑。
```

**新選項**：

| 選項 | 效果 |
|---|---|
| 吞下（原選項保留）| masterArtus -8、mood -15、moral:mercy:positive（保護他） |
| **痛扁他（私戰）**🆕 | 進入 mini-battle（見下）|
| 指認（仍保留）| hector -20、moral:reliability:positive |

**痛扁他 · 私戰 mini-battle**：

設定：
- 非正式戰鬥（非 arena）
- 回合限制 **8 回合**（超過 = 主人/長官發現）
- 敵人：hector（STR 14, DEX 18, CON 14, AGI 18, 武器 dagger, 護甲 leather）
- **不計入競技場名聲**

結果：
- **8 回合內贏**：
  - flag `beat_hector_duel`
  - 赫克托好感反向 +30（從 -20 → +10！）
  - 赫克托不再騷擾（停止騷擾事件池觸發）
  - resultDialogue: 赫克特（站起來擦血）：「⋯⋯哈。有你的。」
- **超過 8 回合**（主人發現）：
  - 主人好感 -15
  - HP 懲罰（鞭刑 -20 HP）
  - flag `private_fight_caught`
  - 雙方都被罰，騷擾暫停 3 天但繼續
- **輸**：
  - 依對手 DEX 高 → 隨機中傷 arms 或 legs（靈巧傷）
  - 錢 -50%
  - 赫克托好感 -5 到 -25（更恨你）
  - 但未來如果玩家翻身（好感反升）還是可以走黑市路線

### 5.5 臭臉路線也可以翻轉

如果玩家「痛扁他贏了」或「連續介入保護他人、某次反而救了赫克托」→ 赫克托好感可能翻正 → **切換到笑臉路線後續**（情報 → 黑市）

這給臭臉玩家一個「打到他服」的翻盤路徑。

## § 6 共通：Day 40 倒地求救（改版）

### 保留原事件 + 加其他 NPC 反應

原事件保留，三選項保留，但**所有選項都加 NPC 連動反應**：

**幫助（過去扶他 / 叫醫生）**：
- hector +25（或 +10）
- 🆕 **其他被他欺負過的 NPC 好感降低**（過去介入過保護達吉的話，達吉 -5；奧蘭 -3 並 搖頭走掉）

**繼續訓練**：
- hector -20（原）
- 🆕 有介入過保護他人的玩家：達吉 +3、奧蘭 +3（「他知道你站對邊」）

### 笑臉 vs 臭臉路線的差異

- **笑臉路線**（好感高）：扶他是自然的。對話溫情。
- **臭臉路線**（好感低 + 曾被他撞）：扶他帶戲劇張力。赫克托對白改：
  ```
  赫克特：⋯⋯你怎麼來了。
  赫克特：（他沉默很久。）
  赫克特：欠你一次。
  ```

## § 7 敵人弱點系統（戰鬥端連動）

### 資料端（testbattle.js TB_ENEMIES 或 content/enemy.js）

每個敵人加 `weakness` 欄位：

```js
morras_ironarm: {
  name: '鐵臂',
  // ... existing
  weakness: 'PEN',   // 怕穿透（板甲但縫隙鬆）
},
morras_nightingale: {
  weakness: 'CRT',   // 怕爆擊（速度型但心理脆）
},
morras_limpwise: {
  weakness: 'ACC',   // 瞎眼 → 怕命中強的打擊
},
```

**預設 weakness map**（可先批量設）：
- 板甲/重甲系 → PEN
- 敏捷/雙持系 → CRT
- 老練/有盾系 → ACC（穿他的防守）
- 靈巧/刺客系 → SPD（硬對硬慢）

### 戰鬥端（battle.js）

開戰前檢查：
```js
const opponentId = _enemy.id;
if (Flags.has(`hector_tip_real_${opponentId}`)) {
  const weakness = TB_ENEMIES[opponentId]?.weakness;
  // 顯示開場訊息
  addLog(`✦ 你知道 [${_enemy.name}] 的弱點：${_weaknessLabel(weakness)}`, '#d4af37', true, true);
  // 套加乘
  _applyHectorWeaknessBuff(weakness);
  Flags.unset(`hector_tip_real_${opponentId}`);   // 一次性
}
```

加乘效果（建議）：
- PEN：玩家 PEN × 1.25（例 20 → 25）
- CRT：玩家 CRT +15（例 20% → 35%）
- ACC：玩家 ACC +10（例 85 → 95）
- SPD：玩家 SPD +8（例 30 → 38）
