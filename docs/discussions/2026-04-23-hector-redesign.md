## § 1 核心哲學

**赫克托是個狡猾生物**：
- 他不是「純惡霸」也不是「暗黑朋友」——他是**評估你能不能利用**的人
- 新人出現 → 他先試探看能不能拉攏
- 成功拉攏 → 你是他生意圈的一員（他示好、但底下計算）
- 拉攏失敗 → 你是他要打壓的新人（騷擾、嫁禍）

**玩家主動選路**：Day 1 遇到他時，**玩家選要不要接受拉攏**。這讓玩家對後續關係**有知情權**。

## § 4 笑臉路線

### 4.1 示好事件池（Day 2-12 隨機）

**觸發率**：赫克托在場 × 25%/天（最多 1/天）
**效果**：mood +5 ~ +8、赫克托 +5（+6 for 帶遊戲教學功能的）

**池子（帶遊戲教學暗示）**：

| # | 對白 | 教學內容 |
|---|---|---|
| 1 | 赫克特湊過來小聲：「長官在的時候別偷懶——他是會抽人的。」 | 偷懶懲罰系統 |
| 2 | 赫克特經過：「場上那個姓卡的（卡西烏斯），別惹他。老兵。」 | 隊友關係暗示 |
| 3 | 赫克特：「累了就去找廚娘。她會唸你，但偶爾會多給你一口飯。」 | 梅拉廚房互動 |
| 4 | 赫克特：「武器壞了找葛拉，別自己修——手藝不到位反而更糟。」 | 鐵匠系統 |
| 5 | 赫克特：「傷成那樣還練？老默那邊有藥。付錢就是了。」 | 治療系統 |
| 6 | 赫克特：「主人看誰多兩眼——那個人下一場準要被派出場。」 | 競技場派遣暗示 |
| 7 | 赫克特：「競技場前一晚睡飽。上次有人沒睡好，第一回合就躺下了。」 | 休息重要 |
| 8 | 赫克特（壓低聲音）：「隔壁莫拉斯訓練所最近要來人。提前準備。」 | 莫拉斯對打預告 |

**其中 1 種（10% 機率）特別**：
- 赫克特直接摸進你袋子塞了顆糖：「謝了，下次記得我。」（money +3、赫克托 +8）

### 4.2 好感累積 → 暗示

**觸發條件**：赫克托好感 ≥ 15（從 baseline -10 起算 = 淨 +25）
**時機**：下次赫克托在場

```
赫克特：嘿，新人——不，該改個稱呼了。
赫克特：看你也差不多習慣了。
赫克特：記得——真的有困難可以來找我。
赫克特：我消息挺靈通的。

```

Flag：`hector_offered_intel`

### 4.3 賣情報事件（莫拉斯對打前觸發）

**觸發條件**：
- 有 `hector_offered_intel` flag
- 派遣系統決定下一場是**莫拉斯訓練所**對手
- 前一天赫克托主動來

```
赫克特：下一場是莫拉斯那邊的人吧？
赫克特：我昨天聽到消息。
赫克特：對手的弱點，我知道。
赫克特：二十個銅幣，換一條命——划算。
```

**玩家選擇**：

| 選項 | 效果 |
|---|---|
| 買（20 金）| 依好感決定真假（見下）|
| 不買 | resultDialogue: 赫克特：「行，你自己看著辦。」 |
| 揍他 | （不給這選項——笑臉路線不出現這種選項）|

**真假判定**：
- 赫克托好感 ≥ 30 → **真情報 90% 機率**
- 赫克托好感 15-29 → 真情報 50% 機率
- 赫克托好感 < 15 → 真情報 20%（基本是假的）

**真情報對白**：
```
赫克特（湊近）：對手是熊羆赫魯斯。
赫克特：他的弱點——怕穿甲。
赫克特：右手握柄特別鬆，試試第三回合抓他換招的空檔。
（他收了錢走了。）
（心想：這情報貌似有用，看來下一場妥了。）
```
Flag：`hector_tip_real_<opponentId>`

**假情報對白（好感低時）**：
```
赫克特（眼神有點閃）：對手⋯⋯他怕重擊。
赫克特：揮斧、揮錘，直接砸。
（他收了錢走了。）
（你總覺得哪裡不對勁。）
```
Flag：`hector_tip_fake_<opponentId>`（不影響戰鬥，但後續會有「你被坑」log）

### 4.4 戰鬥中生效

`battle.js` 戰鬥開始時讀 flag：

**真情報**：
- 開場顯示：「你知道 [對手] 的弱點：[weakness]」
- 對應屬性加乘（+25%）：
  - PEN 弱點 → 玩家 PEN × 1.25
  - CRT 弱點 → 玩家 CRT +15
  - ACC 弱點 → 玩家 ACC +10
  - SPD 弱點 → 玩家 SPD +8
- 清除 flag（一次性）

**假情報**：
- 無加乘
- 戰鬥結束後如果失敗 → log「⋯⋯那情報根本是假的。赫克托賺了你的錢還賣給對手。」
- 赫克托好感 -10

### 4.5 黑市系統（🆕 三功能擴充）

#### 4.5.1 黑市入口暗示（觸發）

**觸發條件**（三者滿足任一 + 好感）：
- 赫克托好感 ≥ 80 **AND** `hector_tip_*` flag 數量 ≥ 2
- **OR** 赫克托好感 ≥ 60 且玩家有永久傷（斷肢/殘廢）
- **OR** 赫克托好感 ≥ 70 且玩家曾與密醫接觸（`got_black_doc_contact`）

**對白**（依觸發路徑差異化）：

**路徑 A（好感 + 交易 2 次）**：

赫克特：⋯⋯有些事不在主人面前講。
赫克特：我認識人。他們能賣你三樣東西。
赫克特：一，壞掉的能接回來。
赫克特：二，上場前的那一口「力氣」。

**路徑 B（永久傷需求）**：

- 加個人物品「黑市銅片」（storyTag: `black_market_token`）
- 與密醫系統互通（`got_black_doc_contact` 為平行路徑）

#### 4.5.4 功能 C：🆕 欲練神功必先自宮（換屬性）

**可用條件**：玩家主動請求 + `hector_black_market_contact` + 金錢 50 金

**哲學**：你可以花永久的代價換一項專精。不可逆。

**換法**（自選一項損失 → 自選一項收益）：

| 損失選項 | 獲得選項 |
|---|---|
| **斷左手小指** — DEX -2 永久 | STR +4 永久 |
| **剃掉一顆大牙** — CHA -2 永久（若有 CHA）或 food 每日 -2 | CON +3 永久 |
| **燒一邊眼角**（半失明）— ACC -10 永久 | CRT +10 永久（痛感讓你更狠）|
| **斷去後腳跟筋**（跛）— SPD -5 永久 | STR +6 永久（站得穩）|
| **割舌**（無法說話）— 無法再觸發對話事件 | WIL +5 永久（內心變強）|

**確認流程**：
1. 黑市選單列出 5 種換法
2. 玩家選一項 → 彈 ConfirmModal「此選擇不可逆，確認？」
3. 確認後：
   - 播 DialogueModal 動刀場景（shake-big + red-flash）
   - 套用屬性變動
   - 加對應特性（例：`one_eyed` / `crippled` / `mute` / `toothless`）
   - 設 flag `swapped_body_<type>`
4. 永久儲存，存檔生效

**對白示意（眼換暴擊）**：
```
密醫：你確定？
主角：⋯⋯做吧。
密醫：咬住這個。
（shake-big + red-flash）
（⋯⋯）
主角心想：（看不見了。右邊只剩模糊。）
主角心想：（但——）
主角心想：（我感覺得到他們的破綻。比以前更清楚。）
```

**平衡原則**：
- 不是無腦 buff — 損失永遠看得到（角色頁「疤痕」區塊列出）
- 特性會改變其他 NPC 反應（老默會擔心、奧蘭會難過、主人會認為你不正常）
- 只能換 **一次** — flag `body_swap_done` 鎖住，避免刷爆屬性
- 與道德軸：每次換 → moral:pride positive（你選擇了極端）

## § 7 敵人弱點系統（戰鬥端連動）

### 資料端（testbattle.js TB_ENEMIES 或 content/enemy.js）

每個敵人加 `weakness` 欄位：

```js
morras_ironarm: {
  name: '鐵臂',
  // ... existing
  weakness: 'PEN',   // 怕穿透（板甲但縫隙鬆）
},
morras_nightingale: {
  weakness: 'CRT',   // 怕爆擊（速度型但心理脆）
},
morras_limpwise: {
  weakness: 'ACC',   // 瞎眼 → 怕命中強的打擊
},
```

**預設 weakness map**（可先批量設）：
- 板甲/重甲系 → PEN
- 敏捷/雙持系 → CRT
- 老練/有盾系 → ACC（穿他的防守）
- 靈巧/刺客系 → SPD（硬對硬慢）

### 戰鬥端（battle.js）

開戰前檢查：
```js
const opponentId = _enemy.id;
if (Flags.has(`hector_tip_real_${opponentId}`)) {
  const weakness = TB_ENEMIES[opponentId]?.weakness;
  // 顯示開場訊息
  addLog(`✦ 你知道 [${_enemy.name}] 的弱點：${_weaknessLabel(weakness)}`, '#d4af37', true, true);
  // 套加乘
  _applyHectorWeaknessBuff(weakness);
  Flags.unset(`hector_tip_real_${opponentId}`);   // 一次性
}
```

加乘效果（建議）：
- PEN：玩家 PEN × 1.25（例 20 → 25）
- CRT：玩家 CRT +15（例 20% → 35%）
- ACC：玩家 ACC +10（例 85 → 95）
- SPD：玩家 SPD +8（例 30 → 38）
