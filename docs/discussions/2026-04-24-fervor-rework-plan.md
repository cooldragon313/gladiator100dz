## § 0 鐵則（先定）

1. **Stage / 對白裡不出現數字**。數字只在下方 log 文字行
2. **獲得任何人事物 / 結束場面** → 震動 + 螢幕大字 POPUP（要看得到）
3. **觸發要有明確講法**，但**不能寫看不懂的具象**（「手心癢」「腳底發癢」太抽象）
4. **訓練名稱統一**：力量 / 靈巧 / 體質 / 反應 / 意志（不要混 鐵耐 / 步法 / 敏）
5. **對白要對應該動作的具體動作**，不能憑空編

## § 2 觸發 / 結束的講法（有講法、但不抽象）

### 鐵則：講「動作的當下」，不講玄學

❌ 不要：「手心癢」「腳底發癢」「身體在說什麼」「你抓到什麼了」
✅ 要：「你伸手去端水桶時——身體記得剛剛舉石頭的弧度」「你坐下來，腰桿自己挺直了」

### 自然觸發講法（5 attr，每套 3 句）

#### 力量狂熱（連續推舉石頭 8 次）
```
（這幾天推石頭，你出力不一樣了。）
（剛剛端碗的時候——指頭夾得太用力，碗裂了。）
⚡ 你進入了【力量狂熱】。
```

#### 靈巧狂熱（連續投接碎石 8 次）
```
（這幾天的投接，你的手指變敏了。）
（剛剛抓飛過來的蒼蠅——抓到了。你自己嚇一跳。）
⚡ 你進入了【靈巧狂熱】。
```

#### 體質狂熱（連續杖擊承受 8 次）
```
（這幾天被棍子打——你的身體變了。）
（剛剛被人撞到肩膀——對方退了兩步，你沒動。）
⚡ 你進入了【體質狂熱】。
```

#### 反應狂熱（連續亂棍格擋 8 次）
```
（這幾天的格擋，你的眼睛變了。）
（剛剛瓦罐從架上滑落——你接住了。你不是看的，是動作先到。）
⚡ 你進入了【反應狂熱】。
```

#### 意志狂熱（連續打坐冥想 8 次）
```
（這幾天的打坐，你的腦子變了。）
（剛剛長官對面那個人在罵你——你聽得很清楚，但他的聲音很遠。）
⚡ 你進入了【意志狂熱】。
```

### 瓶頸觸發講法（每套 4 句）

**結構**：「想升級 → 動作測試 → 還不夠 → 觸發狂熱」

#### 力量瓶頸
```
（你想——這幾天累積夠了吧。）
（你蹲下試了一塊石頭。）
（舉得起來——但放下的時候，肩膀晃了一下。）
（還沒。你需要再推幾次，讓這份力氣真的住進骨頭。）
⚡ 【力量狂熱】啟動。
```

#### 靈巧瓶頸
```
（你想——這幾天的手感應該夠了。）
（你撿起一顆碎石拋了一次。）
（接到了——但接的時候手指僵了一下。）
（還沒。你需要再投幾次，讓那份精準刻進指尖。）
⚡ 【靈巧狂熱】啟動。
```

#### 體質瓶頸
```
（你想——這幾天的身板應該成形了。）
（你讓人輕輕用棍子敲了肩膀一下。）
（沒倒——但腳跟離地了半寸。）
（還沒。你需要再被打幾次，讓這身肉真的長到骨頭上。）
⚡ 【體質狂熱】啟動。
```

#### 反應瓶頸
```
（你想——這幾天的眼睛應該夠快了。）
（你閉眼，讓人朝你揮一棍。）
（你閃開了——但比對方的棍尾慢了一拍。）
（還沒。你需要再格幾次，讓那份反射真的成為本能。）
⚡ 【反應狂熱】啟動。
```

#### 意志瓶頸
```
（你想——這幾天的心應該定了。）
（你坐下來，閉眼。）
（坐住了——但腦子裡有一段話一直在跑。）
（還沒。你需要再坐幾次，讓那份靜真的沉到底。）
⚡ 【意志狂熱】啟動。
```

## § 6 大字 POPUP 系統（共用元件）

**用在哪**：
- 狂熱觸發（自然 + 瓶頸）
- 狂熱結束（自然 + 瓶頸）
- 屬性升級（含瓶頸後）
- 未來：獲得物品 / 重要旗標 / 結局轉折

**規格**：
- 螢幕中央，大字（48px+）金色發光
- 上半行 emoji + 標題（「⚡ 力量狂熱」）
- 下半行小字描述（可選，「你進入了狂熱狀態」）
- 持續 1.5-2 秒、淡入淡出
- 配音效（acquire / level_up）+ 震動

**API**：`Game.popupBig({ icon, title, subtitle, color, duration, sound, shake })`

實作位置：[src/main.js](src/main.js) 或 [src/ui/stage.js](src/ui/stage.js) 新加一個 `Stage.popupBig()`
