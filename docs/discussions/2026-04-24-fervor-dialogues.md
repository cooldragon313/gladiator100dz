## § 2 狂熱・自然觸發

**位置**：[src/systems/compulsion.js](src/systems/compulsion.js) `_playTriggerScene`（source === 'natural'）

### 【現】（4 attr 共用）

```
（你感覺到身體正在說什麼。）
（這幾天的訓練堆疊起來——你抓到什麼了。）
⚡ 【XX狂熱】進入狀態。
```

抽象、不具體。「身體正在說什麼」太模糊，「你抓到什麼了」更虛。

### 【建議】依 attr 寫四套，給具體感官

#### STR（力量狂熱）
```
（你伸手去拿水壺——指節關節響了一下。）
（不是痛。是飽。）
（這幾天的揮砍，把你身體裡某個東西塞滿了。）
（你想再揮一次。再一次。再一次。）
⚡ 【力量狂熱】 — 練力量 EXP +25%。其它的——你會分心。
```

#### AGI（步法狂熱）
```
（你走路的時候腳掌會自動找好踏點。）
（不是想——是知道。）
（這幾天的步法，把沙子的紋理刻進你的腳底了。）
（你想跑。沿著訓練場的邊緣，繞一圈，再一圈。）
⚡ 【步法狂熱】 — 練步法 EXP +25%。其它的——你的腳會跑神。
```

#### CON（鐵耐狂熱）
```
（你坐下休息的時候——身體在抗議。）
（它不是累。它是還沒練夠。）
（這幾天的耐力，把你的肺撐開了一個你不認識的尺寸。）
（你想再扛一次。再跑一圈。再撐一下。）
⚡ 【鐵耐狂熱】 — 練耐力 EXP +25%。其它的——你會覺得在偷懶。
```

#### WIL（禪定覺醒）
```
（你閉眼的時候——一切聲音突然有了距離。）
（訓練場的吼聲、鞭子的破空、別人的腳步——都變遠了。）
（這幾天的打坐，把你跟世界之間挖出了一道縫。）
（你想再坐下去。多一點。再多一點。）
⚡ 【禪定覺醒】 — 練冥想 EXP +25%。其它的——你會分神。
```

## § 4 狂熱・結束

**位置**：[src/systems/compulsion.js](src/systems/compulsion.js) `_complete`

### 【現】

```
（自然）⚡ 【力量狂熱】結束。那種癮頭散了，你感覺到——你真的變強了。
（瓶頸）⚡ 【力量狂熱】結束——你可以繼續突破到 20 了。

### 【建議】有節奏的小結尾，把「你成為了什麼」說出來

```

```

（沒喘。）

### 【現】

```
STR:
- （你抬手的時候手心癢——想再握一次那塊石頭。）
- （你發現自己又在無意識握拳。）

AGI:
- （你跑到一半停下來——腳底發癢，像在催你走另一條路。）
- （你意識到自己在原地踏步。）

CON:
- （肩膀少了那份重量。你摸了下，下意識的。）

## § 6 狂熱期間・練對屬性的進度回饋

**位置**：[src/systems/compulsion.js](src/systems/compulsion.js) `onTraining`（attr === active 分支）

### 【現】

```
⚡ 【力量狂熱】 1 / 5
⚡ 【力量狂熱】 2 / 5
⚡ 【力量狂熱】 3 / 5
⚡ 【力量狂熱】 4 / 5
⚡ 【力量狂熱】 5 / 5（→ 觸發結束演出）
```

純數字 log。爽感不夠。

### 【建議】依進度給遞進的感受對白

```
1/5: ⚡ 【力量狂熱】 1/5　（手感對了。再來。）
2/5: ⚡ 【力量狂熱】 2/5　（你揮得比昨天順。身體記住了。）
3/5: ⚡ 【力量狂熱】 3/5　（一半了。你開始享受揮的當下。）
4/5: ⚡ 【力量狂熱】 4/5　（差最後一次。手指有點抖——不是累，是滿。）
5/5: → 跳結束演出（不顯示這行）
```

四個 attr 各寫一套（語氣一致、動詞換掉）：

| 進度 | STR | AGI | CON | WIL |
|---|---|---|---|---|
| 1/5 | 手感對了。再來。| 步子對了。再跑。| 喘上來了——但很穩。再來一輪。| 心靜了一點。再坐。|
| 2/5 | 你揮得比昨天順。身體記住了。| 沙子在你腳下沒亂。| 肺撐得開了。| 雜念少了一些。|
| 3/5 | 一半了。你開始享受揮的當下。| 一半了。腳開始自己跑。| 一半了。呼吸自己整齊。| 一半了。你開始聽見更遠的東西。|
| 4/5 | 差最後一次。手指有點抖——不是累，是滿。| 差最後一圈。你想加速。| 差最後一次。你想再多扛一下。| 差最後一次。你不想睜眼。|

## § 8 給使用者的問題

1. **語氣對嗎？** 上面用的「（你發現自己⋯⋯）」「（不是 X，是 Y）」這種句式 — 是你想要的調性嗎？還是要更冷／更燥／更詩？
2. **結束場面**要不要更隆重？例如自然結束時加 1 次震動 + 一行 stage 大字（像 acquire 物品那種 popup）？
3. **進度 1-4 的感受句**會不會太囉嗦？要不要砍成只在 1/5、3/5、4/5 三個點才跳？
4. **升級 log 的格式**：`✦ 力量 → 16　（揮拳的時候你感覺到了——握把比昨天輕。）`
   - 這樣一行夠嗎？還是 log 顯示要拆成兩行（「✦ 力量 → 16」+ italic 灰字一行）?
5. **練錯屬性擺爛**目前是 15% 機率彈一句。要不要連帶**讓 EXP emoji 顯示「」/ 變灰**強化負面感？

回答完我就動手改。
