## § 1 角色定位（兩人對位）

### 塔倫長官 (officer)
**冷靜、遠慮的官僚 — 看數字、看主人、看回報**

- 喜歡能**幫主人賺錢**的人 → reliable / loyal（可靠聽話）
- **派戰士上場 = 商業決策**：
  - 要贏 → 戰士要贏**而且贏得漂亮**（票房）
  - 要輸 → 戰士要懂得**用對的方式輸**（賠率/觀眾愛憎/送對手人情）
- 不在訓練場吼人 — 他在算盤上加減

### 監督官 = 巴布魯斯（暱稱「巴爺」）
**退役角鬥士 — 看肌肉、看意志、看你今天有沒有撐住**

- 一句話人設：**「不管你們是誰，在這都要想盡辦法活下去撐下去站到底。少囉嗦，別找理由。」**
- 喜歡：**勇敢 / 勤勞**
- 討厭：**神經質 / 找理由**
- 跟玩家有**完整好感互動線**（見 § 6）

## § 2 巴爺的背景

### 已定
- **本名**：巴布魯斯（Babrius）
- **暱稱**：「巴爺」— 老夥伴間用（梅拉 / 老默 / 卡西烏斯都這麼叫）
- **背景**：退役角鬥士、被塔倫的「上一代監督官」帶過
- **稱呼演化**（玩家視角）：
  - 一般時候：「監督官」
  - 偶爾聽到 NPC 講「巴爺」 → 玩家慢慢拼湊出他的本名
  - 高好感後：玩家或某些對白可叫「巴布魯斯」（極親近 + 私下）

### ❓ 待決 1：光榮退役 vs 戰敗退役？

| 方案 | 巴爺的退役方式 | 帶來的人設 / 對白 |
|---|---|---|
| **A. 光榮退役**（活下來、被買下當監督）| 場上贏到尾、主人留他訓練後輩 | 自信、有威嚴、講「我那年代」自豪、看不起怕死的鬥士 |
| **B. 戰敗退役**（殘但沒死、被留下當苦力監督） | 場上輸過一次大的、被砍斷某部位、不能再戰 | 有疤、走路微跛、講「你以為那很慘？我那年⋯⋯」、對殘廢/受傷的玩家會多看一眼 |
| **C. 折衷**：贏過巔峰、但**最後一戰受傷**退役 | 兩者混合 | 既有威嚴又有殘缺、最有戲劇張力 |

**我的偏好**：**C 折衷** — 給角色「**他贏過、但他沒贏到底**」的層次。這樣他對玩家說「撐下去站到底」才有重量（他自己最後一場沒站到底）

## § 3 跟塔倫的關係（連動 ❓ 待決 2）

### 已定
- 塔倫是**上一代監督官**（從同位置升上去、現在是長官）
- 兩人共事很久 — 但**互相看不起**
  - 巴爺看塔倫：「不上場的人懂個屁」
  - 塔倫看巴爺：「老粗。但他能讓肉動起來。」

### ❓ 待決 2：上一代監督官（已退）對年輕巴爺的態度

這個會決定巴爺的「人設陰影」+ 影響他現在對待玩家的方式：

| 方案 | 上一代監督對年輕巴爺的態度 | 影響巴爺現在的人設 |
|---|---|---|
| **看好他** | 把巴爺當接班人、教他全部、希望他活下來 | 巴爺現在會把同樣眼光放在某些玩家身上（**iron_will / diligence 玩家會特別被他關注**）。「以前老師看我就像我現在看你。」 |
| **不看好他** | 覺得巴爺是廢物、常常公開羞辱、不期待他活著 | 巴爺現在故意「不挑徒弟」— 對所有人一視同仁地嚴酷。「我那時候被當垃圾。但我活下來了。你們也可以。」 |
| **看好但塔倫嫉妒** | 上一代監督最看好巴爺（不是塔倫）、塔倫因此搶位置/壓他 | 解釋現在塔倫 vs 巴爺的關係根源 — 兩人衝突有歷史 |

**我的偏好**：**看好（方案一）** — 跟「光榮退役 + 巔峰受傷」最相容。巴爺把同樣眼光放在玩家身上 = 故事會很動人。但**方案三**也很有戲（解釋塔倫為什麼是長官、巴爺還在當監督）。

## § 4 likedTraits / dislikedTraits 重排

### 塔倫長官（依使用者 spec）
**核心邏輯：能幫主人賺錢的人 = 可靠 + 聽話 + 會贏會輸**

```js
likedTraits:    { reliable:3, loyal:3, patient:2, iron_will:1 }
dislikedTraits: { impulsive:3, opportunist:3, prideful:2, coward:1 }
```

- `reliable` 升 +3（從 +3 維持）：說到做到 = 派得出去
- `loyal` 升 +3（從 +2）：聽話 = 主人的工具
- `patient` 維持 +2：不衝動 = 場上能控節奏
- `prideful` 變 -2（從 -1 加重）：場上要會輸的人不能驕傲
- `coward` 降 -1（從 -2 減輕）：怕死的人有時候反而會聽話

### 監督官巴爺（依使用者 spec）
**核心邏輯：撐下去站到底 = 勇敢 + 勤勞 / 討厭找理由**

```js
likedTraits:    { iron_will:3, diligence:3, prideful:2, cruel:1 }
dislikedTraits: { neurotic:3, coward:3, opportunist:2, merciful:1 }
```

- `iron_will` +3：撐到底的能力
- `diligence` +3：勤勞 = 不偷懶 = 不找理由
- `prideful` +2：勇敢敢撐才能站到底（跟塔倫相反）
- `cruel` +1：場上能下手 = 能活
- `neurotic` -3：「想太多」最怕（**新增**，配合「不要找理由」）
- `coward` -3：怕死撐不過第一場
- `merciful` -1：場上心軟會死

**塔倫 vs 巴爺最大反差**：
- `prideful`：塔倫 -2 / 巴爺 +2 — 「驕傲是奴隸的奢侈」 vs 「沒驕傲撐不下去」
- `loyal`：塔倫 +3 / 巴爺 0 — 「聽話最重要」 vs 「我管你聽不聽話、能撐就好」

## § 5 抓偷懶分工（依使用者 spec）

### 出現機率階梯（從低到高）
| 抓人者 | 偷懶被抓機率 | 場景 |
|---|---|---|
| **主人 (masterArtus)** | 很低（~5%） | 主人很少巡訓練場 |
| **侍從 (masterServant)** | 低（~10%） | 主人代行 |
| **塔倫長官 (officer)** | 中（~20%） | 平時不巡、出現 = 有事 |
| **監督官（巴爺）** | **75%+** | 他工作就是盯 |

### 沒人抓的情境
- 25% 的時間「**沒人出現**」 = 玩家好好休息的時段（恢復一些 stamina + mood）

### 抓包後的處罰邏輯（依抓人者 + 巴爺好感）

| 抓人者 | 處罰 |
|---|---|
| **主人（masterArtus）出現** | **大懲罰** — HP -10、好感 -15、震動+紅光 + 對白：「主人面前偷懶 = 大忌」 |
| **塔倫長官** | 中懲罰 — HP -5、好感 -10、冷酷對白「沒救的東西」 |
| **巴爺**：好感 < 30 | 直接抽打 — HP -5、好感 -3、抽鞭演出 |
| **巴爺**：好感 ≥ 30 | **只大吼** — HP 0、好感 -1、罵兩句完事「你他媽再讓我看到一次⋯⋯」 |
| **巴爺**：好感 ≥ 60 | 假裝沒看到 — 好感 -0、對白：「⋯⋯我這次當沒看到。下次別再讓我看見。」 |
| **侍從** | 輕懲 — HP -2、好感 -5（侍從會去報告主人）、後續主人可能找上門 |
| **無人** | 平安偷懶 — stamina +5 / mood +3 |

### 設計意圖（玩家學到什麼）
- **跟巴爺打好關係 → 偷懶被抓不會死**（是這個訓練所唯一的緩衝）
- **主人在 = 千萬別偷懶**（最大恐懼）
- 塔倫長官出現 = 不是平時巡邏，是有事

## § 6 巴爺跟玩家的好感互動線（完整故事）

> 使用者要求「**完整的人生跟故事**」+「跟玩家有好感上的互動」 — 比奧蘭 / 赫克托稍輕一點，但比卡西烏斯更厚

### storyReveals（5 段，aff 門檻遞增）
| ID | aff | 揭露 |
|---|---|---|
| `overseer_silent` | 0 | flavor — 他在訓練場不講廢話。罵的時候不重複 |
| `overseer_old_scar` | 15 | flavor — 你發現他左前臂有一道又長又老的疤痕 |
| `overseer_friends_call` | 30 | event — 你聽到梅拉叫他「巴爺」的時候，他笑了一下。第一次看到他笑 |
| `overseer_last_match` | 50 | event — 他喝多的某個夜晚對你講起最後一場 — 但講到一半就停了 |
| `overseer_master_pupil` | 75 | event — 他承認上一代監督官（現在的塔倫長官）是看好他的。「他以前對我也說過——撐到底。」 |

### 互動事件（aff 門檻 + 條件觸發，~3-4 個）

#### 事件 1 — 第一次罵你罵到收手（aff ~25 + 你練得很勤）
```
（巴爺站在你旁邊看了很久。）
巴爺：你今天揮的這幾下——
（他停下。）
巴爺：⋯⋯算了。你自己知道。
（他走開。沒有罵。）
```
玩家心想：（這是肯定嗎？）

#### 事件 2 — 「巴爺」這個外號的由來（aff ~40 + 場上有梅拉）
- 梅拉端飯來訓練場，叫了一聲「巴爺，吃飯了」
- 巴爺沒回頭，但揮了揮手
- 玩家對白選項：
  - 問梅拉「為什麼叫他巴爺？」 → 梅拉笑著講三言兩語的過去
  - 不問 → 心想（⋯⋯這外號是怎麼來的？）

#### 事件 3 — 跟卡西烏斯的對戲（aff ~50 + 卡西烏斯也在場）
- 兩個老角鬥士偶遇 — 沒講話、互相點頭
- 玩家旁觀 → log 揭露「兩人是同期入場的」
- 後續卡西烏斯可能講一句「他比我撐得久」

#### 事件 4 — 訣別 / 託付（aff ~70 + Day ≥ 70）
- 巴爺在某個夜晚找你
- 對白：「你⋯⋯如果我倒了，繼續站到底。聽到沒有？」
- flag set：`overseer_passed_torch`
- 後續結局判定可能加分支

## § 7 場景頻率調整

### 塔倫長官（officer）
- **退出 audience pool**（fields.js 移除或 chance 降到 0.10）
- 重大日子才必到（Day 5 沙洗 / Day 10 淘汰 / Day 25 秋祭 / Day 40 / Day 60 偷藥事件 / Day 85+）
- 平日 ~10% 機率露個臉（讓玩家偶爾被提醒「他在看資料」）

### 監督官巴爺（overseer）
- **大幅提高 audience chance** — 0.65 → **0.85**
- 訓練時段（早 + 下午）幾乎必到
- 唯一缺席的時段 = 玩家「好好休息時間」（25% 沒人在）
