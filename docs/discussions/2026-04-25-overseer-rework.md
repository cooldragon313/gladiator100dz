# 監督官 / 塔倫長官分工 — 設計 spec v2

> 2026-04-25 起。第一版討論完使用者已定方向，這版是依使用者 spec 重寫。
> 仍有 2 個待決議題（標 ❓），其餘已定，回完就動手。

## § 1 角色定位（兩人對位）

### 塔倫長官 (officer)
**冷靜、遠慮的官僚 — 看數字、看主人、看回報**

- 喜歡能**幫主人賺錢**的人 → reliable / loyal（可靠聽話）
- **派戰士上場 = 商業決策**：
  - 要贏 → 戰士要贏**而且贏得漂亮**（票房）
  - 要輸 → 戰士要懂得**用對的方式輸**（賠率/觀眾愛憎/送對手人情）
- 不在訓練場吼人 — 他在算盤上加減

### 監督官 = 巴布魯斯（暱稱「巴爺」）
**退役角鬥士 — 看肌肉、看意志、看你今天有沒有撐住**

- 一句話人設：**「不管你們是誰，在這都要想盡辦法活下去撐下去站到底。少囉嗦，別找理由。」**
- 喜歡：**勇敢 / 勤勞**
- 討厭：**神經質 / 找理由**
- 跟玩家有**完整好感互動線**（見 § 6）

## § 2 巴爺的背景（v2 對齊時間線）

### 已定
- **本名**：巴布魯斯（Babrius）
- **暱稱**：「巴爺」— 老夥伴間用（梅拉 / 老默 / 卡西烏斯都這麼叫）
- **背景**：退役角鬥士、塔倫當年的監督對象（見 § 3 時間線）
- **稱呼演化**（玩家視角）：
  - 一般時候：「監督官」
  - 偶爾聽到 NPC 講「巴爺」 → 玩家慢慢拼湊出他的本名
  - 高好感後：玩家或某些對白可叫「巴布魯斯」（極親近 + 私下）

### 退役方式（已定為 C — 對齊新時間線）

**C 巔峰受傷** — 贏過幾年巔峰，最後一戰**塔倫故意配的強敵**讓他重傷退役。

- 對齊 § 3 v4 時間線：
  - 巴爺**真的贏過**（這就是為什麼他「太強眼」讓塔倫忌憚）
  - 最後一戰是**塔倫安排的死局**（對手太強、巴爺重傷沒死）
  - 塔倫順勢「力保」演好人（救一個重傷沒威脅的人零成本）
- 角色內在矛盾：「**我贏過 — 但我沒贏到底**」+ 隱約懷疑「**那場為什麼這麼難**」
- 對玩家講「撐下去站到底」時帶有自嘲重量
- 身體狀態：左前臂大疤痕（已寫進 § 6 storyReveal）+ 可能微跛 / 某動作不順

## § 3 跟塔倫的關係（**v6：完整動機鏈 — 主人 / 塔倫 / 巴爺三角**）

### ⚠️ 修正紀錄
- v1：上一代監督看好巴爺（邏輯炸）
- v2：兩人版時間線、塔倫公開貶低（缺乏 cunning 人設）
- v3-v4：塔倫設局送死 + 假裝力保
- v5：dramatic irony 結構（巴爺感激→揭露→崩潰）
- **v6（最新）**：使用者升級 — 完整三角動機（**主人愛錢愛名聲、塔倫沒能力只能算計、巴爺成招牌威脅塔倫**）+ 多源頭暗示揭露

### 三角人物動機鏈（v6 核心）

#### 主人 (masterArtus)
- 愛**錢**、愛**名聲**、想討好**領主**
- 喜歡自己的招牌明星（賺錢）— 但**可以被「賺一大筆 + 領主歡心」說服送出去**

#### 塔倫長官 (officer)
- 從**小奴隸戰士**爬起來
- 個人能力**不夠當招牌**（這是 v6 重要修正：他不是強者）
- 知道自己沒辦法走「成招牌→舔主人」的路
- → 走「**幫主人爭取最大回報**」的路上位
- 看數字、設局、賺錢給主人 = 自己上位的唯一方法

#### 巴爺 (overseer)
- 個人能力很好、成訓練場招牌之一
- **努力認真重情義**、深得大家喜好
- → 過於搶眼、塔倫害怕成競爭者
- → **被借局斷了未來**

### 時間線（v6 完整版）

```
多年前
├─ 主人有兩個招牌：巴爺 + 其他幾個（賺錢用）
├─ 塔倫 = 監督官，個人能力一般，但會算
└─ 巴爺 = 場上招牌、深得大家喜歡、塔倫忌憚他取代自己位置

塔倫的賭局（精髓 v6）
├─ 領主生日將至，主人需要討好的禮物
├─ 領主最近訓服了老虎、想找個強對手試試
├─ 塔倫獻計：「把巴爺送上去」
│  ├─ 「會賺一大筆錢 + 取得領主歡心」
│  └─ 主人猶豫（捨不得招牌）— 但被誘惑說服
├─ 條件：「**這次成功，讓我升長官**」（塔倫先談好的籌碼）
└─ 巴爺被推上場 — 對手：領主借來的強敵 / 訓練過的老虎 / 兩者組合

巴爺重傷（沒死）
├─ 主人雖然賠了招牌、但領主開心了 + 賺了一筆
├─ 塔倫獲得允諾：升長官
└─ 但塔倫立刻動下一步...

塔倫的二手（為自己訓練場勢力鋪路）
├─ 升長官後，訓練場需要新監督官
├─ 「重情義的巴爺對救命恩人會死心踏地」
│  → 完美的「有能力 + 聽話」工具人
├─ 力保巴爺：「他是自家人」
│  → 表面演好人、實際是養一條會幫他養更多狗的狗
├─ 巴爺認塔倫為救命恩人、感激痛哭流涕
└─ 塔倫：「妙計」

現在
├─ 塔倫：表面長官、實際在訓練場仍有勢力（透過巴爺）
├─ 巴爺：完全信任塔倫、塔倫要他做什麼都認真做
│  └─ 平常會跟玩家說「跟著長官混不會吃虧」
├─ 塔倫常借巴爺手收拾事情（巴爺以為是恩義）
└─ 巴爺心裡：「我這條命是長官給的」
```

### 巴爺認知狀態（v6 確定）

**完全感激、痛哭流涕級** — 把塔倫當愛護自己人的兄弟長官、非常尊重。
塔倫平常嘴邊「為大人而活」這種話、巴爺真的當價值觀。
塔倫要他做什麼，巴爺都認真做。

### 揭露結構（v6 — 4 層遞進）

v6 把揭露拆 4 層 — 不塞給玩家、要玩家自己拼湊。

#### 層 1 — 多源頭暗示（鋪墊期，跟其他老 NPC 好感觸發）

**老 NPC 不直接說「塔倫害了巴爺」 — 世界就是這樣，講穿沒好處**。
但**跟他們好感變好後**，會給玩家「**不要太相信塔倫**」級別的暗示。

候選暗示者：

| NPC | 為什麼適合 | 暗示對白範例 |
|---|---|---|
| **老默** | 已有「火刑灼痕」未解釋傷 + 「不要記臉。只記傷」沉默觀察人設 → **最佳人選** | 治療玩家時：「⋯⋯這傷我以前看過。被那種對手打的。」（停頓）「⋯⋯算了。傷會好。」 |
| **卡西烏斯** | 老兵 / 看過很多 | 「我跟長官共事很久了。⋯⋯**該說的他從不少說、不該說的他也從不漏一句**。你懂嗎？」 |
| **梅拉** | 母親型 / 私下叫巴爺「巴爺」 | 「孩子⋯⋯有些人對你笑，**不代表你能對他笑回去**。」 |

> ⚠️ **老默的整合**：老默現有對白已寫死、火刑灼痕已 storyReveal。
> 解法：**新增「老默對玩家高好感（≥50）後」的治療雜談對白池**、塞 1-2 句暗示、不動現有對白。

#### 層 2 — 塔倫好感的副作用（玩家親身感受）

**v6 精彩處** — 塔倫好感變高後，**塔倫會給玩家曖昧的指令**，玩家親身感受到他的算計。

範例曖昧指令（待設計具體事件）：
- 「下次切磋時對 OOO **認真打**、下重手 — 那小子最近太囂張」
- 「主人問起你訓練如何，記得說一切都好」
- 「我借用你的好感賣個情報給商人 — 你不用知道細節」

→ 玩家做完後可能心想：「**這真的是好人會給的指令嗎？**」

#### 層 3 — 偷聽密謀事件（v6 核心揭露）

**觸發條件**（你提的數字）：
- 主人好感 ≥ 60
- 塔倫好感 ≥ 40
- 監督官好感 ≥ 60

**事件流程**：
1. 主人召喚玩家
2. 玩家走到主人房門口、**恰巧聽到塔倫跟主人密謀**（不是主動偷聽）
3. 對白：

```
主人：⋯⋯領主生日又到了。總要找個理由討好他。
塔倫：大人說的是。
塔倫：新來的這幾個——有一個不錯。最近名聲也達到了。
塔倫：要不要再送一場好戰給領主開心？
主人：嗯⋯⋯
塔倫：聽說領主前陣子訓服了一頭老虎——
塔倫：他一直想找個夠看的對手試試。
主人：⋯⋯哦？
塔倫：不然推出 [玩家名] 去送頭？
塔倫：像上次老巴一樣⋯⋯
主人：（大笑）妙計！
（兩人竊笑。）
（你站在門口、沒敢進去。心臟在跳。）
```

4. 玩家當下兩個衝擊：
   - 「**上次老巴一樣**」= 塔倫設計巴爺的鐵證
   - **[玩家名]** = 你是下一個被送的人

⚠️ **設計關鍵**：「[玩家名]」要動態替換玩家自己的角色名

#### 層 4 — 告訴巴爺（玩家的選擇）

**觸發**：層 3 後 + 巴爺好感 ≥ 60

玩家可選：
- **A 告訴巴爺真相** → 觸發訣別事件（崩潰版）
- **B 不告訴**（保護巴爺幻覺）→ 訣別不觸發 / 溫和版本

→ **道德重量** — 真相對巴爺是恩賜還是傷害？玩家決定。

---

### 為什麼這個結構強

1. **玩家有特權體驗** — 觀眾比角色先知道（dramatic irony）
2. **多源頭暗示**避免「一個 NPC 講出來」太直白
3. **層 2 親身感受**讓玩家自己懷疑塔倫人格、不靠別人講
4. **層 3 偷聽事件**= 鐵證 + 自己也成下一個目標的雙重衝擊
5. **層 4 玩家選擇**有道德重量（傳真相 vs 保護幻覺）

## § 4 likedTraits / dislikedTraits 重排

### 塔倫長官（依使用者 spec）
**核心邏輯：能幫主人賺錢的人 = 可靠 + 聽話 + 會贏會輸**

```js
likedTraits:    { reliable:3, loyal:3, patient:2, iron_will:1 }
dislikedTraits: { impulsive:3, opportunist:3, prideful:2, coward:1 }
```

- `reliable` 升 +3（從 +3 維持）：說到做到 = 派得出去
- `loyal` 升 +3（從 +2）：聽話 = 主人的工具
- `patient` 維持 +2：不衝動 = 場上能控節奏
- `prideful` 變 -2（從 -1 加重）：場上要會輸的人不能驕傲
- `coward` 降 -1（從 -2 減輕）：怕死的人有時候反而會聽話

### 監督官巴爺（依使用者 spec）
**核心邏輯：撐下去站到底 = 勇敢 + 勤勞 / 討厭找理由**

```js
likedTraits:    { iron_will:3, diligence:3, prideful:2, cruel:1 }
dislikedTraits: { neurotic:3, coward:3, opportunist:2, merciful:1 }
```

- `iron_will` +3：撐到底的能力
- `diligence` +3：勤勞 = 不偷懶 = 不找理由
- `prideful` +2：勇敢敢撐才能站到底（跟塔倫相反）
- `cruel` +1：場上能下手 = 能活
- `neurotic` -3：「想太多」最怕（**新增**，配合「不要找理由」）
- `coward` -3：怕死撐不過第一場
- `merciful` -1：場上心軟會死

**塔倫 vs 巴爺最大反差**：
- `prideful`：塔倫 -2 / 巴爺 +2 — 「驕傲是奴隸的奢侈」 vs 「沒驕傲撐不下去」
- `loyal`：塔倫 +3 / 巴爺 0 — 「聽話最重要」 vs 「我管你聽不聽話、能撐就好」

## § 5 抓偷懶分工（依使用者 spec）

### 出現機率階梯（從低到高）
| 抓人者 | 偷懶被抓機率 | 場景 |
|---|---|---|
| **主人 (masterArtus)** | 很低（~5%） | 主人很少巡訓練場 |
| **侍從 (masterServant)** | 低（~10%） | 主人代行 |
| **塔倫長官 (officer)** | 中（~20%） | 平時不巡、出現 = 有事 |
| **監督官（巴爺）** | **75%+** | 他工作就是盯 |

### 沒人抓的情境
- 25% 的時間「**沒人出現**」 = 玩家好好休息的時段（恢復一些 stamina + mood）

### 抓包後的處罰邏輯（依抓人者 + 巴爺好感）

| 抓人者 | 處罰 |
|---|---|
| **主人（masterArtus）出現** | **大懲罰** — HP -10、好感 -15、震動+紅光 + 對白：「主人面前偷懶 = 大忌」 |
| **塔倫長官** | 中懲罰 — HP -5、好感 -10、冷酷對白「沒救的東西」 |
| **巴爺**：好感 < 30 | 直接抽打 — HP -5、好感 -3、抽鞭演出 |
| **巴爺**：好感 ≥ 30 | **只大吼** — HP 0、好感 -1、罵兩句完事「你他媽再讓我看到一次⋯⋯」 |
| **巴爺**：好感 ≥ 60 | 假裝沒看到 — 好感 -0、對白：「⋯⋯我這次當沒看到。下次別再讓我看見。」 |
| **侍從** | 輕懲 — HP -2、好感 -5（侍從會去報告主人）、後續主人可能找上門 |
| **無人** | 平安偷懶 — stamina +5 / mood +3 |

### 設計意圖（玩家學到什麼）
- **跟巴爺打好關係 → 偷懶被抓不會死**（是這個訓練所唯一的緩衝）
- **主人在 = 千萬別偷懶**（最大恐懼）
- 塔倫長官出現 = 不是平時巡邏，是有事

## § 6 巴爺跟玩家的好感互動線（完整故事）

> 使用者要求「**完整的人生跟故事**」+「跟玩家有好感上的互動」 — 比奧蘭 / 赫克托稍輕一點，但比卡西烏斯更厚

### storyReveals（5 段 — v6 巴爺感激狀態 + 暗示者旁證）

| ID | aff | 揭露（玩家視角）|
|---|---|---|
| `overseer_silent` | 0 | flavor — 他在訓練場不講廢話。罵的時候不重複 |
| `overseer_old_scar` | 15 | flavor — 你發現他左前臂有一道又長又老的疤痕。最後一戰留的 |
| `overseer_friends_call` | 30 | event — 你聽到梅拉叫他「巴爺」的時候，他笑了一下。第一次看到他笑 |
| `overseer_thanks_tarren` | 50 | event — 巴爺喝多了。他舉杯對著塔倫的方向（塔倫不在）：「**⋯⋯這條命是長官給的。**」第一次你聽他這樣說 |
| `overseer_loyal` | 75 | flavor — 你看到塔倫私下叫巴爺去做事、巴爺立刻就去。沒有半句話。對玩家說：「跟著長官混不會吃虧。」 |

**設計**：5 段呈現巴爺**感激痛哭流涕 + 完全信任**的真實狀態 — 玩家**只看到一個會幫塔倫做事的好人**。
真正起疑要靠**層 1 多源頭暗示**（老默 / 卡西烏斯 / 梅拉好感後的提醒）。

> **真相揭露 = 偷聽密謀事件**（見 § 3 揭露結構 層 3，主人 60 / 塔倫 40 / 巴爺 60）
> **告訴巴爺真相 = 玩家選擇**（見 § 3 揭露結構 層 4）→ 觸發訣別事件

### 互動事件（aff 門檻 + 條件觸發，~3-4 個）

#### 事件 1 — 第一次罵你罵到收手（aff ~25 + 你練得很勤）
```
（巴爺站在你旁邊看了很久。）
巴爺：你今天揮的這幾下——
（他停下。）
巴爺：⋯⋯算了。你自己知道。
（他走開。沒有罵。）
```
玩家心想：（這是肯定嗎？）

#### 事件 2 — 「巴爺」這個外號的由來（aff ~40 + 場上有梅拉）
- 梅拉端飯來訓練場，叫了一聲「巴爺，吃飯了」
- 巴爺沒回頭，但揮了揮手
- 玩家對白選項：
  - 問梅拉「為什麼叫他巴爺？」 → 梅拉笑著講三言兩語的過去
  - 不問 → 心想（⋯⋯這外號是怎麼來的？）

#### 事件 3 — 跟卡西烏斯的對戲（aff ~50 + 卡西烏斯也在場）
- 兩個老角鬥士偶遇 — 沒講話、互相點頭
- 玩家旁觀 → log 揭露「兩人是同期入場的」
- 後續卡西烏斯可能講一句「他比我撐得久」

#### 事件 4 — 訣別 / 真相揭露（v6 — 玩家告訴巴爺）

**v6 重構**：這個事件**只在玩家觸發層 4「告訴巴爺真相」時播放**。
玩家**沒告訴** → 不觸發訣別事件、巴爺繼續感激狀態到結局。
玩家**告訴了** → 觸發崩潰版訣別。

**觸發條件**：
- 偷聽密謀事件已觸發（`overheard_master_plot` flag）
- 玩家選擇「告訴巴爺」（層 4）
- 巴爺好感 ≥ 70

**對白（v6 — 玩家告訴巴爺、巴爺沉默崩潰版）**：

```
（夜深。你找到巴爺、把今天聽到的告訴他。）

巴爺：⋯⋯
（他不看你。手拿著酒杯、放在桌上沒喝。）

巴爺：⋯⋯你說「上次老巴一樣」？
你：⋯⋯對。一字不差。

（漫長的沉默。）

巴爺：⋯⋯哦。

（他把酒杯端起來。喝了一口。放下。）

巴爺：當年我最後一場——
巴爺：對手是個從別的訓練所借來的瘋子。
巴爺：那時候我以為——
巴爺：⋯⋯他媽運氣不好。

（他笑了一下。聲音很乾。）

巴爺：然後我重傷下場、長官親自跑來說「不能讓他死」。
巴爺：「我們訓練所要照顧自家人」。
巴爺：⋯⋯我那時候哭了。真的哭了。
巴爺：對著一個我以為救我的人，哭得像個孩子。

（他不再說話。喝光那杯酒。）

巴爺：⋯⋯謝了。

（他沒看你。）

巴爺：你聽好——
巴爺：當年沒人對我說這句話。所以這次我說。
巴爺：如果我倒了——繼續站到底。
巴爺：然後——
巴爺：⋯⋯別他媽讓他騙了你。聽到沒。

（他起身要走、又停下。）

巴爺：⋯⋯這事不要再提了。
巴爺：對誰都別提。包括我。
（他走了。）
```

**flag set**：
- `overseer_passed_torch` — 火炬傳承
- `overseer_realized_tarren` — 巴爺想通了
- `overseer_warned_about_tarren` — 隱含警告
- `overseer_no_more_talk` — 之後對白池切換為「沉默式恭敬」

**設計重量**：
- 玩家是「**告訴他真相的人**」 — 這比 v5「巴爺自己想通」有更重的玩家責任
- 巴爺**沒哭沒罵**、只是接受 — 最符合奴隸社會壓抑感
- 「⋯⋯謝了」是極輕的兩個字、卻是整段最重的對白（謝玩家、也謝這份痛）
- 「**這事不要再提了**」 — 巴爺選擇繼續扛、不讓玩家再為這件事擔事
- 之後巴爺**表面恢復恭敬**（對塔倫聽話如常），但內心已經不一樣
- `overseer_no_more_talk` flag 觸發後巴爺對玩家有特殊保護對白池（可選 Phase H）

**沒告訴巴爺的路線**：
- 玩家保留秘密、自己活在 dramatic irony 裡
- 結局判定可能加分支：「你保護了他的幻覺、也獨自背了這事」
- 訣別事件不觸發、aff 75 storyReveal `overseer_loyal` 是玩家最後看到的巴爺
