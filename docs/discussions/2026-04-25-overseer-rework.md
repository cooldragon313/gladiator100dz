# 監督官 / 塔倫長官分工 — 設計 spec v2

> 2026-04-25 起。第一版討論完使用者已定方向，這版是依使用者 spec 重寫。
> 仍有 2 個待決議題（標 ❓），其餘已定，回完就動手。

---

## § 1 角色定位（兩人對位）

### 塔倫長官 (officer)
**冷靜、遠慮的官僚 — 看數字、看主人、看回報**

- 喜歡能**幫主人賺錢**的人 → reliable / loyal（可靠聽話）
- **派戰士上場 = 商業決策**：
  - 要贏 → 戰士要贏**而且贏得漂亮**（票房）
  - 要輸 → 戰士要懂得**用對的方式輸**（賠率/觀眾愛憎/送對手人情）
- 不在訓練場吼人 — 他在算盤上加減

### 監督官 = 巴布魯斯（暱稱「巴爺」）
**退役角鬥士 — 看肌肉、看意志、看你今天有沒有撐住**

- 一句話人設：**「不管你們是誰，在這都要想盡辦法活下去撐下去站到底。少囉嗦，別找理由。」**
- 喜歡：**勇敢 / 勤勞**
- 討厭：**神經質 / 找理由**
- 跟玩家有**完整好感互動線**（見 § 6）

## § 2 巴爺的背景（v2 對齊時間線）

### 已定
- **本名**：巴布魯斯（Babrius）
- **暱稱**：「巴爺」— 老夥伴間用（梅拉 / 老默 / 卡西烏斯都這麼叫）
- **背景**：退役角鬥士、塔倫當年的監督對象（見 § 3 時間線）
- **稱呼演化**（玩家視角）：
  - 一般時候：「監督官」
  - 偶爾聽到 NPC 講「巴爺」 → 玩家慢慢拼湊出他的本名
  - 高好感後：玩家或某些對白可叫「巴布魯斯」（極親近 + 私下）

### 退役方式（已定為 C — 對齊新時間線）

**C 巔峰受傷** — 贏過幾年巔峰，最後一戰**塔倫故意配的強敵**讓他重傷退役。

- 對齊 § 3 v4 時間線：
  - 巴爺**真的贏過**（這就是為什麼他「太強眼」讓塔倫忌憚）
  - 最後一戰是**塔倫安排的死局**（對手太強、巴爺重傷沒死）
  - 塔倫順勢「力保」演好人（救一個重傷沒威脅的人零成本）
- 角色內在矛盾：「**我贏過 — 但我沒贏到底**」+ 隱約懷疑「**那場為什麼這麼難**」
- 對玩家講「撐下去站到底」時帶有自嘲重量
- 身體狀態：左前臂大疤痕（已寫進 § 6 storyReveal）+ 可能微跛 / 某動作不順

## § 3 跟塔倫的關係（**v6：完整動機鏈 — 主人 / 塔倫 / 巴爺三角**）

### ⚠️ 修正紀錄
- v1：上一代監督看好巴爺（邏輯炸）
- v2：兩人版時間線、塔倫公開貶低（缺乏 cunning 人設）
- v3-v4：塔倫設局送死 + 假裝力保
- v5：dramatic irony 結構（巴爺感激→揭露→崩潰）
- **v6（最新）**：使用者升級 — 完整三角動機（**主人愛錢愛名聲、塔倫沒能力只能算計、巴爺成招牌威脅塔倫**）+ 多源頭暗示揭露

### 三角人物動機鏈（v6 核心）

#### 主人 (masterArtus)
- 愛**錢**、愛**名聲**、想討好**領主**
- 喜歡自己的招牌明星（賺錢）— 但**可以被「賺一大筆 + 領主歡心」說服送出去**

#### 塔倫長官 (officer)
- 從**小奴隸戰士**爬起來
- 個人能力**不夠當招牌**（這是 v6 重要修正：他不是強者）
- 知道自己沒辦法走「成招牌→舔主人」的路
- → 走「**幫主人爭取最大回報**」的路上位
- 看數字、設局、賺錢給主人 = 自己上位的唯一方法

#### 巴爺 (overseer)
- 個人能力很好、成訓練場招牌之一
- **努力認真重情義**、深得大家喜好
- → 過於搶眼、塔倫害怕成競爭者
- → **被借局斷了未來**

### 時間線（v6 完整版）

```
多年前
├─ 主人有兩個招牌：巴爺 + 其他幾個（賺錢用）
├─ 塔倫 = 監督官，個人能力一般，但會算
└─ 巴爺 = 場上招牌、深得大家喜歡、塔倫忌憚他取代自己位置

塔倫的賭局（精髓 v6）
├─ 領主生日將至，主人需要討好的禮物
├─ 領主最近訓服了老虎、想找個強對手試試
├─ 塔倫獻計：「把巴爺送上去」
│  ├─ 「會賺一大筆錢 + 取得領主歡心」
│  └─ 主人猶豫（捨不得招牌）— 但被誘惑說服
├─ 條件：「**這次成功，讓我升長官**」（塔倫先談好的籌碼）
└─ 巴爺被推上場 — 對手：領主借來的強敵 / 訓練過的老虎 / 兩者組合

巴爺重傷（沒死）
├─ 主人雖然賠了招牌、但領主開心了 + 賺了一筆
├─ 塔倫獲得允諾：升長官
└─ 但塔倫立刻動下一步...

塔倫的二手（為自己訓練場勢力鋪路）
├─ 升長官後，訓練場需要新監督官
├─ 「重情義的巴爺對救命恩人會死心踏地」
│  → 完美的「有能力 + 聽話」工具人
├─ 力保巴爺：「他是自家人」
│  → 表面演好人、實際是養一條會幫他養更多狗的狗
├─ 巴爺認塔倫為救命恩人、感激痛哭流涕
└─ 塔倫：「妙計」

現在
├─ 塔倫：表面長官、實際在訓練場仍有勢力（透過巴爺）
├─ 巴爺：完全信任塔倫、塔倫要他做什麼都認真做
│  └─ 平常會跟玩家說「跟著長官混不會吃虧」
├─ 塔倫常借巴爺手收拾事情（巴爺以為是恩義）
└─ 巴爺心裡：「我這條命是長官給的」
```

### 巴爺認知狀態（v7 升級 — 壓抑型不敢想）

**表層**：完全感激、忠誠、痛哭流涕級
- 把塔倫當愛護自己人的兄弟長官
- 塔倫嘴邊「為大人而活」、巴爺真的當價值觀
- 塔倫要他做什麼、巴爺都認真做

**底層（玩家高好感後可能感受到）**：**壓抑的疑慮**
- 為什麼那場對手那麼強？
- 為什麼塔倫剛好那時候出現「救」我？
- 為什麼塔倫升長官的時間點剛好那時候？

**心理機制**：他**不敢去想**
- 去想 = 否定自己這幾年的人生
- 去想 = 承認自己對塔倫的感激是建立在謊言上
- 所以他選擇「不要記臉、只記傷」（這正是老默的話、巴爺也用這套生存）

**訣別事件衝擊（v7 新解讀）**：
- 不是「我學到新事」
- 是「**我一直知道、但我不敢承認**」
- 「⋯⋯謝了」 = 「謝謝你戳破我自己騙自己這麼多年」
- 「這事不要再提了」 = 「我承認了、但我還是要繼續活下去」

### 揭露結構（v6 — 4 層遞進）

v6 把揭露拆 4 層 — 不塞給玩家、要玩家自己拼湊。

#### 層 1 — 多源頭暗示（v8 — 拆 3 個遞進小事件）

**老 NPC 不直接說「塔倫害了巴爺」**。世界就是這樣，講穿沒好處。
但他們會**用各自方式給玩家種種子** — 玩家累積三個線索後才有底氣往下挖。

##### 層 1a — 梅拉播種（aff 梅拉 ≥ 40）

**設計核心**：媽媽式的「**唉、望向、不點破**」— 把懷疑種在玩家心裡 + 自然產生「**巴爺怎了**」這個疑問。

**對白流程**：

```
（你來廚房附近、梅拉在擦手。）

梅拉：孩子。
（她抬頭看你。）
梅拉：我看你最近跟長官走得很好。
（壓低聲音。）
梅拉：自己留點心吧。
梅拉：別像某人——被賣了還傻傻地幫人家算錢⋯⋯
梅拉：（嘆氣。）唉。
（她望向訓練場的某個方向。）
```

**ChoiceModal**：

| 選項 | 結果 |
|---|---|
| **跟著梅拉的眼光看過去** | 你看到——**巴爺**。他正在訓練場上揮著鞭子、對著新人罵。但梅拉看著他的眼神不對。<br>玩家心想：（巴爺？⋯⋯巴爺怎了嗎？）<br>flag：`mela_hinted_overseer` |
| **「妳是說誰？」** | 梅拉笑了一下、沒答。「⋯⋯沒誰。我多嘴了。」（她回去切菜。）<br>flag：`mela_hinted_overseer`（仍 set，但沒順眼光看到巴爺）|
| **「⋯⋯嗯。」**（沉默接受暗示） | 梅拉拍拍你肩膀走了。<br>flag：`mela_hinted_overseer` |

> 📌 設計亮點：
> - **「順眼光看過去」**選項讓玩家自己連結梅拉 + 巴爺，比直接寫對白強百倍
> - 沒順眼光的玩家也能繼續推進（其他 NPC 對白會補）
> - **「巴爺怎了嗎」**這個疑問種進玩家心裡 → 後續去找老默 / 卡西烏斯時自然會帶這個問題

##### 引爆事件 — 主人注意到你（v8 — 事件觸發、不是好感觸發）

**設計核心**：層 1a 是「種子」、引爆事件才是「點火」。從這一刻起、玩家從**平常人**變成**被觀察的目標**、所有後續暗示對白才有重量。

**觸發條件**（任一）：
- 玩家**漂亮贏下**一場跨訓練所對戰（賭局賽事）
  - 「漂亮贏」 = 沒受重傷 / 觀眾喝采 / 對手強敵
- 主人首次來看玩家對戰且玩家表現好
- Day ≥ 30 + 玩家 fame ≥ 一定值

**事件流程**（賽後 stage 演出）：

```
（你贏了。觀眾的歡呼還沒停。）
（你站在沙地中央喘氣、血還沒擦。）

（看台上、主人站起來。）

主人：（拍了兩下手）
主人：這傢伙今天打得不錯。
主人：去給老默看看，幫他把傷口整理一下。
（他坐回去、繼續喝酒。）

（你旁邊——塔倫長官走過來、拍了拍你肩膀。）
塔倫：（笑著）這新人最近表現不錯。也很認真。
（他笑著走開。但他的眼神——）
（你說不清那眼神是什麼。）

（內心獨白：⋯⋯主人剛剛叫了我的名字。）
```

**flag set**：
- `master_noticed_player` — 主人公開稱讚（觸發後續事件條件）
- `tarren_calculating` — 塔倫心裡打算盤（內心獨白未明說、但玩家感受到了）

**設計亮點**：
- **沒對白選項** — 這是純演出、玩家沒得選
- 「**他笑著走開。但他的眼神——你說不清那眼神是什麼。**」這句**埋種子**（之後偷聽密謀時玩家會回想）
- 「主人叫了我的名字」是這個遊戲的**轉折點** — 玩家從「角鬥士」變成「資產」

##### 層 1b — 老默接話（引爆後觸發）

**觸發條件**（必須 AND）：
- `master_noticed_player` flag（引爆事件已發生）
- 玩家因傷找老默（自然劇情接續）
- 老默 aff ≥ 50

**對白**：

```
（老默替你包紮。動作比平常慢一點。）

老默：⋯⋯你也開始參與賭局賽事了嗎？
（他抬頭看你一眼。）
老默：嗯⋯⋯你自己小心吧。
（繼續包紮。）
老默：⋯⋯別步上跟——
（聲音壓低、自言自語式。）
老默：（小聲）⋯⋯的後塵。
（他停下動作。）
老默：唉⋯⋯
```

**ChoiceModal**（玩家追問）：

| 選項 | 結果 |
|---|---|
| **「誰？跟誰的後塵？」** | 老默搖頭不語：「傷口我幫你包紮好了。小心為之吧。」<br>flag：`doctor_hinted_overseer` |
| **「⋯⋯我懂。」**（沉默接受） | 老默看你一眼、點點頭：「⋯⋯這樣最好。記得多吃點、傷會好得快。」<br>flag：`doctor_hinted_overseer` |
| **「賭局賽事怎麼了？」**（裝傻 / 真不懂） | 老默：「⋯⋯沒事。是我多嘴。」（繼續包紮）<br>flag：`doctor_hinted_overseer`（仍 set）|

**為什麼老默不講穿**：他知道這話傳出去會死人。但他無法什麼都不說 — 火刑灼痕的記憶讓他看見玩家走巴爺老路無法視而不見。

##### 層 1c — 卡西烏斯補刀（引爆 + 老默後）

**觸發條件**（必須 AND）：
- `master_noticed_player` flag（引爆事件已發生）
- `doctor_hinted_overseer` flag（老默已暗示）
- 卡西烏斯在場 + aff ≥ 50

**對白**：

```
（你坐在沙地邊休息。卡西烏斯走過來坐下。沒看你。）

卡西烏斯：⋯⋯你最近越來越顯眼了。
（他喝一口水。）
卡西烏斯：也開始進入某些人的視線了。
（沉默很久。）
卡西烏斯：⋯⋯想當年老巴也是。
（他不看你、看著訓練場上揮鞭子的巴爺。）
```

**ChoiceModal**：

| 選項 | 結果 |
|---|---|
| **「巴爺怎麼了？」** | 卡西烏斯沉默很久：「⋯⋯有些事，知道不會比不知道好。」（站起來走了。）<br>flag：`cassius_hinted_overseer` |
| **「⋯⋯什麼意思？」** | 卡西烏斯：「你會懂的。早或晚。」（站起來走了。）<br>flag：`cassius_hinted_overseer` |
| **「我會小心。」**（接受暗示）| 卡西烏斯點點頭：「⋯⋯那就好。」（拍拍你肩膀走了。）<br>flag：`cassius_hinted_overseer` + 卡西烏斯 aff +3 |

#### 層 1 累積判定 → 解鎖層 2/3

**完成條件**：
- `mela_hinted_overseer` （層 1a 梅拉播種）
- `master_noticed_player` （引爆事件）
- `doctor_hinted_overseer` （層 1b 老默）
- `cassius_hinted_overseer` （層 1c 卡西烏斯）

**任 3 個達成** = 層 1 完成、解鎖層 2（塔倫曖昧指令）+ 層 3（偷聽密謀）。

**節奏摘要**：

```
種子（梅拉播種、可任意時候）
      ↓
引爆（玩家漂亮贏一場 → 主人公開稱讚 → 塔倫打算盤）
      ↓
回響（老默在治療時提 → 卡西烏斯在休息時提）
      ↓
證據（層 3 偷聽密謀）
      ↓
選擇（告訴 vs 不告訴巴爺）
```

> ⚠️ **老默的整合**：老默現有對白已寫死、火刑灼痕已 storyReveal。
> 解法：層 1b 是**新增的「玩家受傷 + master_noticed_player flag」分支**對白、不動現有對白。

#### 層 2 — 塔倫好感的副作用（v7 — 3 個曖昧指令 + 特性過濾反應）

**v7 升級**（依使用者建議）：
- **3 個曖昧指令**事件（不是 1-2 個）
- 每個指令彈 **ChoiceModal**、選項依玩家**累積特性**過濾
- 不能讓正直特性玩家「將計就計」、不能讓狡猾特性玩家「正氣凜然」
- 永遠至少有 **baseline 中性選項**（任何特性都能選）

##### 指令 1 — 對切磋對手下重手（aff ≥ 30）

**情境**：塔倫私下叫住玩家：「下次切磋時對 OOO **認真打**、下重手 — 那小子最近太囂張、得敲一下。」
（OOO = 在場某 NPC，依當下場景動態選）

**ChoiceModal 選項**（trait-gated）：

| 選項 | 過濾條件 | 結果 |
|---|---|---|
| **「⋯⋯遵命。」**（baseline）| 所有人可選 | 照做、塔倫 +5、被敲的 NPC -10、moral:loyalty negative weight 1 |
| **「他做錯什麼了？」**（質疑）| `requireAnyTrait: ['reliable','kindness','merciful']` | 塔倫: -2、塔倫沉默會記下來、後續曖昧指令觸發率降低 |
| **「⋯⋯為什麼？」**（衝動爆走）| `requireTrait: 'impulsive'` | 塔倫 -5、`tarren_saw_player_doubt` flag、玩家差點就直接質問了塔倫 |
| **「了解。誰下手都是他活該。」**（將計就計）| `requireAnyTrait: ['cruel','opportunist']` | 塔倫 +8、被敲的 NPC -15、玩家**主動配合**（塔倫從此把你當圈內人）|
| **「我會看狀況。」**（內斂）| `requireAnyTrait: ['patient','prideful','iron_will']` | 塔倫 +2、玩家保留決定權、後續實戰時才決定要不要真的下重手 |

##### 指令 2 — 對主人說謊報訓練狀況（aff ≥ 35）

**情境**：「主人最近會問你訓練如何。記得說一切都好——包括你的傷。聽懂了嗎？」

**為什麼曖昧**：玩家心想（為什麼要對主人隱瞞？是塔倫怕主人知道訓練不順、還是塔倫在搞鬼？）

**ChoiceModal 選項**：

| 選項 | 過濾條件 | 結果 |
|---|---|---|
| **「了解。」**（baseline）| 所有人可選 | 塔倫 +3、後續主人接觸事件玩家會自動說謊 |
| **「主人問什麼是主人的事。」**（正直）| `requireAnyTrait: ['reliable','prideful']` | 塔倫 -3、塔倫眼神冷下來「⋯⋯隨你」 |
| **「為什麼？」**（要解釋）| `requireAnyTrait: ['neurotic','kindness']` | 塔倫含糊帶過「就為你好。」、玩家心中懷疑加一 |
| **「我可以幫你說得更漂亮。」**（投機）| `requireAnyTrait: ['opportunist','cunning']` | 塔倫 +10、`tarren_player_inner_circle` flag — 塔倫開始當你是圈內 |
| **「我不擅長說謊。」**（推託）| `requireTrait: 'humble'` | 塔倫 -1、塔倫給你台階：「那就⋯⋯少說話。」 |

##### 指令 3 — 替塔倫保管「東西」（aff ≥ 50）

**情境**：「這個小布袋你幫我收著。誰問你都不知道。三天後我來拿。」（塔倫遞給你一個小布袋、不解釋裡面是什麼）

**為什麼曖昧**：玩家完全不知道是什麼（毒？情報？金錢？）。**這是塔倫的測試** — 看玩家會不會偷看 / 會不會說溜嘴。

**ChoiceModal 選項**：

| 選項 | 過濾條件 | 結果 |
|---|---|---|
| **接過來、不問**（baseline）| 所有人可選 | 三天後還回去、塔倫 +5 |
| **「⋯⋯這合法嗎？」**（正直）| `requireAnyTrait: ['reliable','prideful','iron_will']` | 塔倫眼神變了「⋯⋯算了。我找別人。」、塔倫 -5 |
| **接過來偷偷看裡面**（狡猾）| `requireAnyTrait: ['opportunist','cunning','curious']` | 看到裡面是密信 / 賭金 / 不該存在的東西、`saw_tarren_secret` flag、揭露線索之一 |
| **「我怕保管不好。」**（膽怯推託）| `requireAnyTrait: ['coward','neurotic']` | 塔倫 -3、塔倫表情失望「⋯⋯沒膽。」 |
| **「我幫你保管。但拿回去那天我們聊聊。」**（將計就計）| `requireTrait: 'patient'` + `requireMinAttr: { WIL: 15 }` | 塔倫 +3、玩家獲得跟塔倫單獨對話機會（後續事件條件） |

→ 玩家做完每個指令後可能心想：「**這真的是好人會給的指令嗎？**」

#### 層 3 — 偷聽密謀事件（v6 核心揭露）

**觸發條件**（你提的數字）：
- 主人好感 ≥ 60
- 塔倫好感 ≥ 40
- 監督官好感 ≥ 60

**事件流程**：
1. 主人召喚玩家
2. 玩家走到主人房門口、**恰巧聽到塔倫跟主人密謀**（不是主動偷聽）
3. 對白：

```
主人：⋯⋯領主生日又到了。總要找個理由討好他。
塔倫：大人說的是。
塔倫：新來的這幾個——有一個不錯。最近名聲也達到了。
塔倫：要不要再送一場好戲給領主開心？
主人：嗯⋯⋯
塔倫：聽說領主前陣子訓服了一頭老虎——
塔倫：他一直想找個夠看的對手試試。
主人：⋯⋯哦？
塔倫：不然推出 [玩家名] 去送頭？
塔倫：像上次老巴一樣⋯⋯
主人：（大笑）妙計！
（兩人竊笑。）
（你站在門口、沒敢進去。心臟在跳。）
```

4. 玩家當下兩個衝擊：
   - 「**上次老巴一樣**」= 塔倫設計巴爺的鐵證
   - **[玩家名]** = 你是下一個被送的人

⚠️ **設計關鍵**：「[玩家名]」要動態替換玩家自己的角色名

#### 層 4 — 告訴巴爺（玩家的選擇）

**觸發**：層 3 後 + 巴爺好感 ≥ 60

玩家可選：
- **A 告訴巴爺真相** → 觸發訣別事件（崩潰版）
- **B 不告訴**（保護巴爺幻覺）→ 訣別不觸發 / 溫和版本

→ **道德重量** — 真相對巴爺是恩賜還是傷害？玩家決定。

---

### 為什麼這個結構強

1. **玩家有特權體驗** — 觀眾比角色先知道（dramatic irony）
2. **多源頭暗示**避免「一個 NPC 講出來」太直白
3. **層 2 親身感受**讓玩家自己懷疑塔倫人格、不靠別人講
4. **層 3 偷聽事件**= 鐵證 + 自己也成下一個目標的雙重衝擊
5. **層 4 玩家選擇**有道德重量（傳真相 vs 保護幻覺）

## § 4 likedTraits / dislikedTraits 重排

### 塔倫長官（依使用者 spec）
**核心邏輯：能幫主人賺錢的人 = 可靠 + 聽話 + 會贏會輸**

```js
likedTraits:    { reliable:3, loyal:3, patient:2, iron_will:1 }
dislikedTraits: { impulsive:3, opportunist:3, prideful:2, coward:1 }
```

- `reliable` 升 +3（從 +3 維持）：說到做到 = 派得出去
- `loyal` 升 +3（從 +2）：聽話 = 主人的工具
- `patient` 維持 +2：不衝動 = 場上能控節奏
- `prideful` 變 -2（從 -1 加重）：場上要會輸的人不能驕傲
- `coward` 降 -1（從 -2 減輕）：怕死的人有時候反而會聽話

### 監督官巴爺（依使用者 spec）
**核心邏輯：撐下去站到底 = 勇敢 + 勤勞 / 討厭找理由**

```js
likedTraits:    { iron_will:3, diligence:3, prideful:2, cruel:1 }
dislikedTraits: { neurotic:3, coward:3, opportunist:2, merciful:1 }
```

- `iron_will` +3：撐到底的能力
- `diligence` +3：勤勞 = 不偷懶 = 不找理由
- `prideful` +2：勇敢敢撐才能站到底（跟塔倫相反）
- `cruel` +1：場上能下手 = 能活
- `neurotic` -3：「想太多」最怕（**新增**，配合「不要找理由」）
- `coward` -3：怕死撐不過第一場
- `merciful` -1：場上心軟會死

**塔倫 vs 巴爺最大反差**：
- `prideful`：塔倫 -2 / 巴爺 +2 — 「驕傲是奴隸的奢侈」 vs 「沒驕傲撐不下去」
- `loyal`：塔倫 +3 / 巴爺 0 — 「聽話最重要」 vs 「我管你聽不聽話、能撐就好」

## § 5 抓偷懶分工（依使用者 spec）

### 出現機率階梯（從低到高）
| 抓人者 | 偷懶被抓機率 | 場景 |
|---|---|---|
| **主人 (masterArtus)** | 很低（~5%） | 主人很少巡訓練場 |
| **侍從 (masterServant)** | 低（~10%） | 主人代行 |
| **塔倫長官 (officer)** | 中（~20%） | 平時不巡、出現 = 有事 |
| **監督官（巴爺）** | **75%+** | 他工作就是盯 |

### 沒人抓的情境
- 25% 的時間「**沒人出現**」 = 玩家好好休息的時段（恢復一些 stamina + mood）

### 抓包後的處罰邏輯（依抓人者 + 巴爺好感）

| 抓人者 | 處罰 |
|---|---|
| **主人（masterArtus）出現** | **大懲罰** — HP -10、好感 -15、震動+紅光 + 對白：「主人面前偷懶 = 大忌」 |
| **塔倫長官** | 中懲罰 — HP -5、好感 -10、冷酷對白「沒救的東西」 |
| **巴爺**：好感 < 30 | 直接抽打 — HP -5、好感 -3、抽鞭演出 |
| **巴爺**：好感 ≥ 30 | **只大吼** — HP 0、好感 -1、罵兩句完事「你他媽再讓我看到一次⋯⋯」 |
| **巴爺**：好感 ≥ 60 | 假裝沒看到 — 好感 -0、對白：「⋯⋯我這次當沒看到。下次別再讓我看見。」 |
| **侍從** | 輕懲 — HP -2、好感 -5（侍從會去報告主人）、後續主人可能找上門 |
| **無人** | 平安偷懶 — stamina +5 / mood +3 |

### 設計意圖（玩家學到什麼）
- **跟巴爺打好關係 → 偷懶被抓不會死**（是這個訓練所唯一的緩衝）
- **主人在 = 千萬別偷懶**（最大恐懼）
- 塔倫長官出現 = 不是平時巡邏，是有事

## § 6 巴爺跟玩家的好感互動線（完整故事）

> 使用者要求「**完整的人生跟故事**」+「跟玩家有好感上的互動」 — 比奧蘭 / 赫克托稍輕一點，但比卡西烏斯更厚

### storyReveals（5 段 — v6 巴爺感激狀態 + 暗示者旁證）

| ID | aff | 揭露（玩家視角）|
|---|---|---|
| `overseer_silent` | 0 | flavor — 他在訓練場不講廢話。罵的時候不重複 |
| `overseer_old_scar` | 15 | flavor — 你發現他左前臂有一道又長又老的疤痕。最後一戰留的 |
| `overseer_friends_call` | 30 | event — 你聽到梅拉叫他「巴爺」的時候，他笑了一下。第一次看到他笑 |
| `overseer_thanks_tarren` | 50 | event — 巴爺喝多了。他舉杯對著塔倫的方向（塔倫不在）：「**⋯⋯這條命是長官給的。**」第一次你聽他這樣說 |
| `overseer_loyal` | 75 | flavor — 你看到塔倫私下叫巴爺去做事、巴爺立刻就去。沒有半句話。對玩家說：「跟著長官混不會吃虧。」 |

**設計**：5 段呈現巴爺**感激痛哭流涕 + 完全信任**的真實狀態 — 玩家**只看到一個會幫塔倫做事的好人**。
真正起疑要靠**層 1 多源頭暗示**（老默 / 卡西烏斯 / 梅拉好感後的提醒）。

> **真相揭露 = 偷聽密謀事件**（見 § 3 揭露結構 層 3，主人 60 / 塔倫 40 / 巴爺 60）
> **告訴巴爺真相 = 玩家選擇**（見 § 3 揭露結構 層 4）→ 觸發訣別事件

### 互動事件（aff 門檻 + 條件觸發，~3-4 個）

#### 事件 1 — 第一次罵你罵到收手（aff ~25 + 你練得很勤）
```
（巴爺站在你旁邊看了很久。）
巴爺：你今天揮的這幾下——
（他停下。）
巴爺：⋯⋯算了。你自己知道。
（他走開。沒有罵。）
```
玩家心想：（這是肯定嗎？）

#### 事件 2 — 「巴爺」這個外號的由來（aff ~40 + 場上有梅拉）
- 梅拉端飯來訓練場，叫了一聲「巴爺，吃飯了」
- 巴爺沒回頭，但揮了揮手
- 玩家對白選項：
  - 問梅拉「為什麼叫他巴爺？」 → 梅拉笑著講三言兩語的過去
  - 不問 → 心想（⋯⋯這外號是怎麼來的？）

#### 事件 3 — 跟卡西烏斯的對戲（aff ~50 + 卡西烏斯也在場）
- 兩個老角鬥士偶遇 — 沒講話、互相點頭
- 玩家旁觀 → log 揭露「兩人是同期入場的」
- 後續卡西烏斯可能講一句「他比我撐得久」

#### 事件 4 — 訣別 / 真相揭露（v6 — 玩家告訴巴爺）

**v6 重構**：這個事件**只在玩家觸發層 4「告訴巴爺真相」時播放**。
玩家**沒告訴** → 不觸發訣別事件、巴爺繼續感激狀態到結局。
玩家**告訴了** → 觸發崩潰版訣別。

**觸發條件**：
- 偷聽密謀事件已觸發（`overheard_master_plot` flag）
- 玩家選擇「告訴巴爺」（層 4）
- 巴爺好感 ≥ 70

**對白（v6 — 玩家告訴巴爺、巴爺沉默崩潰版）**：

```
（夜深。你找到巴爺、把今天聽到的告訴他。）

巴爺：⋯⋯
（他不看你。手拿著酒杯、放在桌上沒喝。）

巴爺：⋯⋯你說「上次老巴一樣」？
你：⋯⋯對。一字不差。

（漫長的沉默。）

巴爺：⋯⋯哦。

（他把酒杯端起來。喝了一口。放下。）

巴爺：當年我最後一場——
巴爺：對手是個從別的訓練所借來的瘋子。
巴爺：那時候我以為——
巴爺：⋯⋯他媽運氣不好。

（他笑了一下。聲音很乾。）

巴爺：然後我重傷下場、長官親自跑來說「不能讓他死」。
巴爺：「我們訓練所要照顧自家人」。
巴爺：⋯⋯我那時候哭了。真的哭了。
巴爺：對著一個我以為救我的人，哭得像個孩子。

（他不再說話。喝光那杯酒。）

巴爺：⋯⋯謝了。

（他沒看你。）

巴爺：你聽好——
巴爺：當年沒人對我說這句話。所以這次我說。
巴爺：如果我倒了——繼續站到底。
巴爺：然後——
巴爺：⋯⋯別他媽讓他騙了你。聽到沒。

（他起身要走、又停下。）

巴爺：⋯⋯這事不要再提了。
巴爺：對誰都別提。包括我。
（他走了。）
```

**flag set**：
- `overseer_passed_torch` — 火炬傳承
- `overseer_realized_tarren` — 巴爺想通了
- `overseer_warned_about_tarren` — 隱含警告
- `overseer_no_more_talk` — 之後對白池切換為「沉默式恭敬」

**設計重量**：
- 玩家是「**告訴他真相的人**」 — 這比 v5「巴爺自己想通」有更重的玩家責任
- 巴爺**沒哭沒罵**、只是接受 — 最符合奴隸社會壓抑感
- 「⋯⋯謝了」是極輕的兩個字、卻是整段最重的對白（謝玩家、也謝這份痛）
- 「**這事不要再提了**」 — 巴爺選擇繼續扛、不讓玩家再為這件事擔事
- 之後巴爺**表面恢復恭敬**（對塔倫聽話如常），但內心已經不一樣
- `overseer_no_more_talk` flag 觸發後巴爺對玩家有特殊保護對白池（可選 Phase H）

**沒告訴巴爺的路線**：
- 玩家保留秘密、自己活在 dramatic irony 裡
- 結局判定可能加分支：「你保護了他的幻覺、也獨自背了這事」
- 訣別事件不觸發、aff 75 storyReveal `overseer_loyal` 是玩家最後看到的巴爺
