# 塔倫長官 / 監督官分工 — 方向 A 細節討論

> 2026-04-25 起。方向 A = 保留兩人 + 強化區分（避免功能撞）。
> 拆 7 個子決策，逐項回。確認後一次落實。

---

## § 1 監督官的名字

目前他叫「**監督官**」（無名）。塔倫有名字，他沒名字 — 這本身就暗示階層。

但**有名字 = 容易讓玩家記住**，敵對 NPC 才能成為「壓在你頭上的人臉」。

| 方案 | 名字 | 感覺 |
|---|---|---|
| A1 | **保留無名**「監督官」 | 強調他是個系統執行者、可被替換的工具人 |
| A2 | **格拉克斯（Grax）** | 短促、硬、軍人感（5 字以內好記） |
| A3 | **巴布魯斯（Babrius）** | 古羅馬風 + 諧音「霸」(Bab- 像「巴爺」) |
| A4 | **使用者自填** | 你想要的 |

**我的偏好**：A2 格拉克斯 — 兩個音節、很硬、跟塔倫（Tarren）長度一樣對等

## § 2 監督官的背景設定

塔倫是「**官僚 / 計算 / 看數字**」。監督要對位 → 應該是**動手 / 直接 / 看肌肉**。

| 方案 | 背景設定 | 帶來的對白可能性 |
|---|---|---|
| B1 | **退伍軍團百夫長**（軍人 → 訓練奴隸） | 可講戰場故事、軍紀風格、看不起「沒上過戰場的官僚」（暗諷塔倫） |
| B2 | **前角鬥士**（活下來、被買下當訓練官） | 跟玩家有共鳴感、會講「我上過場、我知道」、對奧蘭/索爾這種「沒希望的」會更狠 |
| B3 | **訓練奴隸出身**（被養大就是當訓練官） | 沒有外面的世界、訓練場就是他全部、最冷血最系統化 |
| B4 | **混合**：B1 + B2（退伍軍人後被抓當鬥士再被提拔） | 最複雜但最有戲 |

**我的偏好**：**B2 前角鬥士** — 跟主角關係最深、「他就是你的未來如果你活下來」、能跟卡西烏斯有戲（兩個老鬥士的對立或共鳴）

## § 3 監督官跟塔倫的關係

兩個權威 NPC 之間的張力很重要。

| 方案 | 關係 | 帶來的張力 |
|---|---|---|
| C1 | **互相看不起**（塔倫覺得他是粗人、他覺得塔倫是書桌官僚） | 玩家可以利用兩人矛盾（高階任務） |
| C2 | **配合默契**（塔倫定戰略、他執行） | 統一戰線壓玩家、絕望感更強 |
| C3 | **塔倫是他的提拔者**（恩情）| 他絕對忠於塔倫、衝突時站塔倫 |
| C4 | **舊敵**（曾經站對立陣營） | 表面合作底下有刀 |

**我的偏好**：**C1 互相看不起** — 給玩家未來「離間」的設計空間，且符合 B2「前角鬥士看不起官僚」的人設邏輯

## § 4 likedTraits / dislikedTraits 重新分配

目前撞嚴重（兩個都喜歡 iron_will、都討厭 coward/impulsive）。

**塔倫保留現況**（reliable / patient / loyal / iron_will；恨 impulsive / opportunist / coward / prideful）— 存活派的官僚

**監督新版**（依 B2 前角鬥士 + C1 看不起官僚的設定）：

| 方案 | 喜歡 | 厭惡 |
|---|---|---|
| D1 | **diligence(3)、cruel(2)、prideful(2)、impulsive(1)** | merciful(3)、reliable(2)、coward(2) |
|  | 邏輯：肯練的 / 殺氣強的 / 有膽 / 衝動沒關係 | 邏輯：心軟在這活不久 / 太守規矩沒爆發力 |
| D2 | **diligence(3)、aggressive(2)、prideful(2)、iron_will(1)** | coward(3)、merciful(2)、opportunist(1) |
|  | 邏輯：嚴師愛苦練的、看不起算計型 | 邏輯：怕死的看不下去、心軟也是 |
| D3 | 你提的 |  |

**我的偏好**：**D1** — 跟塔倫做出最大區隔（塔倫愛 reliable / 他愛 cruel；塔倫恨 impulsive / 他覺得衝動沒關係）

關鍵差異：**塔倫看「能不能活到下個月」、監督看「現在能不能打」**

## § 5 抓偷懶分工

目前主畫面偷懶被抓會隨機抽 overseer / officer / masterServant。撞最厲害的就在這。

| 方案 | 分工規則 | 體感 |
|---|---|---|
| E1 | **時段切**：14-16 點（監督值班）→ 監督抓；其他時段 → 隨機（侍從 / 巡邏兵） | 玩家學到「下午別偷懶」 |
| E2 | **嚴重度切**：日常偷懶 → 監督抓；重大過失（偷東西、嫁禍）→ 塔倫處理 | 兩人功能完全分開 |
| E3 | **層級切**：監督**永遠**抓你偷懶；塔倫只在「監督報告上去之後」才出現（重複犯才驚動他） | 階層感最強 |
| E4 | **E1 + E3 組合** | 完整 |

**我的偏好**：**E4 組合**
- 14-16 點偷懶 → 監督當場抓（鞭刑 + 罵）
- 其他時段偷懶 → 侍從報告 / 巡邏抓
- 連續 3 次被監督抓 → 監督報告塔倫 → 塔倫親自出場（罕見、嚴重事件）

## § 6 塔倫從訓練場「退場」

目前塔倫常常出現在 audience，跟監督撞畫面。退場後他要「**特殊事件才出場**」，這樣他出場就有重量。

| 方案 | 出場頻率 | 出場時機 |
|---|---|---|
| F1 | **完全退出 audience pool** | 只在 storyEvent / DialogueModal 出場 |
| F2 | **保留低頻 audience（chance 0.15）** | 偶爾巡視，露個臉 |
| F3 | **特定日子才出**（Day 5/10/25/40 等大事日）| 「他來了 = 有事」 |
| F4 | **F2 + F3 混合** | 平日偶爾、大日子必到 |

**我的偏好**：**F4 混合** — 平日 0.15 機率（讓玩家偶爾感受到他的存在），大日子必到。塔倫「**不用整天看著你 — 但你永遠知道他在看資料**」更貼合 cunning 人設

## § 7 監督官的場景戲份

讓監督也有「自己的故事」（不只是抓偷懶）。

| 方案 | 戲份規模 |
|---|---|
| G1 | **不加**：純功能執法者，沒個人故事 |
| G2 | **加 1-2 個 storyReveal**（aff ≥ 30/60 看到他的舊鬥士疤痕、聽他講過往）|
| G3 | **加完整劇情線**（仿奧蘭/赫克托規模）— 觀察玩家、選你當接班人之類 |
| G4 | **加跟卡西烏斯的對戲事件**（兩個老鬥士、不同選擇路線）|

**我的偏好**：**G2 + G4** — G2 給玩家慢慢揭開他的過去（前角鬥士身分），G4 跟卡西烏斯對戲是天然的對比點（同樣老鬥士，一個成監督、一個成隊友）。但 G3 完整劇情線可以等 Phase 2
