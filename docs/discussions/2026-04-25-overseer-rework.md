## § 1 角色定位（兩人對位）

### 塔倫長官 (officer)
**冷靜、遠慮的官僚 — 看數字、看主人、看回報**

- 喜歡能**幫主人賺錢**的人 → reliable / loyal（可靠聽話）
- **派戰士上場 = 商業決策**：
  - 要贏 → 戰士要贏**而且贏得漂亮**（票房）
  - 要輸 → 戰士要懂得**用對的方式輸**（賠率/觀眾愛憎/送對手人情）
- 不在訓練場吼人 — 他在算盤上加減

### 監督官 = 巴布魯斯（暱稱「巴爺」）
**退役角鬥士 — 看肌肉、看意志、看你今天有沒有撐住**

- 一句話人設：**「不管你們是誰，在這都要想盡辦法活下去撐下去站到底。少囉嗦，別找理由。」**
- 喜歡：**勇敢 / 勤勞**
- 討厭：**神經質 / 找理由**
- 跟玩家有**完整好感互動線**（見 § 6）

## § 2 巴爺的背景（v2 對齊時間線）

### 已定
- **本名**：巴布魯斯（Babrius）
- **暱稱**：「巴爺」— 老夥伴間用（梅拉 / 老默 / 卡西烏斯都這麼叫）
- **背景**：退役角鬥士、塔倫當年的監督對象（見 § 3 時間線）
- **稱呼演化**（玩家視角）：
  - 一般時候：「監督官」
  - 偶爾聽到 NPC 講「巴爺」 → 玩家慢慢拼湊出他的本名
  - 高好感後：玩家或某些對白可叫「巴布魯斯」（極親近 + 私下）

### 退役方式（已定為 C — 跟新時間線最相容）

**C 巔峰受傷** — 贏過幾年巔峰，最後一戰重傷退役。

- 跟 § 3 「塔倫想賣掉他、某人力保」最相容
- 角色內在矛盾：「**我贏過 — 但我沒贏到底**」
- 對玩家講「撐下去站到底」時帶有自嘲重量（他自己最後一場沒站到底）
- 身體狀態：左前臂大疤痕（已寫進 § 6 storyReveal）+ 可能微跛 / 某動作不順

> ⚠️ 這項從「待決 1」改為「已定」— 因為跟新時間線（塔倫想賣他、最後一戰沒贏到）邏輯綁死。
> 若你不同意，回我「不要 C，我要 A 或 B」就改回。

## § 3 跟塔倫的關係（**v2 修正：時間線**）

### ⚠️ 修正紀錄
v1 把「上一代監督官」當作第三人，跟「塔倫就是上一代監督」自相矛盾。
**v2 改用使用者提的兩人版** — 沒有第三人、更精煉、衝突更直接。

### 時間線（已定）

```
多年前
└─ 塔倫 = 監督官
   └─ 巴爺 = 那個訓練所的年輕角鬥士
      └─ 塔倫對年輕巴爺：「沒救了。」（不看好 + 偶有打壓）
         └─ 但巴爺撐下來了。場場活下來。

巴爺最後一戰受傷
└─ 塔倫想賣掉巴爺：「沒投資價值了。」
   └─ 某人力保（暫定：當時的廚娘 = 梅拉前任、或是主人賞識）
      └─ 巴爺留下，當訓練助手 / 雜役

幾年後
└─ 塔倫升長官（看數字升上去 — 賣鬥士的本事）
   └─ 監督位置空出
      └─ 主人指派巴爺接 — 因為他是訓練所裡唯一還活著的「老場上人」
         └─ 兩人現在表面合作、底下有當年的壓迫史
            └─ 這就是現在的局面
```

### 兩人現在的互動模式

- **巴爺看塔倫**：「當年瞧不起我的那個人。現在還是瞧不起。但他升上去了。」
  - 不會公開頂撞（這是奴隸社會、塔倫是上司）
  - 但**對塔倫的命令會慢一拍**、不熱絡、最低限度配合
- **塔倫看巴爺**：「沒死掉的那塊肉。會用、不會喜歡。」
  - 不公開貶他（巴爺現在是有用的工具）
  - 但**從不在主人面前提巴爺名字**、不給他任何超出本份的權限
  - 心裡仍認為「我當年沒看錯 — 他就是個粗人」

### 這套關係解鎖的劇情潛力

1. **巴爺對玩家的關注 = 自我投射**
   - 他當年被塔倫看不起、但他撐下來了
   - 看到玩家苦撐 → 「⋯⋯撐住。我看過你這種眼神。」
   - 不是「上一代老師看我所以我看你」（v1 版本），而是「**我當年沒老師、所以你不能也沒有**」 — 更慘更動人
2. **塔倫長官事件可加陰影**
   - 玩家好感巴爺夠高後 + 觀察塔倫指令時 → 偶爾揭露「巴爺臉色不對」的細節
   - 玩家後期可能藉巴爺的口知道「塔倫年輕時對他做過什麼」
3. **§ 6 訣別事件變得超重**
   - 巴爺把火炬交給玩家 = 他親身體會「被壓迫者撐到底」是什麼感覺
   - 「⋯⋯如果我倒了，繼續站到底。」 = 他當年沒人對他說這句、所以他要對你說

### ❓ 待決 2（修正版）：當年塔倫對巴爺的「打壓」程度

| 方案 | 塔倫當年對巴爺 | 影響 |
|---|---|---|
| **A 冷漠不投資** | 沒主動害他、但也沒特別教他 / 沒救他 | 巴爺對塔倫只是「不熟、不喜歡」 |
| **B 公開貶低** | 訓練場上常公開罵巴爺廢物、把他派去送死級的對手 | 巴爺對塔倫是「**恨**」 — 但這是奴隸社會、不能說 |
| **C 私下偷偷欣賞** | 表面打壓、私下其實有認到巴爺的韌性（但從不說出口）| 最複雜 — 兩人有未說破的曖昧情感（適合長劇本） |

**我的偏好**：**B 公開貶低** — 跟「沒救他、想賣掉」相容、解鎖最多戲劇張力。C 太複雜可能寫不出感覺。

## § 4 likedTraits / dislikedTraits 重排

### 塔倫長官（依使用者 spec）
**核心邏輯：能幫主人賺錢的人 = 可靠 + 聽話 + 會贏會輸**

```js
likedTraits:    { reliable:3, loyal:3, patient:2, iron_will:1 }
dislikedTraits: { impulsive:3, opportunist:3, prideful:2, coward:1 }
```

- `reliable` 升 +3（從 +3 維持）：說到做到 = 派得出去
- `loyal` 升 +3（從 +2）：聽話 = 主人的工具
- `patient` 維持 +2：不衝動 = 場上能控節奏
- `prideful` 變 -2（從 -1 加重）：場上要會輸的人不能驕傲
- `coward` 降 -1（從 -2 減輕）：怕死的人有時候反而會聽話

### 監督官巴爺（依使用者 spec）
**核心邏輯：撐下去站到底 = 勇敢 + 勤勞 / 討厭找理由**

```js
likedTraits:    { iron_will:3, diligence:3, prideful:2, cruel:1 }
dislikedTraits: { neurotic:3, coward:3, opportunist:2, merciful:1 }
```

- `iron_will` +3：撐到底的能力
- `diligence` +3：勤勞 = 不偷懶 = 不找理由
- `prideful` +2：勇敢敢撐才能站到底（跟塔倫相反）
- `cruel` +1：場上能下手 = 能活
- `neurotic` -3：「想太多」最怕（**新增**，配合「不要找理由」）
- `coward` -3：怕死撐不過第一場
- `merciful` -1：場上心軟會死

**塔倫 vs 巴爺最大反差**：
- `prideful`：塔倫 -2 / 巴爺 +2 — 「驕傲是奴隸的奢侈」 vs 「沒驕傲撐不下去」
- `loyal`：塔倫 +3 / 巴爺 0 — 「聽話最重要」 vs 「我管你聽不聽話、能撐就好」

## § 5 抓偷懶分工（依使用者 spec）

### 出現機率階梯（從低到高）
| 抓人者 | 偷懶被抓機率 | 場景 |
|---|---|---|
| **主人 (masterArtus)** | 很低（~5%） | 主人很少巡訓練場 |
| **侍從 (masterServant)** | 低（~10%） | 主人代行 |
| **塔倫長官 (officer)** | 中（~20%） | 平時不巡、出現 = 有事 |
| **監督官（巴爺）** | **75%+** | 他工作就是盯 |

### 沒人抓的情境
- 25% 的時間「**沒人出現**」 = 玩家好好休息的時段（恢復一些 stamina + mood）

### 抓包後的處罰邏輯（依抓人者 + 巴爺好感）

| 抓人者 | 處罰 |
|---|---|
| **主人（masterArtus）出現** | **大懲罰** — HP -10、好感 -15、震動+紅光 + 對白：「主人面前偷懶 = 大忌」 |
| **塔倫長官** | 中懲罰 — HP -5、好感 -10、冷酷對白「沒救的東西」 |
| **巴爺**：好感 < 30 | 直接抽打 — HP -5、好感 -3、抽鞭演出 |
| **巴爺**：好感 ≥ 30 | **只大吼** — HP 0、好感 -1、罵兩句完事「你他媽再讓我看到一次⋯⋯」 |
| **巴爺**：好感 ≥ 60 | 假裝沒看到 — 好感 -0、對白：「⋯⋯我這次當沒看到。下次別再讓我看見。」 |
| **侍從** | 輕懲 — HP -2、好感 -5（侍從會去報告主人）、後續主人可能找上門 |
| **無人** | 平安偷懶 — stamina +5 / mood +3 |

### 設計意圖（玩家學到什麼）
- **跟巴爺打好關係 → 偷懶被抓不會死**（是這個訓練所唯一的緩衝）
- **主人在 = 千萬別偷懶**（最大恐懼）
- 塔倫長官出現 = 不是平時巡邏，是有事

## § 6 巴爺跟玩家的好感互動線（完整故事）

> 使用者要求「**完整的人生跟故事**」+「跟玩家有好感上的互動」 — 比奧蘭 / 赫克托稍輕一點，但比卡西烏斯更厚

### storyReveals（5 段，aff 門檻遞增 — v2 對齊新時間線）
| ID | aff | 揭露 |
|---|---|---|
| `overseer_silent` | 0 | flavor — 他在訓練場不講廢話。罵的時候不重複 |
| `overseer_old_scar` | 15 | flavor — 你發現他左前臂有一道又長又老的疤痕。最後一戰留的 |
| `overseer_friends_call` | 30 | event — 你聽到梅拉叫他「巴爺」的時候，他笑了一下。第一次看到他笑 |
| `overseer_last_match` | 50 | event — 他喝多的某個夜晚對你講起最後一場 — 但講到塔倫的部分就停了 |
| `overseer_tarren_history` | 75 | event — 他終於說出來：「⋯⋯當年是長官想把我賣掉。是某個人留下我的。我從沒謝過那個人。」（沒說那是誰）|

### 互動事件（aff 門檻 + 條件觸發，~3-4 個）

#### 事件 1 — 第一次罵你罵到收手（aff ~25 + 你練得很勤）
```
（巴爺站在你旁邊看了很久。）
巴爺：你今天揮的這幾下——
（他停下。）
巴爺：⋯⋯算了。你自己知道。
（他走開。沒有罵。）
```
玩家心想：（這是肯定嗎？）

#### 事件 2 — 「巴爺」這個外號的由來（aff ~40 + 場上有梅拉）
- 梅拉端飯來訓練場，叫了一聲「巴爺，吃飯了」
- 巴爺沒回頭，但揮了揮手
- 玩家對白選項：
  - 問梅拉「為什麼叫他巴爺？」 → 梅拉笑著講三言兩語的過去
  - 不問 → 心想（⋯⋯這外號是怎麼來的？）

#### 事件 3 — 跟卡西烏斯的對戲（aff ~50 + 卡西烏斯也在場）
- 兩個老角鬥士偶遇 — 沒講話、互相點頭
- 玩家旁觀 → log 揭露「兩人是同期入場的」
- 後續卡西烏斯可能講一句「他比我撐得久」

#### 事件 4 — 訣別 / 託付（aff ~70 + Day ≥ 70）
- 巴爺在某個夜晚找你
- 對白（重量級）：
  ```
  巴爺：⋯⋯小子。
  巴爺：當年沒人對我說這句話。
  巴爺：所以這次我說。
  巴爺：如果我倒了——繼續站到底。聽到沒有？
  ```
- flag set：`overseer_passed_torch`
- 後續結局判定可能加分支
- 設計重量：呼應 § 3 時間線「巴爺當年沒人保 / 但他撐下來了」 — 他現在把這句話傳下去
