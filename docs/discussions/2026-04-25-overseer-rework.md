## § 1 角色定位（兩人對位）

### 塔倫長官 (officer)
**冷靜、遠慮的官僚 — 看數字、看主人、看回報**

- 喜歡能**幫主人賺錢**的人 → reliable / loyal（可靠聽話）
- **派戰士上場 = 商業決策**：
  - 要贏 → 戰士要贏**而且贏得漂亮**（票房）
  - 要輸 → 戰士要懂得**用對的方式輸**（賠率/觀眾愛憎/送對手人情）
- 不在訓練場吼人 — 他在算盤上加減

### 監督官 = 巴布魯斯（暱稱「巴爺」）
**退役角鬥士 — 看肌肉、看意志、看你今天有沒有撐住**

- 一句話人設：**「不管你們是誰，在這都要想盡辦法活下去撐下去站到底。少囉嗦，別找理由。」**
- 喜歡：**勇敢 / 勤勞**
- 討厭：**神經質 / 找理由**
- 跟玩家有**完整好感互動線**（見 § 6）

## § 2 巴爺的背景（v2 對齊時間線）

### 已定
- **本名**：巴布魯斯（Babrius）
- **暱稱**：「巴爺」— 老夥伴間用（梅拉 / 老默 / 卡西烏斯都這麼叫）
- **背景**：退役角鬥士、塔倫當年的監督對象（見 § 3 時間線）
- **稱呼演化**（玩家視角）：
  - 一般時候：「監督官」
  - 偶爾聽到 NPC 講「巴爺」 → 玩家慢慢拼湊出他的本名
  - 高好感後：玩家或某些對白可叫「巴布魯斯」（極親近 + 私下）

### 退役方式（已定為 C — 對齊新時間線）

**C 巔峰受傷** — 贏過幾年巔峰，最後一戰**塔倫故意配的強敵**讓他重傷退役。

- 對齊 § 3 v4 時間線：
  - 巴爺**真的贏過**（這就是為什麼他「太強眼」讓塔倫忌憚）
  - 最後一戰是**塔倫安排的死局**（對手太強、巴爺重傷沒死）
  - 塔倫順勢「力保」演好人（救一個重傷沒威脅的人零成本）
- 角色內在矛盾：「**我贏過 — 但我沒贏到底**」+ 隱約懷疑「**那場為什麼這麼難**」
- 對玩家講「撐下去站到底」時帶有自嘲重量
- 身體狀態：左前臂大疤痕（已寫進 § 6 storyReveal）+ 可能微跛 / 某動作不順

## § 3 跟塔倫的關係（**v4：設局陷害 + 假裝力保**）

### ⚠️ 修正紀錄
- v1：上一代監督看好巴爺（邏輯炸）
- v2：兩人版時間線、塔倫公開貶低（缺乏 cunning 人設）
- **v4（最新）**：使用者提的精彩版本 — 塔倫**暗中打壓 + 設局送死 → 失敗 → 順勢演好人**

### 時間線（已定）

```
多年前
└─ 塔倫 = 監督官（年輕、有野心）
   └─ 巴爺 = 訓練所的年輕角鬥士
      └─ 巴爺太強眼 / 太得人緣 / 太搶風頭
         → 塔倫覺得礙眼（這小子可能威脅我位置 / 太招搖不好控）
         → 但塔倫不公開打壓（cunning 不會留把柄）

塔倫的暗手
└─ 派巴爺打對手太強的場
   └─ 期待巴爺死或重傷 = 一勞永逸去掉一個變數
      └─ 結果：巴爺重傷、但沒死（撐下來了）

塔倫的反手（精髓）
└─ 「力保」巴爺：「不行不行，他是我們訓練所的人」
   └─ 表面：演照顧自己人、樹立好人形象（給主人看）
      └─ 實則：巴爺已經重傷、不能再上場 = 沒威脅了
                救他**零成本**、還能賺名聲（「塔倫是個重情義的長官」）
   └─ 主人/同期 NPC 至此都認為「塔倫是有情有義的人」
      → 這成了塔倫升職的助力之一

幾年後
└─ 塔倫升長官（這一手「重情義」形象幫他升）
   └─ 監督位置空 → 塔倫順勢推巴爺接
      → 又一次「我照顧自己人」的形象加分

現在
└─ 塔倫：「他不知道。他以為我是他恩人。」
   └─ 對玩家、對其他 NPC、表面是慷慨長官
└─ 巴爺：⚠️ **微妙曖昧** — 見下方「巴爺自己知道多少」
```

### ❓ 待決：巴爺自己知道多少？

這個是整條劇情的重心 — 影響所有對白的演法。

| 方案 | 巴爺的認知狀態 | 揭露給玩家的方式 |
|---|---|---|
| **A 完全不知道** | 塔倫真的救了他，他真的感激 | 沒戲 — 玩家從旁觀者看到不對勁 |
| **B 模糊懷疑** | 隱約感覺當年那場對手太強不對勁、但沒證據、選擇不深想 | **慢慢累積線索** — storyReveal 揭露他的疑慮、玩家後期可能幫他想通 |
| **C 心裡確定但沒證據** | 早就懂了、但忍著 / 沒辦法 / 不想承認 | 強角色內傷感、所有對白底下都有刀 |
| **D 幾乎確定 + 在等機會** | 他在等一個時機讓塔倫付代價 | 最戲劇但也最危險（容易過火） |

**我的偏好**：**B 模糊懷疑** — 最有層次。巴爺「想不通」+「不敢想」+「不想想」 — 這是被打壓過的人共同的心理機制。玩家高好感後可以幫他理清。揭露時的衝擊最大。

### 兩人現在的表面互動

- **塔倫對巴爺**（公開）：稱兄道弟、講「我當年救了你」、「自己人要顧好」
  - 但**從不讓巴爺接近主人 / 業務決策**（怕他知道更多）
  - 巴爺有問題 → 塔倫總是「私下處理」（不讓主人聽到）
- **巴爺對塔倫**（公開）：恭敬、聽話、但**眼神不對**
  - 喝酒會避開塔倫
  - 同場時話特別少
  - 玩家高好感前期看不出問題、後期才發現

### 這套關係解鎖的劇情潛力

1. **塔倫的「好人形象」是面具** — 玩家剛開始會被騙（看他演照顧自己人）
2. **真相揭露的層次感** — 不是一句話揭露、是慢慢拼湊
   - 看到巴爺喝酒避塔倫
   - 聽到老默/卡西烏斯的隻字片語
   - 巴爺自己醉後說的半句話
   - 最後玩家可能比巴爺自己還早想通
3. **§ 6 訣別事件變得**超有重量 — 巴爺說「⋯⋯如果我倒了，繼續站到底」可能還帶著「⋯⋯別讓他騙了你」的隱含警告
4. **未來可加路線**：玩家把真相告訴巴爺 → 觸發某些激烈事件（後期 / 結局影響）

### 巴爺對玩家的關注 = 自我投射（v4 重新定義）
- 他當年也是「太強眼」、被塔倫覺得礙眼、被推去送死
- 看到玩家也太突出 → 私下提醒「別站太前面」
- 看到玩家被塔倫派危險任務 → 臉色不對、會幫忙看
  - 後期甚至可能告訴玩家「**那場別接**」（救命線索）

### 為什麼塔倫沒升更高？
- 塔倫升到長官後就卡住 — 他**不是主人家族成員**
- 阿圖斯主人才是真正的權力源頭
- 塔倫對主人永遠是工具人，他的所有計算最終服務主人
- 這也解釋為什麼巴爺能活下來：**塔倫只能在他職權內動手腳，動不了主人決策**

## § 4 likedTraits / dislikedTraits 重排

### 塔倫長官（依使用者 spec）
**核心邏輯：能幫主人賺錢的人 = 可靠 + 聽話 + 會贏會輸**

```js
likedTraits:    { reliable:3, loyal:3, patient:2, iron_will:1 }
dislikedTraits: { impulsive:3, opportunist:3, prideful:2, coward:1 }
```

- `reliable` 升 +3（從 +3 維持）：說到做到 = 派得出去
- `loyal` 升 +3（從 +2）：聽話 = 主人的工具
- `patient` 維持 +2：不衝動 = 場上能控節奏
- `prideful` 變 -2（從 -1 加重）：場上要會輸的人不能驕傲
- `coward` 降 -1（從 -2 減輕）：怕死的人有時候反而會聽話

### 監督官巴爺（依使用者 spec）
**核心邏輯：撐下去站到底 = 勇敢 + 勤勞 / 討厭找理由**

```js
likedTraits:    { iron_will:3, diligence:3, prideful:2, cruel:1 }
dislikedTraits: { neurotic:3, coward:3, opportunist:2, merciful:1 }
```

- `iron_will` +3：撐到底的能力
- `diligence` +3：勤勞 = 不偷懶 = 不找理由
- `prideful` +2：勇敢敢撐才能站到底（跟塔倫相反）
- `cruel` +1：場上能下手 = 能活
- `neurotic` -3：「想太多」最怕（**新增**，配合「不要找理由」）
- `coward` -3：怕死撐不過第一場
- `merciful` -1：場上心軟會死

**塔倫 vs 巴爺最大反差**：
- `prideful`：塔倫 -2 / 巴爺 +2 — 「驕傲是奴隸的奢侈」 vs 「沒驕傲撐不下去」
- `loyal`：塔倫 +3 / 巴爺 0 — 「聽話最重要」 vs 「我管你聽不聽話、能撐就好」

## § 5 抓偷懶分工（依使用者 spec）

### 出現機率階梯（從低到高）
| 抓人者 | 偷懶被抓機率 | 場景 |
|---|---|---|
| **主人 (masterArtus)** | 很低（~5%） | 主人很少巡訓練場 |
| **侍從 (masterServant)** | 低（~10%） | 主人代行 |
| **塔倫長官 (officer)** | 中（~20%） | 平時不巡、出現 = 有事 |
| **監督官（巴爺）** | **75%+** | 他工作就是盯 |

### 沒人抓的情境
- 25% 的時間「**沒人出現**」 = 玩家好好休息的時段（恢復一些 stamina + mood）

### 抓包後的處罰邏輯（依抓人者 + 巴爺好感）

| 抓人者 | 處罰 |
|---|---|
| **主人（masterArtus）出現** | **大懲罰** — HP -10、好感 -15、震動+紅光 + 對白：「主人面前偷懶 = 大忌」 |
| **塔倫長官** | 中懲罰 — HP -5、好感 -10、冷酷對白「沒救的東西」 |
| **巴爺**：好感 < 30 | 直接抽打 — HP -5、好感 -3、抽鞭演出 |
| **巴爺**：好感 ≥ 30 | **只大吼** — HP 0、好感 -1、罵兩句完事「你他媽再讓我看到一次⋯⋯」 |
| **巴爺**：好感 ≥ 60 | 假裝沒看到 — 好感 -0、對白：「⋯⋯我這次當沒看到。下次別再讓我看見。」 |
| **侍從** | 輕懲 — HP -2、好感 -5（侍從會去報告主人）、後續主人可能找上門 |
| **無人** | 平安偷懶 — stamina +5 / mood +3 |

### 設計意圖（玩家學到什麼）
- **跟巴爺打好關係 → 偷懶被抓不會死**（是這個訓練所唯一的緩衝）
- **主人在 = 千萬別偷懶**（最大恐懼）
- 塔倫長官出現 = 不是平時巡邏，是有事

## § 6 巴爺跟玩家的好感互動線（完整故事）

> 使用者要求「**完整的人生跟故事**」+「跟玩家有好感上的互動」 — 比奧蘭 / 赫克托稍輕一點，但比卡西烏斯更厚

### storyReveals（5 段，aff 門檻遞增 — v4 對齊「設局陷害+假裝力保」）

| ID | aff | 揭露（內容暗示玩家從哪一層看到塔倫的真面目）|
|---|---|---|
| `overseer_silent` | 0 | flavor — 他在訓練場不講廢話。罵的時候不重複 |
| `overseer_old_scar` | 15 | flavor — 你發現他左前臂有一道又長又老的疤痕。**最後一戰留的** |
| `overseer_friends_call` | 30 | event — 你聽到梅拉叫他「巴爺」的時候，他笑了一下。第一次看到他笑 |
| `overseer_avoids_tarren` | 50 | event — 慶功宴（或某場合）— **巴爺看到塔倫進來，默默把酒杯放下走了**。沒人提起。 |
| `overseer_doubt` | 75 | event — 他酒後對你講最後一戰：「⋯⋯那場對手強得不像話。我從來沒搞懂為什麼把那種人配給我。」 然後他說「⋯⋯算了」。 |

**設計**：5 段揭露走的是「**從感激恩人 → 隱約懷疑**」的路。第 5 段不講破真相、留給玩家自己拼湊。完整揭露在訣別事件（aff 70+）。

### 互動事件（aff 門檻 + 條件觸發，~3-4 個）

#### 事件 1 — 第一次罵你罵到收手（aff ~25 + 你練得很勤）
```
（巴爺站在你旁邊看了很久。）
巴爺：你今天揮的這幾下——
（他停下。）
巴爺：⋯⋯算了。你自己知道。
（他走開。沒有罵。）
```
玩家心想：（這是肯定嗎？）

#### 事件 2 — 「巴爺」這個外號的由來（aff ~40 + 場上有梅拉）
- 梅拉端飯來訓練場，叫了一聲「巴爺，吃飯了」
- 巴爺沒回頭，但揮了揮手
- 玩家對白選項：
  - 問梅拉「為什麼叫他巴爺？」 → 梅拉笑著講三言兩語的過去
  - 不問 → 心想（⋯⋯這外號是怎麼來的？）

#### 事件 3 — 跟卡西烏斯的對戲（aff ~50 + 卡西烏斯也在場）
- 兩個老角鬥士偶遇 — 沒講話、互相點頭
- 玩家旁觀 → log 揭露「兩人是同期入場的」
- 後續卡西烏斯可能講一句「他比我撐得久」

#### 事件 4 — 訣別 / 託付（aff ~70 + Day ≥ 70）
- 巴爺在某個夜晚找你
- 對白（重量級，依「方案 B 模糊懷疑」設定）：
  ```
  巴爺：⋯⋯小子。
  巴爺：你最近上場的對手——
  （他停下。）
  巴爺：⋯⋯我跟你說個事。
  巴爺：當年我最後一場——對手太強。
  巴爺：強到我一直在想，是不是有人故意配那種人給我。
  （他沉默很久。）
  巴爺：但⋯⋯算了。我活下來了。
  巴爺：你聽好——
  巴爺：當年沒人對我說這句話。
  巴爺：所以這次我說。
  巴爺：如果我倒了——繼續站到底。
  巴爺：然後——
  巴爺：（他看你一眼。）
  巴爺：⋯⋯別讓人騙了你。聽到沒？
  ```
- flag set：`overseer_passed_torch` + `overseer_warned_about_tarren`
- 後續結局判定可能加分支
- 設計重量：
  - 巴爺把火炬傳下去（「當年沒人對我說這句」）
  - 巴爺自己也終於開始懷疑（「對手太強⋯⋯是不是有人故意」）
  - 隱含警告（「別讓人騙了你」）— 玩家心領神會
  - **不講破塔倫的名字**（巴爺自己也沒確證、且這是奴隸社會、說了會死）
