## 使用方式

### 在手機上寫東西（3 個方法擇一）

**方法 A：GitHub 網頁**（最簡單，不裝 app）
1. 打開手機瀏覽器 → `https://github.com/cooldragon313/gladiator100dz`
2. 找到 `NOTES.md` → 點右上角鉛筆圖示（Edit）
3. 寫完 → 拉到最下 → 填一句 commit message → 按 Commit
4. 完成（無須 pull/push，GitHub 直接寫入 main 分支）

**方法 B：iOS 用 Working Copy app**
1. 裝 Working Copy（免費版就夠用）
2. clone 這個 repo
3. 打開 NOTES.md 編輯 → commit → push

**方法 C：Android 用 Termux**
1. 裝 Termux + git
2. `git clone` 一次、以後 `git pull / git push` 同步
